## 二、不定积分

### 2.1 原函数与不定积分的定义

**【原函数】** 若 $F'(x) = f(x)$ 在区间 $I$ 成立，则称 $F(x)$ 是 $f(x)$ 的一个原函数。

**【不定积分】** $f(x)$ 的全体原函数称为 $f(x)$ 的不定积分，记为：
$$\int f(x) dx = F(x) + C$$

其中 $C$ 是任意常数。

### 2.2 基本积分公式表 ⚠️ 必背

| 序号 | 积分公式 | 序号 | 积分公式 |
|------|---------|------|---------|
| 1 | $\int 0 \, dx = C$ | 2 | $\int x^a dx = \dfrac{x^{a+1}}{a+1} + C$（$a \neq -1$）|
| 3 | $\int \dfrac{1}{x} dx = \ln|x| + C$ | 4 | $\int e^x dx = e^x + C$ |
| 5 | $\int a^x dx = \dfrac{a^x}{\ln a} + C$ | 6 | $\int \cos x \, dx = \sin x + C$ |
| 7 | $\int \sin x \, dx = -\cos x + C$ | 8 | $\int \sec^2 x \, dx = \tan x + C$ |
| 9 | $\int \csc^2 x \, dx = -\cot x + C$ | 10 | $\int \dfrac{1}{\sqrt{1-x^2}} dx = \arcsin x + C$ |
| 11 | $\int \dfrac{1}{1+x^2} dx = \arctan x + C$ | 12 | $\int \sec x \tan x \, dx = \sec x + C$ |
| 13 | $\int \csc x \cot x \, dx = -\csc x + C$ | 14 | $\int \dfrac{1}{a^2 + x^2} dx = \dfrac{1}{a}\arctan\dfrac{x}{a} + C$ |
| 15 | $\int \dfrac{1}{\sqrt{a^2 - x^2}} dx = \arcsin\dfrac{x}{a} + C$ | 16 | $\int \tan x \, dx = -\ln|\cos x| + C$ |
| 17 | $\int \cot x \, dx = \ln|\sin x| + C$ | | |

**【记忆方法】** 公式9和公式3的导数是1/x, sin/cos的导数是cos/-sin

---

### 2.3 不定积分的性质

| 性质 | 公式 |
|------|------|
| 线性性 | $\int [af(x) + bg(x)] dx = a\int f(x) dx + b\int g(x) dx$ |
| 求导 | $\dfrac{d}{dx}[\int f(x) dx] = f(x)$ |
| 还原 | $\int F'(x) dx = F(x) + C$ |

## 三、换元积分法 ⚠️ 必考

### 3.1 第一类换元法（凑微分法）

**【原理】**
$$\int f[\varphi(x)] \varphi'(x) dx = \int f[\varphi(x)] d\varphi(x) = F[\varphi(x)] + C$$

**【关键】** "看谁像谁"——将被积函数中的一部分凑成 $d\varphi(x)$

**【例题3.1】** 求 $\int \dfrac{\ln x}{x} dx$。

**解**：
$$\int \frac{\ln x}{x} dx = \int \ln x \, d(\ln x) = \frac{(\ln x)^2}{2} + C$$

**【例题3.2】** 求 $\int x e^{x^2} dx$。

**解**：
$$\int x e^{x^2} dx = \frac{1}{2}\int e^{x^2} d(x^2) = \frac{1}{2} e^{x^2} + C$$

**【例题3.3】** 求 $\int \tan x \, dx$。

**解**：
$$\int \tan x \, dx = \int \frac{\sin x}{\cos x} dx = -\int \frac{d(\cos x)}{\cos x} = -\ln|\cos x| + C$$

**【例题3.4】** 求 $\int \dfrac{1}{x^2 + 2x + 5} dx$。

**解**：
$$\int \frac{1}{(x+1)^2 + 4} dx = \frac{1}{2} \arctan\frac{x+1}{2} + C$$

### 3.2 第二类换元法

**【适用】** 根号下的换元（三角代换）

| 被积函数类型 | 换元 |
|-------------|------|
| $\sqrt{a^2 - x^2}$ | $x = a\sin t$ |
| $\sqrt{a^2 + x^2}$ | $x = a\tan t$ |
| $\sqrt{x^2 - a^2}$ | $x = a\sec t$ |

**【例题3.5】** 求 $\int \sqrt{a^2 - x^2} \, dx$。

**解**：
设 $x = a\sin t$，$dx = a\cos t \, dt$，$\sqrt{a^2 - x^2} = a\cos t$

$$\int a\cos t \cdot a\cos t \, dt = a^2 \int \cos^2 t \, dt = \frac{a^2}{2} \int (1 + \cos 2t) dt = \frac{a^2}{2}\left(t + \frac{\sin 2t}{2}\right) + C$$

$$= \frac{a^2}{2}(t + \sin t \cos t) + C = \frac{a^2}{2}\arcsin\frac{x}{a} + \frac{x}{2}\sqrt{a^2 - x^2} + C$$

## 四、分部积分法 ⚠️ 必考

### 4.1 公式

$$\int u \, dv = uv - \int v \, du$$

### 4.2 适用类型

| 类型 | 口诀 | 例子 |
|------|------|------|
| 反对幂三指 | "反"对数、"对"数、"幂"函数、"三"角、"指"数 | $\int x^2 \ln x \, dx$，$\int x \sin x \, dx$ |

**【关键】** 设 $u$ 的优先级：**反 → 对 → 幂 → 三 → 指**

### 4.3 常用 $v$ 公式

| 导数 $dv$ | 原函数 $v$ |
|----------|----------|
| $dx$ | $x$ |
| $e^x dx$ | $e^x$ |
| $\sin x \, dx$ | $-\cos x$ |
| $\cos x \, dx$ | $\sin x$ |
| $\sec^2 x \, dx$ | $\tan x$ |
| $\dfrac{1}{x} dx$ | $\ln x$ |
| $\dfrac{1}{\sqrt{1-x^2}} dx$ | $\arcsin x$ |
| $\dfrac{1}{1+x^2} dx$ | $\arctan x$ |

### 4.4 例题精讲

**【例题3.6】** 求 $\int x \cos x \, dx$。

**解**：
设 $u = x$，$dv = \cos x \, dx$，则 $du = dx$，$v = \sin x$

$$\int x \cos x \, dx = x \sin x - \int \sin x \, dx = x \sin x + \cos x + C$$

**【例题3.7】** 求 $\int x^2 e^x \, dx$。

**解**：
第一次分部：$u = x^2$，$dv = e^x dx$，$v = e^x$
$$\int x^2 e^x dx = x^2 e^x - \int 2x e^x dx$$
第二次分部：$u = 2x$，$dv = e^x dx$
$$\int 2x e^x dx = 2x e^x - \int 2e^x dx = 2x e^x - 2e^x$$
合并：$\int x^2 e^x dx = x^2 e^x - 2x e^x + 2e^x + C$

**【例题3.8】** 求 $\int \ln x \, dx$。

**解**：
$u = \ln x$，$dv = dx$，$v = x$
$$\int \ln x \, dx = x \ln x - \int x \cdot \frac{1}{x} dx = x \ln x - x + C$$

**【例题3.9】** 求 $\int e^x \sin x \, dx$。

**解**：
设 $I = \int e^x \sin x \, dx$

$u = \sin x$，$dv = e^x dx$
$$I = e^x \sin x - \int e^x \cos x \, dx$$

再次分部（$u = \cos x$）：
$$I = e^x \sin x - e^x \cos x - \int e^x \sin x \, dx = e^x \sin x - e^x \cos x - I$$

$$2I = e^x (\sin x - \cos x) + C$$
$$I = \frac{e^x(\sin x - \cos x)}{2} + C$$

## 五、有理函数积分

### 5.1 部分分式分解

| 分母形式 | 分解 |
|---------|------|
| $(x-a)(x-b)$ | $\dfrac{A}{x-a} + \dfrac{B}{x-b}$ |
| $(x-a)^2$ | $\dfrac{A}{x-a} + \dfrac{B}{(x-a)^2}$ |
| $(x-a)(x^2+b)$ | $\dfrac{A}{x-a} + \dfrac{Bx+C}{x^2+b}$ |

**【例题3.10】** 求 $\int \dfrac{1}{x^2 - 1} dx$。

**解**：
$$\int \frac{1}{(x-1)(x+1)} dx = \frac{1}{2}\int \left(\frac{1}{x-1} - \frac{1}{x+1}\right) dx = \frac{1}{2}\ln\left|\frac{x-1}{x+1}\right| + C$$

**【例题3.11】** 求 $\int \dfrac{x+1}{x^2 + x + 1} dx$。

**解**：
分子拆为 $x + \frac{3}{2}$ 和 $\frac{1}{2}$：
$$\int \frac{x + \frac{1}{2}}{x^2 + x + 1} dx + \int \frac{\frac{1}{2}}{x^2 + x + 1} dx$$

第一部分 $= \frac{1}{2} \ln(x^2 + x + 1)$

第二部分 配方 $x^2 + x + 1 = (x + \frac{1}{2})^2 + \frac{3}{4}$
$$= \frac{1}{2} \cdot \frac{1}{\sqrt{3/4}} \arctan \frac{x + 1/2}{\sqrt{3/4}} = \frac{1}{\sqrt{3}}\arctan \frac{2x+1}{\sqrt{3}}$$

最终结果：$\frac{1}{2}\ln(x^2 + x + 1) + \frac{1}{\sqrt{3}}\arctan \frac{2x+1}{\sqrt{3}} + C$

## 六、定积分

### 6.1 定义

$$\int_a^b f(x) dx = \lim_{\lambda \to 0} \sum_{i=1}^{n} f(\xi_i) \Delta x_i$$

其中 $\lambda = \max\{\Delta x_i\}$。

### 6.2 几何意义

- $\int_a^b f(x) dx$ 表示 $y = f(x)$ 与 $x$ 轴、直线 $x=a, x=b$ 围成的**有向面积**
- 上方取正，下方取负

### 6.3 性质

| 性质 | 公式 |
|------|------|
| 线性性 | $\int_a^b [\alpha f(x) + \beta g(x)] dx = \alpha \int_a^b f(x) dx + \beta \int_a^b g(x) dx$ |
| 区间可加 | $\int_a^c f dx = \int_a^b f dx + \int_b^c f dx$ |
| 保号性 | 若 $f(x) \geq 0$，则 $\int_a^b f(x) dx \geq 0$（$a < b$）|
| 估值 | $m(b-a) \leq \int_a^b f(x) dx \leq M(b-a)$ |
| 中值定理 | $\exists \xi \in [a,b]$，使 $\int_a^b f(x) dx = f(\xi)(b-a)$ |

### 6.4 牛顿-莱布尼茨公式 ⚠️ 必考

$$\boxed{\int_a^b f(x) dx = F(b) - F(a) = F(x)\Big|_a^b}$$

其中 $F(x)$ 是 $f(x)$ 的任意一个原函数。

**【例题3.12】** 求 $\int_0^1 x^2 dx$。

**解**：
$$\int_0^1 x^2 dx = \left.\frac{x^3}{3}\right|_0^1 = \frac{1}{3} - 0 = \frac{1}{3}$$

## 七、变上限积分函数 ⚠️ 必考

### 7.1 定理

设 $F(x) = \int_a^{g(x)} f(t) dt$，则：
$$F'(x) = f[g(x)] \cdot g'(x)$$

更一般：
$$F(x) = \int_{\varphi(x)}^{\psi(x)} f(t) dt \Rightarrow F'(x) = f[\psi(x)] \psi'(x) - f[\varphi(x)] \varphi'(x)$$

**【例题3.13】** 设 $F(x) = \int_0^{x^2} \sin t \, dt$，求 $F'(x)$。

**解**：
$$F'(x) = \sin(x^2) \cdot 2x = 2x \sin(x^2)$$

**【例题3.14】** 设 $F(x) = \int_x^{x^2} e^{-t^2} dt$，求 $F'(x)$。

**解**：
$$F'(x) = e^{-x^4} \cdot 2x - e^{-x^2} \cdot 1 = 2x e^{-x^4} - e^{-x^2}$$

### 7.2 综合应用

**【例题3.15】** 求 $\lim_{x \to 0} \dfrac{\int_0^x e^{t^2} dt}{x}$。

**解**：
$\frac{0}{0}$ 型，洛必达：
$$= \lim_{x \to 0} \frac{e^{x^2}}{1} = 1$$

## 八、定积分的计算

### 8.1 换元法

$$\int_a^b f[\varphi(x)] \varphi'(x) dx = \int_{\varphi(a)}^{\varphi(b)} f(u) du$$

**注意**：换元必换限！

**【例题3.16】** 求 $\int_0^4 \dfrac{1}{1 + \sqrt{x}} dx$。

**解**：
设 $\sqrt{x} = t$，$x = t^2$，$dx = 2t \, dt$。$x=0$ 时 $t=0$；$x=4$ 时 $t=2$。

$$= \int_0^2 \frac{2t}{1 + t} dt = 2\int_0^2 \left(1 - \frac{1}{1+t}\right) dt = 2[t - \ln(1+t)]_0^2 = 2(2 - \ln 3) = 4 - 2\ln 3$$

### 8.2 分部积分法

$$\int_a^b u \, dv = [uv]_a^b - \int_a^b v \, du$$

**【例题3.17】** 求 $\int_0^1 x e^x \, dx$。

**解**：
$$= [x e^x]_0^1 - \int_0^1 e^x dx = e - [e^x]_0^1 = e - (e - 1) = 1$$

### 8.3 偶函数与奇函数

| 函数类型 | 区间 | 公式 |
|---------|------|------|
| 偶函数 | $[-a, a]$ | $\int_{-a}^a f(x) dx = 2\int_0^a f(x) dx$ |
| 奇函数 | $[-a, a]$ | $\int_{-a}^a f(x) dx = 0$ |

**【例题3.18】** 求 $\int_{-1}^1 \dfrac{x^3 + x^2 \sin x}{1 + x^2 + x^4} dx$。

**解**：
$\dfrac{x^3}{1 + x^2 + x^4}$ 在 $[-1, 1]$ 是奇函数，积分为 0

$\dfrac{x^2 \sin x}{1 + x^2 + x^4}$ 在 $[-1, 1]$ 是奇函数，积分为 0

所以原式 $= 0$

## 九、广义积分

### 9.1 无穷区间广义积分

$$\int_a^{+\infty} f(x) dx = \lim_{b \to +\infty} \int_a^b f(x) dx$$

如果极限存在，称广义积分**收敛**；否则**发散**。

**【例题3.19】** 判断 $\int_0^{+\infty} e^{-x} dx$ 的敛散性。

**解**：
$$\int_0^{+\infty} e^{-x} dx = \lim_{b \to +\infty} [-e^{-x}]_0^b = \lim_{b \to +\infty} (1 - e^{-b}) = 1$$

所以收敛，值为 1。

### 9.2 无界函数广义积分

$$\int_a^b f(x) dx = \lim_{\varepsilon \to 0^+} \int_a^{b-\varepsilon} f(x) dx$$

## 十、定积分应用 ⚠️ 必考

### 10.1 平面图形面积

**【公式】**
$$A = \int_a^b |f(x) - g(x)| dx$$

**【例题3.20】** 求 $y = x^2$ 与 $y = x$ 围成图形的面积。

**解**：
交点：$x^2 = x \Rightarrow x = 0, x = 1$

在 $[0, 1]$ 上 $x > x^2$

$$A = \int_0^1 (x - x^2) dx = \left[\frac{x^2}{2} - \frac{x^3}{3}\right]_0^1 = \frac{1}{2} - \frac{1}{3} = \frac{1}{6}$$

### 10.2 旋转体体积

**【绕 x 轴】**
$$V_x = \pi \int_a^b f^2(x) dx$$

**【绕 y 轴】**
$$V_y = 2\pi \int_a^b x |f(x)| dx$$

**【例题3.21】** 求 $y = x^2$，$x = 1$，$y = 0$ 围成图形绕 $x$ 轴的体积。

**解**：
$$V = \pi \int_0^1 (x^2)^2 dx = \pi \int_0^1 x^4 dx = \frac{\pi}{5}$$

### 10.3 平行截面面积已知的体积

$$V = \int_a^b A(x) dx$$

## 十一、配套练习（10题精练）

### 练习1
求 $\int (2x + 3)^5 dx$。
<details>
<summary>🔑 答案</summary>

设 $u = 2x + 3$，$du = 2dx$
$$= \frac{1}{2}\int u^5 du = \frac{1}{2} \cdot \frac{u^6}{6} + C = \frac{(2x+3)^6}{12} + C$$

</details>

### 练习2
求 $\int x \ln x \, dx$。
<details>
<summary>🔑 答案</summary>

$u = \ln x$，$dv = x \, dx$，$v = \frac{x^2}{2}$
$$= \frac{x^2}{2}\ln x - \int \frac{x^2}{2} \cdot \frac{1}{x} dx = \frac{x^2}{2}\ln x - \frac{x^2}{4} + C$$

</details>

### 练习3
求 $\int_0^{\pi/2} \sin^2 x \, dx$。
<details>
<summary>🔑 答案</summary>

$$\int_0^{\pi/2} \frac{1 - \cos 2x}{2} dx = \frac{1}{2}\left[x - \frac{\sin 2x}{2}\right]_0^{\pi/2} = \frac{1}{2} \cdot \frac{\pi}{2} = \frac{\pi}{4}$$

</details>

### 练习4
求 $\int \dfrac{1}{x^2 + 4x + 5} dx$。
<details>
<summary>🔑 答案</summary>

$x^2 + 4x + 5 = (x+2)^2 + 1$
$$= \int \frac{1}{(x+2)^2 + 1} dx = \arctan(x+2) + C$$

</details>

### 练习5
设 $F(x) = \int_0^{\sin x} \sqrt{1 + t^3} dt$，求 $F'(0)$。
<details>
<summary>🔑 答案</summary>

$F'(x) = \sqrt{1 + \sin^3 x} \cdot \cos x$
$F'(0) = \sqrt{1 + 0} \cdot 1 = 1$

</details>

### 练习6
求 $\int_0^1 \dfrac{x^2}{1 + x^2} dx$。
<details>
<summary>🔑 答案</summary>

$$\int_0^1 \frac{x^2 + 1 - 1}{1 + x^2} dx = \int_0^1 1 \, dx - \int_0^1 \frac{1}{1+x^2} dx = 1 - \frac{\pi}{4}$$

</details>

### 练习7
求 $y = \sin x$，$y = \cos x$ 在 $[0, \pi/2]$ 围成的面积。
<details>
<summary>🔑 答案</summary>

交点 $\sin x = \cos x \Rightarrow x = \pi/4$

$$A = \int_0^{\pi/4} (\cos x - \sin x) dx + \int_{\pi/4}^{\pi/2} (\sin x - \cos x) dx$$
$$= [\sin x + \cos x]_0^{\pi/4} + [-\cos x - \sin x]_{\pi/4}^{\pi/2}$$
$$= (\sqrt{2} - 1) + (1 - \sqrt{2}) \text{(取绝对值)}$$

最终 $A = 2(\sqrt{2} - 1)$

</details>

### 练习8
求 $\int_0^{+\infty} \dfrac{1}{1 + x^2} dx$。
<details>
<summary>🔑 答案</summary>

$$= \lim_{b \to +\infty} \arctan x \Big|_0^b = \lim_{b \to +\infty} (\arctan b - 0) = \frac{\pi}{2}$$

</details>

### 练习9
求 $\int \sin^3 x \, dx$。
<details>
<summary>🔑 答案</summary>

$$\int \sin^2 x \cdot \sin x \, dx = -\int (1 - \cos^2 x) d(\cos x) = -\cos x + \frac{\cos^3 x}{3} + C$$

</details>

### 练习10
求 $y = x^2$ 与 $x = y^2$ 围成的面积。
<details>
<summary>🔑 答案</summary>

两曲线：$y = x^2$（开口向上）和 $x = y^2$（开口向右）
交点：$(0, 0)$ 和 $(1, 1)$

$$A = \int_0^1 (\sqrt{x} - x^2) dx = \left[\frac{2x^{3/2}}{3} - \frac{x^3}{3}\right]_0^1 = \frac{2}{3} - \frac{1}{3} = \frac{1}{3}$$

</details>
