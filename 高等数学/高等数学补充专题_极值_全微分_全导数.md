## 一、一元函数极值

### 1.1 极值定义

设 f(x) 在 x₀ 附近有定义：

- 若存在 δ>0，使 0<|x-x₀|<δ 时 **f(x) ≤ f(x₀)**，则 f(x₀) 为**极大值**
- 若存在 δ>0，使 0<|x-x₀|<δ 时 **f(x) ≥ f(x₀)**，则 f(x₀) 为**极小值**

x₀ 称为极值点。

> 注意：极值是局部概念，不是全局最大最小。

---

### 1.2 极值的必要条件

**费马定理：** 若 f(x) 在 x₀ 处可导且取得极值，则 **f'(x₀) = 0**

满足 f'(x₀)=0 的点称为**驻点**。

| 关系 | 说明 |
|------|------|
| 极值点 → f'(x₀)=0 或不可导 | 可导极值点必是驻点 |
| 驻点 ⇏ 极值点 | 驻点不一定是极值点（如 y=x³ 在 x=0） |
| 不可导点也可能是极值点 | 如 y=\|x\| 在 x=0 |

**找极值候选点：** 驻点 + 导数不存在的点

---

### 1.3 极值的第一充分条件（一阶导数判别法）

设 f(x) 在 x₀ 处连续，在 x₀ 附近可导（x₀ 可除外）：

| 条件 | 结论 |
|------|------|
| f'(x) 左正右负（左增右减） | f(x₀) 为**极大值** |
| f'(x) 左负右正（左减右增） | f(x₀) 为**极小值** |
| f'(x) 左右同号 | **不是极值** |

**口诀：** 左正右负极大值，左负右正极小值，左右同号非极值

---

### 1.4 极值的第二充分条件（二阶导数判别法）

设 f'(x₀)=0，f''(x₀) 存在：

| 条件 | 结论 |
|------|------|
| f''(x₀) > 0 | f(x₀) 为**极小值** |
| f''(x₀) < 0 | f(x₀) 为**极大值** |
| f''(x₀) = 0 | **无法判断**，改用第一充分条件 |

**记忆：** 二阶导正→凹→极小（碗口朝上）；二阶导负→凸→极大（碗口朝下）

---

### 1.5 求一元函数极值的完整步骤

```
第一步：求 f'(x)
第二步：令 f'(x)=0，解出全部驻点；找出 f'(x) 不存在的点
第三步：对每个候选点判别
  ├─ 方法A：看 f'(x) 在该点两侧的符号变化（第一充分条件）
  └─ 方法B：求 f''(x)，代入驻点看正负（第二充分条件）
第四步：写出极值点和极值
```

### 1.6 最大值与最小值

**闭区间 [a,b] 上连续函数的最值：**

1. 求 (a,b) 内所有驻点和不可导点
2. 计算这些点及端点 a、b 的函数值
3. 比较大小，最大者为最大值，最小者为最小值

> **实际应用题技巧：** 若 f(x) 在区间内只有一个驻点，且问题本身存在最值，则该驻点即为最值点，不必判别。

---

### 1.7 极值应用典型题型

| 题型 | 关键步骤 |
|------|---------|
| 几何最值（面积/体积最大最小） | 设变量→建目标函数→求导→令为零 |
| 距离最值 | 距离平方作为目标函数（避免根号） |
| 经济最值（成本最低/利润最大） | 成本/利润函数求导 |
| 物理最值（速度/功/能量） | 物理公式建立函数关系 |

## 二、多元函数极值

### 2.1 二元函数极值定义

设 f(x,y) 在点 (x₀,y₀) 附近有定义：

- 若附近所有点满足 **f(x,y) ≤ f(x₀,y₀)**，则 f(x₀,y₀) 为**极大值**
- 若附近所有点满足 **f(x,y) ≥ f(x₀,y₀)**，则 f(x₀,y₀) 为**极小值**

---

### 2.2 极值的必要条件

**若 f(x,y) 在 (x₀,y₀) 偏导数存在且取得极值，则：**

fₓ(x₀,y₀) = 0 且 fᵧ(x₀,y₀) = 0

满足上式的点称为**驻点**。

> 同样：驻点不一定是极值点（如鞍点）；偏导不存在的点也可能是极值点。

---

### 2.3 极值的充分条件（判别法）

设 f(x,y) 在 (x₀,y₀) 有二阶连续偏导，且 (x₀,y₀) 是驻点。

记：

- **A = fₓₓ(x₀,y₀)**（对x求两次偏导）
- **B = fₓᵧ(x₀,y₀)**（先对x后对y求偏导）
- **C = fᵧᵧ(x₀,y₀)**（对y求两次偏导）
- **Δ = AC - B²**（判别式）

| 判别式 | A的符号 | 结论 |
|--------|---------|------|
| Δ > 0 | A > 0 | **极小值** |
| Δ > 0 | A < 0 | **极大值** |
| Δ < 0 | — | **不是极值**（鞍点） |
| Δ = 0 | — | **无法判断**，需用其他方法 |

**记忆口诀：**
- Δ 正看 A：A 正极小，A 负极大
- Δ 负必非极值
- Δ 零不确定

---

### 2.4 求二元函数极值的完整步骤

```
第一步：求 fₓ 和 fᵧ
第二步：令 fₓ=0 且 fᵧ=0，解方程组，求出全部驻点
第三步：求二阶偏导 fₓₓ、fₓᵧ、fᵧᵧ
第四步：对每个驻点计算 A=fₓₓ、B=fₓᵧ、C=fᵧᵧ、Δ=AC-B²
第五步：根据判别表确定极值类型
第六步：若 Δ=0，用定义或其他方法判断
```

### 2.5 条件极值（拉格朗日乘数法）

求 f(x,y) 在约束条件 **φ(x,y)=0** 下的极值：

**步骤：**
1. 构造拉格朗日函数：**L = f(x,y) + λφ(x,y)**
2. 解方程组：

| 方程 | 内容 |
|------|------|
| Lₓ = 0 | fₓ + λφₓ = 0 |
| Lᵧ = 0 | fᵧ + λφᵧ = 0 |
| L_λ = 0 | φ(x,y) = 0 |

3. 解出的 (x,y) 即为可能的极值点
4. 根据实际问题判断是极大还是极小

**多个约束条件：** L = f + λ₁φ₁ + λ₂φ₂

**实际应用：** 常用于求"在某某条件下，最大/最小是多少"的问题

## 三、全微分

### 3.1 全微分定义

设 z = f(x,y) 在 (x,y) 处：

若 Δz = f(x+Δx, y+Δy) - f(x,y) 可表示为：

**Δz = A·Δx + B·Δy + o(ρ)**

其中 ρ = √(Δx²+Δy²)，A、B 不依赖于 Δx、Δy，则称 f 在该点**可微**。

**全微分：** **dz = A·Δx + B·Δy = fₓ·dx + fᵧ·dy**

> 其中 A = fₓ，B = fᵧ

---

### 3.2 可微的必要条件

| 条件 | 结论 |
|------|------|
| f 可微 → | 偏导数 fₓ、fᵧ 存在 |
| f 可微 → | f 连续 |

**重要：** 偏导数存在 ≠ 可微！偏导存在但函数不一定连续，更不一定可微。

---

### 3.3 可微的充分条件

**若 fₓ 和 fᵧ 在 (x,y) 处连续，则 f 在该点可微。**

> 这是做题时最常用的判断依据。

---

### 3.4 关系总结（重要！）

```
偏导数连续
    ↓ 推出
  可微
    ↓ 推出
偏导数存在 + 连续
    ↓ 推出
  连续
```

| 已知条件 | 能推出的结论 |
|---------|-------------|
| 偏导连续 | → 可微 → 偏导存在 → 连续 |
| 可微 | → 偏导存在 + 连续 |
| 偏导存在 | → 什么也推不出（不一定连续，不一定可微） |
| 连续 | → 什么也推不出（偏导不一定存在） |

---

### 3.5 全微分计算

**二元函数：** z = f(x,y)

**dz = fₓ·dx + fᵧ·dy**

**三元函数：** u = f(x,y,z)

**du = fₓ·dx + fᵧ·dy + f_z·dz**

### 3.6 全微分在近似计算中的应用

当 |Δx|、|Δy| 很小时：

**f(x₀+Δx, y₀+Δy) ≈ f(x₀,y₀) + fₓ(x₀,y₀)·Δx + fᵧ(x₀,y₀)·Δy**

**典型用法：** 估算 (1.02)³·⁰¹ 等近似值

## 四、全导数

### 4.1 什么是全导数

当复合函数的所有中间变量都最终依赖于**一个自变量**时，求导结果称为**全导数**。

设 z = f(u,v)，而 u = u(t)，v = v(t)，则 z 最终是 t 的一元函数。

**全导数公式：**

**dz⁄dt = ∂z⁄∂u · du⁄dt + ∂z⁄∂v · dv⁄dt**

> 本质：链式法则在"多中间变量、单自变量"情形下的应用

---

### 4.2 全导数的几种情形

#### 情形一：z = f(u,v)，u=u(t)，v=v(t)

dz⁄dt = ∂z⁄∂u · du⁄dt + ∂z⁄∂v · dv⁄dt

**例：** z = u²+v²，u = sin t，v = cos t

dz⁄dt = 2u·cos t + 2v·(-sin t) = 2sin t·cos t - 2cos t·sin t = 0

---

#### 情形二：z = f(x,y)，x=x(t)，y=y(t)

dz⁄dt = ∂z⁄∂x · dx⁄dt + ∂z⁄∂y · dy⁄dt

（同情形一，只是变量名不同）

---

#### 情形三：z = f(u,v,t)，u=u(t)，v=v(t)

z 直接含 t，同时 u、v 也依赖 t：

**dz⁄dt = ∂z⁄∂u · du⁄dt + ∂z⁄∂v · dv⁄dt + ∂z⁄∂t**

> 注意：最后一项 ∂z⁄∂t 是 z 对第三位置变量 t 的偏导，不能漏！

**记忆：** 有几条路径到 t，就有几项相加

---

### 4.3 全导数 vs 偏导数 vs 全微分 对比

| 概念 | 适用情形 | 公式 |
|------|---------|------|
| **偏导数** | 多元函数，对一个变量求导，其他当常数 | ∂z⁄∂x = fₓ |
| **全导数** | 最终只有一个自变量的复合函数 | dz⁄dt = Σ ∂z⁄∂uᵢ · duᵢ⁄dt |
| **全微分** | 多元函数的线性近似 | dz = fₓ·dx + fᵧ·dy |

**区别要点：**
- 偏导：只动一个变量，其他不动
- 全导：所有中间变量都在动，但最终只沿一个方向
- 全微分：描述各变量微小变化引起的总变化量

---

### 4.4 求全导数的步骤

```
第一步：画出变量依赖关系（树状图）
  z → u → t
  z → v → t
  （z 可能直接→t）

第二步：找出所有从 z 到 t 的路径

第三步：每条路径上的偏导/导数相乘

第四步：所有路径的结果相加

第五步：将中间变量用 t 表示，化简
```

## 五、解题技巧速查

### 5.1 极值问题判别流程

```
一元函数极值？
  ├─ 求 f'(x)=0 → 驻点
  ├─ 找 f'(x) 不存在的点
  └─ 判别：
      ├─ f'(x) 两侧变号 → 第一充分条件
      └─ f''(x₀) 正负 → 第二充分条件

二元函数极值？
  ├─ 解 fₓ=0, fᵧ=0 → 驻点
  └─ 计算 Δ=AC-B²
      ├─ Δ>0, A>0 → 极小
      ├─ Δ>0, A<0 → 极大
      ├─ Δ<0 → 非极值
      └─ Δ=0 → 其他方法

条件极值？
  └─ 拉格朗日乘数法
```

### 5.2 全微分判别流程

```
问：f(x,y) 在某点是否可微？

第一步：偏导 fₓ、fᵧ 是否存在？
  ├─ 不存在 → 不可微
  └─ 存在 → 第二步

第二步：偏导是否连续？
  ├─ 连续 → 可微（充分条件）
  └─ 不确定 → 第三步

第三步：检验 lim [Δz - fₓΔx - fᵧΔy] / ρ 是否为 0
  ├─ = 0 → 可微
  └─ ≠ 0 或不存在 → 不可微
```

### 5.3 全导数求法

```
看到复合函数求导 → 先画树状图

所有路径汇聚到一个自变量 t → 全导数
  → 每条路径相乘后相加

路径汇聚到多个自变量 → 偏导数
  → 对某个自变量，只走通向它的路径
```

## 六、常见错误提醒

| 错误类型 | 正确做法 |
|---------|---------|
| 驻点当极值点 | 驻点不一定是极值，需判别 |
| 忘记检查不可导点 | 极值可能在导数不存在的点取到 |
| 二阶导=0 就放弃 | f''(x₀)=0 时改用第一充分条件 |
| 偏导存在就认为可微 | 偏导存在≠可微，需偏导连续或用定义验证 |
| 全导数漏掉某条路径 | 画树状图，数清路径条数 |
| 全导数漏 ∂z⁄∂t 项 | z 直接含 t 时，不要漏最后那项 |
| Δ=AC-B² 算反 | A=fₓₓ, B=fₓᵧ, C=fᵧᵧ，别把 B 当成 fᵧₓ 搞混 |
| 条件极值不用拉格朗日法 | 有约束条件时，拉格朗日乘数法最规范 |

---

> **复习建议：** 极值问题是考试必考题，务必熟练判别流程。全微分的"可微"判断是易错点，记住"偏导连续→可微"这条充分条件。全导数关键是画树状图，路径不要漏。
