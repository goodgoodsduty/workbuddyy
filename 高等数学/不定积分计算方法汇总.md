## 一、不定积分的相关概念

### 1. 原函数与不定积分

设函数 $f(x)$ 定义在某区间 $I$ 上，若存在可导函数 $F(x)$，对于 **该区间上任意一点** 都有 $F'(x)=f(x)$ 成立，则称 $F(x)$ 是 $f(x)$ 在区间 $I$ 上的一个 **原函数**。

于是称

$$\int f(x)\,dx = F(x)+C$$

为 $f(x)$ 在区间 $I$ 上的 **不定积分**，其中 $C$ 为任意常数（**后面不再强调**）。

> *PS：谈到函数 $f(x)$ 的原函数和不定积分，**必须指明** $f(x)$ **所在定义的区间。***

### 2. 原函数与不定积分的区别

我们通过对概念的说明去加以区别。

1. **原函数**：若 $f(x)$ 在区间 $I$ 上有原函数 $F(x)$，则 $f(x)$ 就有无限多个原函数，且任意两个原函数之间仅相差一个常数。所以 $f(x)$ 的全体原函数所构成的集合为 $\{F(x)+C \mid -\infty<C<+\infty\}$

2. **不定积分**：设 $F(x),H(x)$ 是 $f(x)$ 在区间 $I$ 上的原函数，虽有 $\int f(x)\,dx=F(x)+C_1$ 和 $\int f(x)\,dx=H(x)+C_2$，但 $F(x)=H(x)$ 不一定成立，因为常数 $C$ 一般是不相同的。

> *由此可见，二者在概念上存在较大的差异：**后者是个无限集，前者是后者中的一个元素。***

### 3. 不定积分与微分的关系

**口诀：先积后微，形式不变；先微后积，相差一个常数。**

1. $\left[\int f(x)\,dx\right]'=f(x)$ 或 $d\int f(x)\,dx=f(x)\,dx$ **（先积后微，形式不变）**
2. $\int F'(x)\,dx=F(x)+C$ 或 $\int dF(x)=F(x)+C$ **（先微后积，相差个常数）**

## 二、常用不定积分公式（基本积分公式）

> 这一板块灰常重要！It's important↓↓↓

| 编号 | 公式 |
|------|------|
| 1-① | $\displaystyle\int k\,dx=kx+C$（$k$ 是常数） |
| 1-② | $\displaystyle\int x^k\,dx=\frac{1}{k+1}x^{k+1}+C$（$k\ne -1$）。其中当 $k$ 取 $-\frac{1}{2}$、$-\frac{1}{3}$、$-2$ 时可得到常用的结论 |
| 1-③ | $\displaystyle\int \frac{1}{x}\,dx=\ln\|x\|+C$ |
| 1-④ | $\displaystyle\int e^x\,dx=e^x+C$；$\displaystyle\int a^x\,dx=\frac{a^x}{\ln a}+C$（$a>0,\,a\ne1$） |
| 1-⑤ | $\displaystyle\int \sin x\,dx=-\cos x+C$；$\displaystyle\int \cos x\,dx=\sin x+C$ |
| | $\displaystyle\int \tan x\,dx=-\ln\|\cos x\|+C$；$\displaystyle\int \cot x\,dx=\ln\|\sin x\|+C$ |
| 1-⑥ | $\displaystyle\int \sec x\,dx=\int \frac{dx}{\cos x}=\ln\|\sec x+\tan x\|+C$ |
| | $\displaystyle\int \csc x\,dx=\int \frac{dx}{\sin x}=\ln\|\csc x-\cot x\|+C$ |
| 1-⑦ | $\displaystyle\int \sec^2 x\,dx=\tan x+C$；$\displaystyle\int \csc^2 x\,dx=-\cot x+C$ |
| 1-⑧ | $\displaystyle\int \sec x\tan x\,dx=\sec x+C$；$\displaystyle\int \csc x\cot x\,dx=-\csc x+C$ |
| 1-⑨ | $\displaystyle\int \frac{1}{x^2+a^2}\,dx=\frac{1}{a}\arctan\frac{x}{a}+C$（$a>0$） |
| | $\displaystyle\int \frac{1}{x^2+1}\,dx=\arctan x+C$（$a=1$） |
| 1-⑩ | $\displaystyle\int \frac{1}{\sqrt{a^2-x^2}}\,dx=\arcsin\frac{x}{a}+C$（$a>0$） |
| | $\displaystyle\int \frac{1}{\sqrt{1-x^2}}\,dx=\arcsin x+C$（$a=1$） |
| 1-⑪ | $\displaystyle\int \frac{1}{\sqrt{x^2+a^2}}\,dx=\ln(x+\sqrt{x^2+a^2})+C$ |
| | $\displaystyle\int \frac{1}{\sqrt{x^2+1}}\,dx=\ln(x+\sqrt{x^2+1})+C$（$a=1$） |
| | $\displaystyle\int \frac{1}{\sqrt{x^2-a^2}}\,dx=\ln\left\|x+\sqrt{x^2-a^2}\right\|+C$（$\|x\|>\|a\|$） |
| 1-⑫ | $\displaystyle\int \frac{1}{x^2-a^2}\,dx=\frac{1}{2a}\ln\left\|\frac{x-a}{x+a}\right\|+C$ |
| | $\displaystyle\int \frac{1}{a^2-x^2}\,dx=\frac{1}{2a}\ln\left\|\frac{x+a}{x-a}\right\|+C$ |
| 1-⑬ | $\displaystyle\int \sin^2 x\,dx=\frac{x}{2}-\frac{\sin 2x}{4}+C$ |
| | $\displaystyle\int \cos^2 x\,dx=\frac{x}{2}+\frac{\sin 2x}{4}+C$ |
| | $\displaystyle\int \tan^2 x\,dx=\tan x-x+C$ |
| | $\displaystyle\int \cot^2 x\,dx=-\cot x-x+C$ |
| 1-⑭ | $\displaystyle\int \sqrt{x^2+a^2}\,dx=\frac{x}{2}\sqrt{x^2+a^2}+\frac{a^2}{2}\ln(x+\sqrt{x^2+a^2})+C$ |
| | $\displaystyle\int \sqrt{x^2-a^2}\,dx=\frac{x}{2}\sqrt{x^2-a^2}-\frac{a^2}{2}\ln\left\|x+\sqrt{x^2-a^2}\right\|+C$ |
| | $\displaystyle\int \sqrt{a^2-x^2}\,dx=\frac{x}{2}\sqrt{a^2-x^2}+\frac{a^2}{2}\arcsin\frac{x}{a}+C$ |

> **重要求导公式：**
> - $\left[\ln(x+\sqrt{x^2+a^2})\right]'=\frac{1}{\sqrt{x^2+a^2}}$
> - $\left[\ln\left\|x+\sqrt{x^2-a^2}\right\|\right]'=\frac{1}{\sqrt{x^2-a^2}}$（$\|x\|>\|a\|$）

> 这些不定积分请大家熟悉在心！

## 三、常用不定积分计算方法

> 这一个板块将为大家呈现常用的计算方法，也是做题的基本依据。部分内容引用自数分、高数18讲。

### （一）凑微分法（第一类换元积分）

**1. 基本思想：**

$$\int f[\varphi(x)]\varphi'(x)\,dx=\int f[\varphi(x)]\,d[\varphi(x)]=\int f(u)\,du, \quad u=\varphi(x)$$

**2. 说明：** 当被积函数有一部分比较复杂时，我们可以通过观察把某些函数放到 $d$ 的后面（放在 $d$ 后面的函数会发生变化），使得 $d$ 后面的函数与前面复杂的被积函数 **具有相似的结构**，最后运用基本积分公式将其求出（若不能求出的话则进一步运用其它方法求出）。

**3. 举例说明**

**例1：** 计算 $\displaystyle\int x^{-2}\sin\frac{2}{x}\,dx$

通过观察我们发现 $\sin\frac{2}{x}$ 这部分较复杂，且 $\int x^{-2}\,dx=-\frac{1}{x}+C$，我们发现将 $x^{-2}$ 进行积分后的函数与前面复杂的函数具有相似的结构（都有 $\frac{N}{x}$），最后运用基本积分公式求出（当然这里凑微分时要凑成 $\frac{2}{x}$，然后不定积分前乘 $\frac{1}{2}$ 即可）。

**解：原式**

$$=-\int \sin\frac{2}{x}\,d\left(\frac{1}{x}\right)=-\frac{1}{2}\int \sin\frac{2}{x}\,d\left(\frac{2}{x}\right)=\frac{1}{2}\cos\frac{2}{x}+C$$

**例2：** 计算 $\displaystyle\int \frac{dx}{x\sqrt{x^{2}-2x-3}}$（数学分析例题）

这种类型积分比较复杂，直接给大家说明，这种不定积分凑 $\frac{1}{x}$ 比较合适，最常见的方法是三角代换（第二类换元积分将会陈述）。

**解：原式**

$$=\int \frac{dx}{x^{2}\sqrt{1-2x^{-1}-3x^{-2}}} \quad \textbf{（根式提个x出来，便于凑 $\frac{1}{x}$）}$$

$$=-\int \frac{d(x^{-1})}{\sqrt{1-2x^{-1}-3x^{-2}}} \quad \textbf{（凑微分）}$$

$$=-\frac{1}{\sqrt{3}}\int \frac{d(x^{-1})}{\sqrt{\frac{1}{3}-\frac{2}{3}x^{-1}-x^{-2}}} \quad \textbf{（根式提个3出来，使得2次项系数为1）}$$

$$=-\frac{1}{\sqrt{3}}\int \frac{d\left(x^{-1}+\frac{1}{3}\right)}{\sqrt{\left(\frac{2}{3}\right)^{2}-\left(x^{-1}+\frac{1}{3}\right)^{2}}} \quad \textbf{（分母凑完全平方）}$$

$$=-\frac{\sqrt{3}}{3}\arcsin\frac{x+3}{2x}+C \quad \textbf{（基本积分1-⑩）}$$

> *PS：凑微分时加不加常数无影响，即 $d[f(x)]=d[f(x)+C]$

> 第一类换元法实质上是求复合函数导数的逆过程！

**4. 常见凑微分公式总结**

> （部分公式来源：《剑指150数一》——高昆轮）

| 编号 | 公式 |
|------|------|
| 2-① | $\displaystyle\int f(ax+b)\,dx=\frac{1}{a}\int f(ax+b)\,d(ax+b)$（$a\ne0$） |
| 2-② | $\displaystyle\int f(ax^2+b)x\,dx=\frac{1}{2a}\int f(ax^2+b)\,d(ax^2+b)$（$a\ne0$） |
| 2-③ | $\displaystyle\int f(ax^n+b)x^{n-1}\,dx=\frac{1}{na}\int f(ax^n+b)\,d(ax^n+b)$（$a,n\ne0$） |

> - $(1+\ln x)\,dx=d(x\ln x)$

### （二）换元法（第二类换元积分）

**1. 基本思想：**

$$\int f(x)\,dx=\int f[\varphi(u)]\,d[\varphi(u)]=\int f[\varphi(u)]\varphi'(u)\,du, \quad x=\varphi(u)$$

**2. 说明：** 当被积函数比较复杂时，可以通过换元的方法从 $d$ 后面的函数放一部分到前面来，使其更容易积分。

**3. 举例说明**

**例1：** 计算 $\displaystyle\int \frac{x+1}{(1+x^{2})^{2}}\,dx$

通过观察发现分母是 $1+x^{2}$ 的形式，于是想到三角代换（$1+\tan^2 x=\sec^2 x$）。

**解：** 令 $x=\tan u$，则 $dx=\sec^2 u\,du$，**原式**

$$=\int \frac{\tan u+1}{(1+\tan^{2}u)^{2}}\sec^2 u\,du$$

$$=\int \frac{\tan u+1}{\sec^{4}u}\sec^2 u\,du=\int \frac{\tan u+1}{\sec^2 u}\,du$$

$$=\int \sin u\cdot\cos u\,du+\int \cos^2 u\,du$$

$$=\int \sin u\,d(\sin u)+\int \cos^2 u\,du$$

$$=\frac{1}{2}\sin^2 u+\frac{u}{2}+\frac{1}{4}\sin 2u+C$$

$$=-\frac{1}{4}\cos 2u+\frac{u}{2}+\frac{1}{4}\sin 2u+C_1$$

> *PS：$\int \sin u\cdot\cos u\,du=\frac{1}{2}\int \sin 2u\,du=-\frac{1}{4}\cos 2u+C$，其中 $C=C_1-\frac{1}{4}$*

**然后画一个三角形**（刚才令的 $x=\tan u$，画草图的时候对边为 $x$，邻边为 $1$，角度为 $u$）

**由三角形可以得到：**

$$\sin u=\frac{x}{\sqrt{x^{2}+1}}, \quad \cos u=\frac{1}{\sqrt{x^{2}+1}}$$

**代入上式得：**

$$\int \frac{x+1}{(1+x^{2})^{2}}\,dx=\frac{x^2}{2(1+x^2)}+\frac{1}{2}\arctan x+\frac{x}{2(1+x^2)}+C$$

**例2：** 计算 $\displaystyle\int \frac{dx}{x\sqrt{x^{2}-2x-3}}$

**解：原式**

$$=\int \frac{dx}{x\sqrt{(x-1)^{2}-2^2}} \quad \textbf{（想到 $\sec^2 x-1=\tan^2 x$，令 $x-1=2\sec u$）}$$

$$=\int \frac{2\sec u\cdot\tan u}{(2\sec u+1)\cdot 2\tan u}\,du$$

$$=\int \frac{du}{2+\cos u} \quad \textbf{（用万能代换——$t=\tan\frac{u}{2}$，具体内容见总结⑤）}$$

$$=\int \frac{\frac{2}{1+t^2}}{2+\frac{1-t^2}{1+t^2}}\,dt=\int \frac{2}{t^{2}+3}\,dt$$

$$=\frac{2}{\sqrt{3}}\int \frac{d\!\left(\frac{t}{\sqrt{3}}\right)}{\left(\frac{t}{\sqrt{3}}\right)^2+1}$$

$$=\frac{2}{\sqrt{3}}\arctan\frac{t}{\sqrt{3}}+C \quad \textbf{（基本积分1-⑨）}$$

$$=\frac{2}{\sqrt{3}}\arctan\!\left(\frac{1}{\sqrt{3}}\tan\frac{u}{2}\right)+C$$

**由** $x-1=2\sec u$ **可知，** $\cos u=\frac{2}{x-1}$

**画出辅助三角形，由三角形可以得到：**

$$\sin u=\frac{\sqrt{x^2-2x-3}}{x-1}$$

**根据公式** $\tan\frac{u}{2}=\frac{\sin u}{1+\cos u}$，将 $\sin u$、$\cos u$ 的表达式代入上式得：

$$\int \frac{dx}{x\sqrt{x^{2}-2x-3}}=\frac{2}{\sqrt{3}}\arctan\frac{\sqrt{x^{2}-2x-3}}{\sqrt{3}(x+1)}+C$$

**4. 总结常见的换元方法**（部分引用张宇18讲）

#### ① 三角函数代换

当被积函数含有以下根式时（$a>0$）：

| 根式形式 | 换元 | 关系 |
|----------|------|------|
| $\sqrt{a^2-x^2}$ | $x=a\sin u$ | $1-\sin^2 u=\cos^2 u$ |
| $\sqrt{a^2+x^2}$ | $x=a\tan u$ | $1+\tan^2 u=\sec^2 u$ |
| $\sqrt{x^2-a^2}$ | $x=a\sec u$ | $\sec^2 u-1=\tan^2 u$ |

> *PS：某些根式 $\sqrt{Ax^2+Bx+C}$，可通过配方后恒等变形化为以下三种模型，再作相应换元。*

$\sqrt{u^2+a^2}$、$\sqrt{u^2-a^2}$、$\sqrt{a^2-u^2}$（比如：$\int \frac{dx}{x\sqrt{x^{2}-2x-3}}$）

**具体说来，**（节选自《数学复习全书：数学一》——李正元、范培华、尤承业）

**Ⅰ、若 $A>0$，** 配方后得：

$$Ax^2+Bx+C=\left(\sqrt{A}\,x+\frac{B}{2\sqrt{A}}\right)^2+\frac{4AC-B^2}{4A}$$

令 $u=\sqrt{A}\,x+\frac{B}{2\sqrt{A}}$。

- 当 $4AC-B^2>0$ 时，令 $a^2=\frac{4AC-B^2}{4A}$；
- 当 $4AC-B^2<0$ 时，令 $a^2=\frac{B^2-4AC}{4A}$；

此时，$\sqrt{Ax^2+Bx+C}$ 分别化为 $\sqrt{u^2+a^2}$ 或 $\sqrt{u^2-a^2}$。

**Ⅱ、若 $A<0$，** $\sqrt{Ax^2+Bx+C}$ 经过配方后可以化为 $\sqrt{a^2-u^2}$。

#### ② 根式代换

当被积函数含有 $\sqrt[n]{ax+b}$、$\sqrt[n]{\frac{ax+b}{cx+d}}$（$ad-bc\ne0$）、$\sqrt[n]{ae^{bx}+c}$ 等时，一般令 $\sqrt{★}=t$（有时候根号很难去掉），具体说来：

**（1）** $\displaystyle\int R(x,\sqrt[n]{ax+b})\,dx$

令 $\sqrt[n]{ax+b}=t$，$x=\frac{t^n-b}{a}$，$dx=\frac{n}{a}t^{n-1}\,dt$

**（2）** $\displaystyle\int R\!\left(x,\sqrt[n]{\frac{ax+b}{cx+d}}\right)dx$

令 $\sqrt[n]{\frac{ax+b}{cx+d}}=t$，$x=-\frac{b-dt^n}{a-ct^n}$，$dx=\frac{n(ad-bc)t^{n-1}}{(a-ct^{n})^2}\,dt$

**（3）** $\displaystyle\int R(x,\sqrt[n]{ae^{bx}+c})\,dx$

令 $\sqrt[n]{ae^{bx}+c}=t$，$x=\frac{1}{b}\ln\!\left(\frac{t^n-c}{a}\right)$，$dx=\frac{n}{b}\cdot\frac{t^{n-1}}{t^n-c}\,dt$

**例：** 计算 $\displaystyle\int \sqrt{\frac{1-x}{1+x}}\frac{dx}{x}$（同济七版教材习题4-4，NO.23）

**解法一：** 令 $\sqrt{\frac{1-x}{1+x}}=t$，则 $x=\frac{1-t^2}{1+t^2}$，$dx=-\frac{4t}{(1+t^2)^2}\,dt$

**原式**

$$=\int t\cdot\frac{1+t^2}{1-t^2}\cdot\frac{-4t}{(1+t^2)^2}\,dt$$

$$=4\int \frac{t^2}{(t^2-1)(t^2+1)}\,dt$$

$$=2\int\!\left(\frac{1}{t^2-1}+\frac{1}{t^2+1}\right)dt$$

$$=\ln\left\|\frac{t-1}{t+1}\right\|+2\arctan t+C \quad \textbf{（公式1-⑨、⑫）}$$

$$=\ln\left\|\frac{\sqrt{1-x}-\sqrt{1+x}}{\sqrt{1-x}+\sqrt{1+x}}\right\|+2\arctan\sqrt{\frac{1-x}{1+x}}+C$$

**解法二：**

$$=\int\sqrt{\frac{(1-x)(1+x)}{(1+x)^2}}\frac{dx}{x}=\int\sqrt{\frac{1-x^2}{(1+x)^2}}\frac{dx}{x}$$

令 $x=\sin u$，$dx=\cos u\,du$

**原式**

$$=\int \frac{\cos^2 u}{\sin u(1+\sin u)}\,du=\int\frac{1-\sin u}{\sin u}\,du$$

$$=\int \csc u\,du-u+C$$

$$=\ln\|\csc u-\cot u\|-u+C$$

根据三角代换得，$\csc u=\frac{1}{x}$，$\cot u=\frac{\sqrt{1-x^2}}{x}$

**原式**

$$=\ln\left\|\frac{1-\sqrt{1-x^2}}{x}\right\|-\arcsin x+C$$

> 若被积函数中即含有 $\sqrt[n]{ax+b}$，又含有 $\sqrt[m]{ax+b}$ 的结构，令 $\sqrt[l]{ax+b}=t$（$l=\text{lcm}(m,n)$，$l$ 为 $m$、$n$ 的最小公倍数），于是 $x=\frac{t^l-b}{a}$，$dx=\frac{l\cdot t^{l-1}}{a}\,dt$

**例：** 计算 $\displaystyle\int \frac{dx}{\sqrt{x}+\sqrt[4]{x}}$（同济七版教材习题4-4，NO.22）

**解：** 首先观察被积函数中即含有 $\sqrt{x}$（2次根），又含有 $\sqrt[4]{x}$（4次根）的结构，则最小公倍数为4，于是令 $\sqrt[4]{x}=t$（$x=t^{4}$，$dx=4t^{3}\,dt$）。

**原式**

$$=\int \frac{4t^{3}}{t^2+t}\,dt=4\int \frac{t^{2}}{1+t}\,dt$$

$$=4\!\left[\int \frac{(t^2-1)\,dt}{1+t}+\int \frac{dt}{1+t}\right]$$

$$=4\!\left[\frac{t^2}{2}-t+\ln(1+t)\right]+C$$

$$=2\sqrt{x}-4\sqrt[4]{x}+4\ln(1+\sqrt[4]{x})+C$$

#### ③ 倒代换

在被积函数中，分母的次数比分子的次数高2次及以上时（不是所有都行得通），可令 $x=\frac{1}{t}$。

**例：** 计算 $\displaystyle\int \frac{dx}{x(x^{8}+2)}$

**解：** 宏观地看，分母次数高于分子次数，令 $x=\frac{1}{t}$。

**原式**

$$=\int \frac{t}{\frac{1}{t^{8}}+2}\cdot\!\left(-\frac{1}{t^{2}}\right)dt$$

$$=-\int \frac{t^{7}\,dt}{1+2t^{8}}$$

$$=-\frac{1}{16}\int \frac{d(1+2t^{8})}{1+2t^{8}}$$

$$=\frac{1}{16}\ln\left\|\frac{1}{1+2t^{8}}\right|+C$$

$$=\frac{1}{16}\ln\!\left(\frac{x^{8}}{x^{8}+2}\right)+C$$

#### ④ 复杂函数直接代换

若被积函数中含有 $e^{x}$、$\ln x$、$\arctan x$、$\arcsin x$ 等之类时，可以把这些函数令为 $t$。若 $\ln x$、$\arctan x$、$\arcsin x$ 与 $P_{n}(x)$ 或 $e^{ax}$ 作乘法时（$P_{n}(x)$ 为 $x$ 的 $n$ 次多项式），优先考虑分部积分法。

**例：** 计算 $\displaystyle\int \frac{x\cdot\arctan x}{\sqrt{1+x^2}}\,dx$

**解：** 令 $\arctan x=u$，则 $x=\tan u$，$dx=\sec^2 u\,du$。

**原式：**

$$=\int \frac{\tan u\cdot u}{\sqrt{1+\tan^2 u}}\sec^2 u\,du$$

$$=\int u\cdot(\tan u\cdot\sec u)\,du=\int u\,d(\sec u)$$

$$=u\cdot\sec u-\int \sec u\,du \quad \textbf{（分部积分法）}$$

$$=u\cdot\sec u-\ln\|\sec u+\tan u\|+C$$

**画一个辅助三角形**（$x=\tan u$），由图可知，$\sec u=\sqrt{1+x^2}$

**故原式**

$$=\sqrt{1+x^2}\arctan x-\ln\left\|x+\sqrt{1+x^2}\right\|+C$$

#### ⑤ 万能代换

$\displaystyle\int R(\sin x,\cos x)\,dx$ 是三角函数有理式不定积分，一般令 $t=\tan\frac{x}{2}$ 可以将其化为有理函数的不定积分。

由 $t=\tan\frac{x}{2}$，根据万能公式得：

$$\sin x=\frac{2t}{1+t^2},\quad \cos x=\frac{1-t^2}{1+t^2},\quad dx=\frac{2}{1+t^2}\,dt$$

则 $\displaystyle\int R(\sin x,\cos x)\,dx=\int R\!\left(\frac{2t}{1+t^2},\frac{1-t^2}{1+t^2}\right)\frac{2}{1+t^2}\,dt$

**例：** 计算 $\displaystyle\int \frac{dx}{2\sin x-\cos x+5}$

**解：** 令 $u=\tan\frac{x}{2}$，$dx=\frac{2}{1+u^2}\,du$

**原式**

$$=\int \frac{\frac{2}{1+u^2}}{2\cdot\frac{2u}{1+u^2}-\frac{1-u^2}{1+u^2}+5}\,du$$

$$=\int \frac{du}{3u^2+2u+2}$$

$$=\frac{1}{3}\int \frac{d\!\left(u+\frac{1}{3}\right)}{\left(u+\frac{1}{3}\right)^2+\frac{5}{9}}$$

$$=\frac{1}{3}\cdot\frac{3}{\sqrt{5}}\arctan\frac{u+\frac{1}{3}}{\frac{\sqrt{5}}{3}}+C$$

$$=\frac{\sqrt{5}}{5}\arctan\frac{3\tan\frac{x}{2}+1}{\sqrt{5}}+C$$

#### ⑥ 关于三角函数的几种变换

遇到三角有理函数的不定积分，并不是所有的都要通过万能代换去处理，这里总结了部分相关结论（实质上是某些凑微分的过程换了个说法而已）。

**对于 $\displaystyle\int f(\sin x,\cos x)\,dx$ 型的积分：**

- **（1）** 若 $f(-\sin x,\cos x)=-f(\sin x,\cos x)$，一般凑 $d(\cos x)$；
- **（2）** 若 $f(\sin x,-\cos x)=-f(\sin x,\cos x)$，一般凑 $d(\sin x)$；
- **（3）** 若 $f(-\sin x,-\cos x)=f(\sin x,\cos x)$，一般凑 $d(\tan x)$；

> *注：若 $f$ 中出现 $\tan x$、$\cot x$、$\sec x$、$\csc x$，皆可以通过三角函数恒等变换为只含有 $\sin x$、$\cos x$ 的形式。*

**例（第1种情况）：** 计算 $\displaystyle\int \sin^3 x\cdot\cos^6 x\,dx$

**解：** $\because (-\sin x)^3\cdot\cos^6 x=-\sin^3 x\cdot\cos^6 x$，属于第（1）种情况，所以凑 $d(\cos x)$。

**原式**

$$=-\int \sin^2 x\cdot\cos^6 x\,d(\cos x)$$

$$=\int (\cos^2 x-1)\cos^6 x\,d(\cos x)$$

$$=\int \cos^8 x\,d(\cos x)-\int \cos^6 x\,d(\cos x)$$

$$=\frac{1}{9}\cos^9 x-\frac{1}{7}\cos^7 x+C$$

**例（第3种情况）：** 计算 $\displaystyle\int \frac{1}{a^2\sin^2 x+b^2\cos^2 x}\,dx$（其中 $a$、$b$ 全不为0）

**解：** 显然被积函数属于第三种情况，即关于 $\sin x$、$\cos x$ 的偶函数，故凑 $d(\tan x)$。

**原式**

$$=\int \frac{\sec^2 x}{a^2\tan^2 x+b^2}\,dx$$

$$=\int \frac{d(\tan x)}{(a\tan x)^2+b^2}$$

$$=\frac{1}{a}\int \frac{d(a\tan x)}{(a\tan x)^2+b^2}$$

$$=\frac{1}{ab}\arctan\frac{a\tan x}{b}+C$$

#### ⑦ 欧拉（Euler）变换

欧拉变换也可以将含有根式的不定积分化为有理函数的积分。

- **（1）** 当 $a>0$ 时，令 $\sqrt{ax^2+bx+c}=t\pm\sqrt{a}\,x$；
- **（2）** 当 $c>0$ 时，令 $\sqrt{ax^2+bx+c}=xt\pm\sqrt{c}$；
- **（3）** 当 $ax^2+bx+c=a(x-\lambda)(x-\mu)$ 时，令 $\sqrt{ax^2+bx+c}=t(x-\lambda)$ 或 $\sqrt{ax^2+bx+c}=t(x-\mu)$。

**例：** 计算 $\displaystyle\int \frac{dx}{x+\sqrt{x^{2}+2x+2}}$

**解：** 作欧拉变换，令 $\sqrt{x^{2}+2x+2}=t-x$，解得：

$$x=\frac{t^2-2}{2(t+1)},\quad dx=\frac{1}{2}\!\left[1+\frac{1}{(t+1)^2}\right]dt$$

**原式**

$$=\frac{1}{2}\int \frac{1}{t}\!\left[1+\frac{1}{(t+1)^2}\right]dt$$

$$=\frac{1}{2}\int\!\left[\frac{2}{t}-\frac{1}{t+1}-\frac{1}{(t+1)^2}\right]dt$$

$$=\frac{1}{2}\ln\frac{t^2}{t+1}+\frac{1}{2(t+1)}+C$$

$$=\frac{1}{2}\ln\frac{(x+\sqrt{x^2+2x+2})^2}{x+\sqrt{x^2+2x+2}+1}+\frac{1}{2(x+\sqrt{x^2+2x+2}+1)}+C$$

#### ⑧ 积化和差

对于 $\displaystyle\int \sin mx\cdot\sin nx\,dx$、$\displaystyle\int \sin mx\cdot\cos nx\,dx$、$\displaystyle\int \cos mx\cdot\cos nx\,dx$（$m\ne n$）的类型，可利用积化和差来计算。

#### ⑨ $\sin^m x\cdot\cos^n x$ 型

对于 $\displaystyle\int \sin^m x\cdot\cos^n x\,dx$ 类型，若当 $m$、$n$ 中有一个奇数，可拆开利用凑微分法来计算（具体见⑥）；若 $m$、$n$ 都是偶数，可利用倍角公式逐步求出不定积分。

#### ⑩ $\sin^m x$、$\cos^n x$ 型

对于 $\displaystyle\int \sin^m x\,dx$、$\displaystyle\int \cos^n x\,dx$ 类型积分，可利用分部积分法导出递推公式计算。

### （三）分部积分法

**1. 基本思想：** $\displaystyle\int u\,dv=uv-\int v\,du$（更好积分）

**2. 口诀：反、对、幂、三、指（指、三），谁在前，谁不动；谁在后，d进去（放在d后面）。**

**3. 说明：**

- **①** 比如被积函数中出现了反函数和三角函数，根据口诀顺序就把三角函数放在 $d$ 后面，其它的情况类似（若函数中出现三角函数和指数函数的情形，把谁放在 $d$ 后面都可以）。
- **②** 分部积分法习惯上去用表格去计算。

**4. 例题分析**

**（1）常规型**

**计算：** $\displaystyle\int (x^2-1)\sin 2x\,dx$（同济教材习题4-3，NO.17）

**解：** 观察发现被积函数是由幂函数和三角函数组成，根据口诀把三角函数放在 $d$ 后面。令 $v=\sin 2x$，$u=x^2-1$。

**原式**

$$=-\frac{1}{2}(x^2-1)\cos 2x+\frac{x}{2}\sin 2x+\frac{1}{4}\cos 2x+C$$

$$=-\frac{1}{2}\!\left(x^2-\frac{3}{2}\right)\cos 2x+\frac{x}{2}\sin 2x+C$$

**（2）循环型**

**计算：** $\displaystyle\int e^{-2x}\sin\frac{x}{2}\,dx$（同济教材习题4-3，NO.7）

**解：** 观察发现被积函数是由指数函数和三角函数组成，根据口诀可以把三角函数或指数函数放在 $d$ 后面。令 $v=\sin\frac{x}{2}$，$u=e^{-2x}$。

由表格可知：

$$\int e^{-2x}\sin\frac{x}{2}\,dx=-2e^{-2x}\cos\frac{x}{2}-8e^{-2x}\sin\frac{x}{2}-16\int e^{-2x}\sin\frac{x}{2}\,dx$$

$$\therefore \int e^{-2x}\sin\frac{x}{2}\,dx=-\frac{2}{17}e^{-2x}\!\left(\cos\frac{x}{2}+4\sin\frac{x}{2}\right)+C$$

> 由此可见，这种算法多见于指数函数和三角函数的情形。

**（3）变通型**

**计算：** $\displaystyle\int x^2\arctan x\,dx$（同济教材习题4-3，NO.9）

**解：** 观察发现被积函数是由反三角函数和幂函数组成，根据口诀把幂函数放在 $d$ 后面。令 $v=x^2$，$u=\arctan x$。

这种方法中 $\arctan x$ 求导后一直不为0，需要对表格求导后的那一列作一个调配。

**原式**

$$=\frac{x^{3}}{3}\arctan x-\frac{1}{6}x^2+\frac{1}{6}\ln(1+x^2)+C$$

> *PS：①该方法实质上是部分计算过程中换了种形式而已。②重新调配的结果不影响符号变化：因为我们是将第二列作了调配，所以后面的符号按照第二列确定。*

## 四、不定(定)积分计算特殊方法

### （一）抵消型不定积分计算

这种类型的不定积分如果用常规的方法会比较麻烦。这种积分在处理的时候往往先将其拆成两项，拆成两项后先对第一项进行积分，**第一项（或第二项）不定积分计算的同时必然会用到分部积分法**，分部计算出的结果必然会抵消掉第二项（或第一项）不定积分。

> *PS：以上我说的只是我在做大量练习时遇到的情况。*

这里只通过几道题来说明。

**例1：** 计算 $\displaystyle\int \frac{2x+1}{x^2}e^{-2x}\,dx$

这种积分用常规的方法是不好处理的，于是先将积分中含有分式的部分拆为2项。

**解：**

$$I=\int \frac{2x+1}{x^2}e^{-2x}\,dx=\int\!\left(\frac{2}{x}+\frac{1}{x^2}\right)e^{-2x}dx$$

$$=\int \frac{2}{x}e^{-2x}\,dx+\int \frac{1}{x^2}e^{-2x}\,dx$$

这里就要考虑对哪一个不定积分运用分部积分的问题了。我们这样去想，首先这两个积分都是一个指数函数和幂函数的乘积形式，但进行分部积分后，幂函数必然会进行求导运算使其次方数减1。有了这样的考虑后，就对第一项运用分部积分法，因为只有这样才能使分母变为2次方结构。

**故，原式**

$$=-\int\frac{1}{x}\,d(e^{-2x})+\int \frac{1}{x^2}e^{-2x}\,dx$$

$$=-\frac{e^{-2x}}{x}+\int e^{-2x}\,d\!\left(\frac{1}{x}\right)+\int \frac{1}{x^2}e^{-2x}\,dx$$

$$=-\frac{e^{-2x}}{x}-\int \frac{1}{x^2}e^{-2x}\,dx+\int \frac{1}{x^2}e^{-2x}\,dx$$

$$=-\frac{e^{-2x}}{x}+C$$

**例2：** 计算 $\displaystyle\int \frac{x+\sin x}{1+\cos x}\,dx$

**解：** 先拆成两项，

$$I=\int\frac{x}{1+\cos x}\,dx+\int \frac{\sin x}{1+\cos x}\,dx$$

这里就对第二项运用分部积分法，道理上来说 $\sin x$ 求导后会出现 $\cos x$。如果实在不清楚选哪一个的话，可以依次检验。

$$I=\int\frac{x}{1+\cos x}\,dx+\frac{x\sin x}{1+\cos x}-\int x\,d\!\left(\frac{\sin x}{1+\cos x}\right)$$

$$=\int\frac{x}{1+\cos x}\,dx+\frac{x\sin x}{1+\cos x}-\int\frac{x}{1+\cos x}\,dx$$

$$=\frac{x\sin x}{1+\cos x}+C$$

当然，常规方法也可以做出来。

**法二：**

$$I=\int\frac{x}{1+\cos x}\,dx+\int \frac{\sin x}{1+\cos x}\,dx$$

其中，

$$\int \frac{\sin x}{1+\cos x}\,dx=-\ln(1+\cos x)+C_1$$

$$\int\frac{x}{1+\cos x}\,dx=\frac{1}{2}\int x\sec^{2}\!\frac{x}{2}\,dx$$

$$=x\tan\frac{x}{2}-\int \tan\frac{x}{2}\,dx$$

$$=x\tan\frac{x}{2}+2\ln\left\|\cos\frac{x}{2}\right\|+C_2$$

所以，

$$I=x\tan\frac{x}{2}+2\ln\left\|\cos\frac{x}{2}\right\|-\ln(1+\cos x)+C$$

$$=x\tan\frac{x}{2}+C$$

**例3：** 计算 $\displaystyle\int e^{x}\!\left(\frac{1-x}{1+x^2}\right)^2 dx$

还是常规方法，将其拆成两项，于是果断打开被积函数的平方。

**解：**

$$I=\int e^{x}\cdot\frac{1+x^2-2x}{(1+x^2)^2}\,dx$$

$$=\int e^{x}\cdot\frac{1}{1+x^2}\,dx-\int e^{x}\cdot\frac{2x}{(1+x^2)^2}\,dx$$

$$=\int \frac{e^x}{1+x^2}\,dx+\int e^x\,d\!\left(\frac{1}{1+x^2}\right)$$

$$=\int \frac{e^x}{1+x^2}\,dx+\frac{e^x}{1+x^2}-\int \frac{e^x}{1+x^2}\,dx \quad \textbf{（第二项运用分部积分）}$$

$$=\frac{e^x}{1+x^2}+C$$

> 这个题大家可以不妨尝试将第一项运用分部积分，也可以得出相同的结果。

### （二）求导观察型不定积分

**思想：** 当被积函数为 $f(x)/g(x)$，其中分子较为复杂，若对分母的一部分进行求导运算可以得到分子的常数倍或者是函数倍，从而可以进行凑微分进行计算。

**例1：** 计算 $\displaystyle\int \frac{(\cos^{2}x-\sin x)e^{\sin x}}{\cos x\cdot e^{\sin x}(1+\cos x\cdot e^{\sin x})}\,dx$（来源：张宇18讲）

首先观察这个被积函数，分子与分母都比较复杂，但是通过观察发现：

$$[\cos x\cdot e^{\sin x}]'=(\cos^2 x-\sin x)e^{\sin x}$$

于是将分母进行凑微分运算。

**解：**

$$I=\int \frac{d(\cos x\cdot e^{\sin x})}{\cos x\cdot e^{\sin x}(1+\cos x\cdot e^{\sin x})}$$

$$=\int\!\left(\frac{1}{\cos x\cdot e^{\sin x}}-\frac{1}{1+\cos x\cdot e^{\sin x}}\right)d(\cos x\cdot e^{\sin x})$$

$$=\ln\left\|\frac{\cos x\cdot e^{\sin x}}{1+\cos x\cdot e^{\sin x}}\right\|+C$$

有时候，这种观察往往不是特别容易，这时候需要自己创造条件。**（没有条件，创造条件！）**

**例2：** 计算 $\displaystyle\int \frac{x^2}{(x\sin x+\cos x)^2}\,dx$（来源：张宇18讲）

记 $f(x)=x\sin x+\cos x$，于是 $f'(x)=x\cos x$。由于被积函数分母没有 $\cos x$，于是想到将被积函数分子凑成 $x\cos x$，以便于像上个题那样通过凑微分进行处理。

**解：**

$$I=\int \frac{x}{\cos x}\cdot\frac{x\cos x}{(x\sin x+\cos x)^2}\,dx \quad \textbf{（这时就可以将乘号后的项进行凑微分）}$$

$$=\int \frac{x}{\cos x}\,d\!\left[-\left(\frac{1}{x\sin x+\cos x}\right)\right] \quad \textbf{（PS：$d\!\left[\frac{1}{f(x)}\right]=-\frac{f'(x)}{f^2(x)}\,dx$）}$$

$$=-\frac{x}{\cos x}\cdot\frac{1}{x\sin x+\cos x}+\int \frac{1}{x\sin x+\cos x}\!\left(\frac{x}{\cos x}\right)'dx$$

$$=-\frac{x}{\cos x}\cdot\frac{1}{x\sin x+\cos x}+\int \frac{1}{\cos^2 x}\,dx$$

$$=-\frac{x}{\cos x}\cdot\frac{1}{x\sin x+\cos x}+\tan x+C$$

$$=\frac{\sin x-x\cos x}{x\sin x+\cos x}+C$$

### （三）区间再现计算型（定积分）

这种类型"定"积分和上面两种特殊类型一样，用常规方法不好处理。然而，通过一定换元处理就会变得比较容易 **且保持上下限不变**。

**基本思想：** $\displaystyle\int_a^b f(x)\,dx=\int_a^b f(a+b-x)\,dx$

该思想的好处就是计算定积分时既可以保持上下限不变，又"刚好"可以计算。

**例1：** 计算 $\displaystyle\int_{0}^{\pi}x\sin^9 x\,dx$（来源：张宇18讲）

显然，常规方法不适用。于是利用"区间再现"，令 $x=0+\pi-t$。

$$I=\int_{\pi}^{0}(\pi-t)\sin^{9}(\pi-t)(-dt)$$

$$=\int_{\pi}^{0}(\pi-t)\sin^{9}(t)(-dt)$$

$$=\int_{0}^{\pi}(\pi-t)\sin^{9}(t)\,dt$$

$$=\int_{0}^{\pi}\pi\sin^9 t\,dt-\int_{0}^{\pi}t\sin^9 t\,dt$$

$$\therefore I=\frac{1}{2}\int_{0}^{\pi}\pi\sin^9 t\,dt$$

$$=\frac{1}{2}\!\left(\int_{0}^{\frac{\pi}{2}}\sin^9 t\,dt+\int_{\frac{\pi}{2}}^{\pi}\sin^9 t\,dt\right)$$

其中，

$$\int_{\frac{\pi}{2}}^{\pi}\sin^9 t\,dt=\int_{\frac{\pi}{2}}^{0}\sin^9(\pi-t)(-dt)=\int_{0}^{\frac{\pi}{2}}\sin^9 t\,dt$$

$$\therefore I=\int_{0}^{\pi}\pi\sin^9 t\,dt=\pi\cdot\frac{8}{9}\cdot\frac{6}{7}\cdot\frac{4}{5}\cdot\frac{2}{3}=\frac{128}{315}\pi$$

**这里用到了几个结论！**（以下结论均来源于 **张宇18讲**）

**① 华里士公式：**

**Ⅰ、** 若 $I_{n}=\displaystyle\int_{0}^{\frac{\pi}{2}}\sin^n x\,dx=\int_{0}^{\frac{\pi}{2}}\cos^n x\,dx$：

- **a.** 当 $n$ 为大于1的 **正奇数**：$I_{n}=\dfrac{n-1}{n}\cdot\dfrac{n-3}{n-2}\cdots\dfrac{4}{5}\cdot\dfrac{2}{3}\cdot I_1$（$I_1=1$），或 $I_{n}=\dfrac{(n-1)!!}{n!!}$
- **b.** 当 $n$ 为 **正偶数**：$I_{n}=\dfrac{n-1}{n}\cdot\dfrac{n-3}{n-2}\cdots\dfrac{3}{4}\cdot\dfrac{1}{2}\cdot I_0$（$I_0=\dfrac{\pi}{2}$），或 $I_{n}=\dfrac{(n-1)!!}{n!!}\cdot\dfrac{\pi}{2}$

**Ⅱ、** 若 $I_{n}=\displaystyle\int_{0}^{\pi}\sin^n x\,dx$：

- **c.** 当 $n$ 为大于1的 **正奇数**：$I_{n}=2\cdot\dfrac{n-1}{n}\cdot\dfrac{n-3}{n-2}\cdots\dfrac{4}{5}\cdot\dfrac{2}{3}\cdot I_1$（$I_1=1$），或 $I_{n}=2\cdot\dfrac{(n-1)!!}{n!!}$
- **d.** 当 $n$ 为 **正偶数**：$I_{n}=2\cdot\dfrac{n-1}{n}\cdot\dfrac{n-3}{n-2}\cdots\dfrac{3}{4}\cdot\dfrac{1}{2}\cdot I_0$（$I_0=\dfrac{\pi}{2}$），或 $I_{n}=\dfrac{(n-1)!!}{n!!}\cdot\pi$

**Ⅲ、** 若 $I_{n}=\displaystyle\int_{0}^{\pi}\cos^n x\,dx$：

- **e.** 当 $n$ 为 **正奇数**：$I_{n}=0$
- **f.** 当 $n$ 为 **正偶数**：$I_{n}=2\cdot\dfrac{(n-1)!!}{n!!}\cdot\dfrac{\pi}{2}=\dfrac{(n-1)!!}{n!!}\cdot\pi$

**Ⅳ、** $I_{n}=\displaystyle\int_{0}^{2\pi}\sin^n x\,dx=\int_{0}^{2\pi}\cos^n x\,dx$：

- **g.** 当 $n$ 为 **正奇数**：$I_{n}=0$
- **h.** 当 $n$ 为 **正偶数**：$I_{n}=4\cdot\dfrac{(n-1)!!}{n!!}\cdot\dfrac{\pi}{2}=\dfrac{(n-1)!!}{n!!}\cdot 2\pi$

**②-1、** $\displaystyle\int_{0}^{\pi}x\,f(\sin x)\,dx=\frac{\pi}{2}\int_{0}^{\pi}f(\sin x)\,dx=\pi\int_{0}^{\frac{\pi}{2}}f(\sin x)\,dx$

> *（PS：只有0到π才能这样用）*

**②-2、** $\displaystyle\int_{0}^{\frac{\pi}{2}}f(\sin x)\,dx=\int_{0}^{\frac{\pi}{2}}f(\cos x)\,dx$

**②-3、** $\displaystyle\int_{0}^{\frac{\pi}{2}}f(\sin x,\cos x)\,dx=\int_{0}^{\frac{\pi}{2}}f(\cos x,\sin x)\,dx$（被积函数中 $\sin x$、$\cos x$ 可以互换）

再来看一个常见 **对称区间积分情形。**

**例2：** 计算 $\displaystyle\int_{-\frac{\pi}{4}}^{\frac{\pi}{4}} \frac{2^{x-1}}{2^x+1}\cos^{4}2x\,dx$（来源：22张宇数二题源探析1K题）

> *PS：在做定积分时候，遇到对称区间优先考虑被积函数的奇偶性！*

由区间再现，令 $x=-\frac{\pi}{4}+\frac{\pi}{4}-u$，即 $x=-u$。

**解：原式**

$$I=\int_{\frac{\pi}{4}}^{-\frac{\pi}{4}} \frac{2^{-u-1}}{2^{-u}+1}\cos^{4}(-2u)(-du)$$

$$=\frac{1}{2}\int_{-\frac{\pi}{4}}^{\frac{\pi}{4}} \frac{2^{-u}}{2^{-u}+1}\cos^{4}2u\,du$$

$$=\frac{1}{2}\int_{-\frac{\pi}{4}}^{\frac{\pi}{4}} \frac{1}{2^u+1}\cos^{4}2u\,du$$

$$=\frac{1}{2}\int_{-\frac{\pi}{4}}^{\frac{\pi}{4}} \frac{1}{2^x+1}\cos^{4}2x\,dx$$

$$\therefore 2I=\int_{-\frac{\pi}{4}}^{\frac{\pi}{4}} \frac{2^{x-1}+2^{-1}}{2^x+1}\cos^{4}2x\,dx$$

$$=2^{-1}\int_{-\frac{\pi}{4}}^{\frac{\pi}{4}}\frac{2^x+1}{2^x+1}\cos^{4}2x\,dx$$

$$2I=2^{-1}\int_{-\frac{\pi}{4}}^{\frac{\pi}{4}}\cos^{4}2x\,dx$$

$$=\int_{0}^{\frac{\pi}{4}}\cos^{4}2x\,dx=\frac{1}{2}\int_{0}^{\frac{\pi}{2}}\cos^4 u\,du$$

$$\therefore I=\frac{1}{4}\cdot\frac{3}{4}\cdot\frac{1}{2}\cdot\frac{\pi}{2}=\frac{3\pi}{64}$$

这种类型题，有这样一个结论，大家可以熟记于心。

**③-1、** $\displaystyle\int_{-a}^{a}f(x)\,dx=\int_{0}^{a}[f(x)+f(-x)]\,dx$

**例3：** 计算 $\displaystyle\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}}\frac{\sin^4 x}{1+e^{-x}}\,dx$

**解：原式**

$$=\int_{0}^{\frac{\pi}{2}}\!\left[\frac{\sin^4 x}{1+e^{-x}}+\frac{\sin^4(-x)}{1+e^{-(-x)}}\right]dx$$

$$=\int_{0}^{\frac{\pi}{2}}\sin^4 x\,dx=\frac{3\pi}{16}$$

当然，这个结论属于特殊情况。当区间不是对称区间时，我们有以下结论。

**③-2、** $\displaystyle\int_{a}^{b}f(x)\,dx=\frac{1}{2}\int_{a}^{b}[f(x)+f(a+b-x)]\,dx$

**③-3、** $\displaystyle\int_{a}^{b}f(x)\,dx=\int_{a}^{\frac{a+b}{2}}[f(x)+f(a+b-x)]\,dx$

**③-4、** $\displaystyle\int_{a}^{b}f(x)\,dx=\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}}f\!\left(\frac{a+b}{2}+\frac{b-a}{2}\sin t\right)\cdot\frac{b-a}{2}\cos t\,dt$

> *PS：公式③-4称之为 **区间简化公式**，令 $x-\frac{a+b}{2}=\frac{b-a}{2}\sin t$，可得上式。*

**③-5、** $\displaystyle\int_{a}^{b}f(x)\,dx=\int_{0}^{1}(b-a)f[a+(b-a)t]\,dt$

**例4：** 计算 $\displaystyle\int_{a}^{b}\sqrt{(x-a)(b-x)}\,dx$

**解：** 令 $x-\frac{a+b}{2}=\frac{b-a}{2}\sin t$

**原式**

$$=\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}}\frac{b-a}{2}\cdot\cos t\cdot\frac{b-a}{2}\cdot\cos t\,dt$$

$$=\left(\frac{b-a}{2}\right)^2\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}}\cos^2 t\,dt$$

$$=\frac{(b-a)^2\pi}{8}$$

### （四）分式三角函数型

在这里介绍一种关于 $\displaystyle\int\frac{A\sin x+B\cos x}{D\sin x+E\cos x}\,dx$ **不定积分** 的求解方法。

遇到这种类型的积分，需对被积函数中的分子作处理：

**将分子中 "$A\sin x+B\cos x$" 这部分拆成两项，其中一部分为 "$D\sin x+E\cos x$" 的倍数，另一部分是 "$D\sin x+E\cos x$" 求导后结果的倍数。也即，**

$$\frac{A\sin x+B\cos x}{D\sin x+E\cos x}=\frac{p(D\sin x+E\cos x)+q(D\cos x-E\sin x)}{D\sin x+E\cos x}=p+q\cdot\frac{D\cos x-E\sin x}{D\sin x+E\cos x}$$

于是，

$$\int\frac{A\sin x+B\cos x}{D\sin x+E\cos x}\,dx=\int\!\left[p+q\cdot\frac{D\cos x-E\sin x}{D\sin x+E\cos x}\right]dx=px+q\cdot\ln\|D\sin x+E\cos x\|+C$$

然后运用待定系数法将 $p$、$q$ 求出。这里举个例子就明白了。

**例：** 计算 $\displaystyle\int \frac{3\sin x-2\cos x}{\sin x+2\cos x}\,dx$

记 $\frac{3\sin x-2\cos x}{\sin x+2\cos x}=\frac{p(\sin x+2\cos x)+q(\cos x-2\sin x)}{\sin x+2\cos x}$

通过整理得到：$p-2q=3$；$2p+q=-2$，则 $p=-\frac{1}{5}$、$q=-\frac{8}{5}$。

于是，

$$\int \frac{3\sin x-2\cos x}{\sin x+2\cos x}\,dx$$

$$=\int\frac{(-1/5)(\sin x+2\cos x)-8/5(\cos x-2\sin x)}{\sin x+2\cos x}\,dx$$

$$=-\frac{1}{5}x-\frac{8}{5}\int\frac{\cos x-2\sin x}{\sin x+2\cos x}\,dx$$

$$=-\frac{1}{5}x-\frac{8}{5}\int\frac{d(\sin x+2\cos x)}{\sin x+2\cos x}$$

$$=-\frac{1}{5}x-\frac{8}{5}\ln\|\sin x+2\cos x\|+C$$
