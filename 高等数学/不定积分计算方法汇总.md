## 一、不定积分的相关概念

### 1. 原函数与不定积分

设函数 $f(x)$ 定义在某区间 $I$ 上，若存在可导函数 $F(x)$，对于 **该区间上任意一点** 都有 $F'(x)=f(x)$ 成立，则称 $F(x)$ 是 $f(x)$ 在区间 $I$ 上的一个 **原函数**。

于是称

$$\int f(x)\,dx = F(x)+C$$

为 $f(x)$ 在区间 $I$ 上的 **不定积分**，其中 $C$ 为任意常数。

> *PS：谈到函数 $f(x)$ 的原函数和不定积分，**必须指明** $f(x)$ 所在定义的区间。*

### 2. 原函数与不定积分的区别

1. **原函数**：若 $f(x)$ 在区间 $I$ 上有原函数 $F(x)$，则 $f(x)$ 就有无限多个原函数，且任意两个原函数之间仅相差一个常数。所以 $f(x)$ 的全体原函数所构成的集合为 $\{F(x)+C \mid -\infty<C<+\infty\}$

2. **不定积分**：设 $F(x),H(x)$ 是 $f(x)$ 在区间 $I$ 上的原函数，虽有 $\int f(x)\,dx=F(x)+C_1$ 和 $\int f(x)\,dx=H(x)+C_2$，但 $F(x)=H(x)$ 不一定成立，因为常数 $C$ 一般是不相同的。

> *二者在概念上存在较大的差异：**后者是个无限集，前者是后者中的一个元素。***

### 3. 不定积分与微分的关系

**口诀：先积后微，形式不变；先微后积，相差一个常数。**

1. $\left[\int f(x)\,dx\right]'=f(x)$ 或 $d\int f(x)\,dx=f(x)\,dx$ **（先积后微，形式不变）**
2. $\int F'(x)\,dx=F(x)+C$ 或 $\int dF(x)=F(x)+C$ **（先微后积，相差个常数）**

## 二、常用不定积分公式（基本积分公式）

> 这一板块非常重要！

| 编号 | 公式 |
|------|------|
| 1-① | $\displaystyle\int k\,dx=kx+C$（$k$ 是常数） |
| 1-② | $\displaystyle\int x^k\,dx=\frac{1}{k+1}x^{k+1}+C$（$k\ne -1$） |
| 1-③ | $\displaystyle\int \frac{1}{x}\,dx=\ln\|x\|+C$ |
| 1-④ | $\displaystyle\int e^x\,dx=e^x+C$；$\displaystyle\int a^x\,dx=\frac{a^x}{\ln a}+C$（$a>0,\,a\ne1$） |
| 1-⑤ | $\displaystyle\int \sin x\,dx=-\cos x+C$；$\displaystyle\int \cos x\,dx=\sin x+C$ |
| | $\displaystyle\int \tan x\,dx=-\ln\|\cos x\|+C$；$\displaystyle\int \cot x\,dx=\ln\|\sin x\|+C$ |
| 1-⑥ | $\displaystyle\int \sec x\,dx=\ln\|\sec x+\tan x\|+C$；$\displaystyle\int \csc x\,dx=\ln\|\csc x-\cot x\|+C$ |
| 1-⑦ | $\displaystyle\int \sec^2 x\,dx=\tan x+C$；$\displaystyle\int \csc^2 x\,dx=-\cot x+C$ |
| 1-⑧ | $\displaystyle\int \sec x\tan x\,dx=\sec x+C$；$\displaystyle\int \csc x\cot x\,dx=-\csc x+C$ |
| 1-⑨ | $\displaystyle\int \frac{1}{x^2+a^2}\,dx=\frac{1}{a}\arctan\frac{x}{a}+C$（$a>0$） |
| 1-⑩ | $\displaystyle\int \frac{1}{\sqrt{a^2-x^2}}\,dx=\arcsin\frac{x}{a}+C$（$a>0$） |
| 1-⑪ | $\displaystyle\int \frac{1}{\sqrt{x^2+a^2}}\,dx=\ln(x+\sqrt{x^2+a^2})+C$ |
| | $\displaystyle\int \frac{1}{\sqrt{x^2-a^2}}\,dx=\ln\left\|x+\sqrt{x^2-a^2}\right\|+C$（$\|x\|>\|a\|$） |
| 1-⑫ | $\displaystyle\int \frac{1}{x^2-a^2}\,dx=\frac{1}{2a}\ln\left\|\frac{x-a}{x+a}\right\|+C$ |
| 1-⑬ | $\displaystyle\int \sin^2 x\,dx=\frac{x}{2}-\frac{\sin 2x}{4}+C$；$\displaystyle\int \cos^2 x\,dx=\frac{x}{2}+\frac{\sin 2x}{4}+C$ |
| 1-⑭ | $\displaystyle\int \sqrt{x^2+a^2}\,dx=\frac{x}{2}\sqrt{x^2+a^2}+\frac{a^2}{2}\ln(x+\sqrt{x^2+a^2})+C$ |
| | $\displaystyle\int \sqrt{a^2-x^2}\,dx=\frac{x}{2}\sqrt{a^2-x^2}+\frac{a^2}{2}\arcsin\frac{x}{a}+C$ |

> **重要求导公式：**
> - $\left[\ln(x+\sqrt{x^2+a^2})\right]'=\frac{1}{\sqrt{x^2+a^2}}$
> - $\left[\ln\left\|x+\sqrt{x^2-a^2}\right\|\right]'=\frac{1}{\sqrt{x^2-a^2}}$（$\|x\|>\|a\|$）

## 三、常用不定积分计算方法

### （一）凑微分法（第一类换元积分）

**基本思想：**

$$\int f[\varphi(x)]\varphi'(x)\,dx=\int f[\varphi(x)]\,d[\varphi(x)]=\int f(u)\,du, \quad u=\varphi(x)$$

**说明：** 当被积函数有一部分比较复杂时，可以通过观察把某些函数放到 $d$ 的后面，使得 $d$ 后面的函数与前面复杂的被积函数 **具有相似的结构**，最后运用基本积分公式将其求出。

**例1：** 计算 $\displaystyle\int x^{-2}\sin\frac{2}{x}\,dx$

**解：**

$$=-\int \sin\frac{2}{x}\,d\left(\frac{1}{x}\right)=-\frac{1}{2}\int \sin\frac{2}{x}\,d\left(\frac{2}{x}\right)=\frac{1}{2}\cos\frac{2}{x}+C$$

**常见凑微分公式总结：**

| 编号 | 公式 |
|------|------|
| 2-① | $\displaystyle\int f(ax+b)\,dx=\frac{1}{a}\int f(ax+b)\,d(ax+b)$ |
| 2-② | $\displaystyle\int f(ax^2+b)x\,dx=\frac{1}{2a}\int f(ax^2+b)\,d(ax^2+b)$ |
| 2-③ | $\displaystyle\int f(ax^n+b)x^{n-1}\,dx=\frac{1}{na}\int f(ax^n+b)\,d(ax^n+b)$ |
| 2-④ | $\displaystyle\int f\!\left(\frac{1}{x}\right)\frac{1}{x^2}\,dx=-\int f\!\left(\frac{1}{x}\right)d\!\left(\frac{1}{x}\right)$ |
| 2-⑤ | $\displaystyle\int f(\sqrt{x})\frac{1}{\sqrt{x}}\,dx=2\int f(\sqrt{x})\,d(\sqrt{x})$ |
| 2-⑥ | $\displaystyle\int f(\ln x)\frac{1}{x}\,dx=\int f(\ln x)\,d(\ln x)$ |
| 2-⑦ | $\displaystyle\int f(e^{ax})e^{ax}\,dx=\frac{1}{a}\int f(e^{ax})\,d(e^{ax})$ |
| 2-⑧ | $\displaystyle\int f(\sin x)\cos x\,dx=\int f(\sin x)\,d(\sin x)$ |
| 2-⑨ | $\displaystyle\int f(\cos x)\sin x\,dx=-\int f(\cos x)\,d(\cos x)$ |
| 2-⑩ | $\displaystyle\int f(\tan x)\sec^2 x\,dx=\int f(\tan x)\,d(\tan x)$ |
| 2-⑪ | $\displaystyle\int f(\sec x)\sec x\tan x\,dx=\int f(\sec x)\,d(\sec x)$ |
| 2-⑫ | $\displaystyle\int f(\arctan x)\frac{1}{1+x^2}\,dx=\int f(\arctan x)\,d(\arctan x)$ |
| 2-⑬ | $\displaystyle\int f(\arcsin x)\frac{1}{\sqrt{1-x^2}}\,dx=\int f(\arcsin x)\,d(\arcsin x)$ |
| 2-⑯ | $\displaystyle\int \frac{f'(x)}{f(x)}\,dx=\ln\|f(x)\|+C$ |

> 补充：$(1\pm\frac{1}{x^2})\,dx=d(x\mp\frac{1}{x})$；$(1+\ln x)\,dx=d(x\ln x)$

### （二）换元法（第二类换元积分）

**基本思想：**

$$\int f(x)\,dx=\int f[\varphi(u)]\varphi'(u)\,du, \quad x=\varphi(u)$$

#### ① 三角函数代换

当被积函数含有以下根式时（$a>0$）：

| 根式形式 | 换元 | 关系 |
|----------|------|------|
| $\sqrt{a^2-x^2}$ | $x=a\sin u$ | $1-\sin^2 u=\cos^2 u$ |
| $\sqrt{a^2+x^2}$ | $x=a\tan u$ | $1+\tan^2 u=\sec^2 u$ |
| $\sqrt{x^2-a^2}$ | $x=a\sec u$ | $\sec^2 u-1=\tan^2 u$ |

> 某些根式 $\sqrt{Ax^2+Bx+C}$ 可通过配方后恒等变形化为以上三种模型。

#### ② 根式代换

- $\displaystyle\int R(x,\sqrt[n]{ax+b})\,dx$：令 $\sqrt[n]{ax+b}=t$
- $\displaystyle\int R\!\left(x,\sqrt[n]{\frac{ax+b}{cx+d}}\right)dx$：令 $\sqrt[n]{\frac{ax+b}{cx+d}}=t$
- $\displaystyle\int R(x,\sqrt[n]{ae^{bx}+c})\,dx$：令 $\sqrt[n]{ae^{bx}+c}=t$
- 若同时含 $\sqrt[n]{ax+b}$ 和 $\sqrt[m]{ax+b}$，令 $\sqrt[l]{ax+b}=t$（$l=\text{lcm}(m,n)$）

#### ③ 倒代换

分母的次数比分子的次数高2次及以上时，可令 $x=\frac{1}{t}$。

**例：** $\displaystyle\int \frac{dx}{x(x^8+2)}$

令 $x=\frac{1}{t}$，得 $=\frac{1}{16}\ln\!\left(\frac{x^8}{x^8+2}\right)+C$

#### ④ 复杂函数直接代换

被积函数含 $e^x$、$\ln x$、$\arctan x$、$\arcsin x$ 等时，可令其为 $t$。若与 $P_n(x)$ 或 $e^{ax}$ 作乘法，优先考虑分部积分法。

#### ⑤ 万能代换

对于 $\int R(\sin x,\cos x)\,dx$，令 $t=\tan\frac{x}{2}$：

$$\sin x=\frac{2t}{1+t^2},\quad \cos x=\frac{1-t^2}{1+t^2},\quad dx=\frac{2}{1+t^2}\,dt$$

#### ⑥ 三角函数的几种变换

对于 $\int f(\sin x,\cos x)\,dx$ 型积分：

- 若 $f(-\sin x,\cos x)=-f(\sin x,\cos x)$，凑 $d(\cos x)$
- 若 $f(\sin x,-\cos x)=-f(\sin x,\cos x)$，凑 $d(\sin x)$
- 若 $f(-\sin x,-\cos x)=f(\sin x,\cos x)$，凑 $d(\tan x)$

#### ⑦ 欧拉（Euler）变换

- 当 $a>0$ 时，令 $\sqrt{ax^2+bx+c}=t\pm\sqrt{a}\,x$
- 当 $c>0$ 时，令 $\sqrt{ax^2+bx+c}=xt\pm\sqrt{c}$
- 当 $ax^2+bx+c=a(x-\lambda)(x-\mu)$ 时，令 $\sqrt{ax^2+bx+c}=t(x-\lambda)$

#### ⑧ 积化和差

对于 $\int \sin mx\cdot\sin nx\,dx$、$\int \sin mx\cdot\cos nx\,dx$、$\int \cos mx\cdot\cos nx\,dx$（$m\ne n$），可利用积化和差公式计算。

### （三）分部积分法

**基本公式：** $\displaystyle\int u\,dv=uv-\int v\,du$

**口诀：反、对、幂、三、指（指、三），谁在前，谁不动；谁在后，d进去。**

> 顺序：反三角函数 > 对数函数 > 幂函数 > 三角函数 > 指数函数

**三种类型：**

1. **常规型** — 幂函数与三角函数/指数函数乘积，表格法直接求解
2. **循环型** — 指数函数与三角函数乘积，分部积分两次后出现循环，解方程求得
3. **变通型** — 反三角函数与幂函数乘积，需对表格列进行调配

**例（循环型）：** $\displaystyle\int e^{-2x}\sin\frac{x}{2}\,dx$

$$=-\frac{2}{17}e^{-2x}\!\left(\cos\frac{x}{2}+4\sin\frac{x}{2}\right)+C$$

## 四、不定(定)积分计算特殊方法

### （一）抵消型不定积分

将被积函数拆成两项，对其中一项运用分部积分法，分部计算出的结果抵消掉另一项。

**例：** $\displaystyle\int \frac{2x+1}{x^2}e^{-2x}\,dx$

$$=\int\!\left(\frac{2}{x}+\frac{1}{x^2}\right)e^{-2x}dx=-\frac{e^{-2x}}{x}+C$$

**例：** $\displaystyle\int \frac{x+\sin x}{1+\cos x}\,dx=\frac{x\sin x}{1+\cos x}+C$

**例：** $\displaystyle\int e^x\!\left(\frac{1-x}{1+x^2}\right)^2 dx=\frac{e^x}{1+x^2}+C$

---

### （二）求导观察型不定积分

当被积函数为 $f(x)/g(x)$，若对分母的一部分求导可得分子的常数倍或函数倍，从而凑微分。

**例：** $\displaystyle\int \frac{(\cos^2 x-\sin x)e^{\sin x}}{\cos x\cdot e^{\sin x}(1+\cos x\cdot e^{\sin x})}\,dx$

$\because\, [\cos x\cdot e^{\sin x}]'=(\cos^2 x-\sin x)e^{\sin x}$

$$=\ln\left\|\frac{\cos x\cdot e^{\sin x}}{1+\cos x\cdot e^{\sin x}}\right\|+C$$

---

### （三）区间再现计算型（定积分）

**基本思想：** $\displaystyle\int_a^b f(x)\,dx=\int_a^b f(a+b-x)\,dx$

**华里士公式：**

$$I_n=\int_0^{\pi/2}\sin^n x\,dx=\int_0^{\pi/2}\cos^n x\,dx$$

- $n$ 为正奇数：$I_n=\dfrac{(n-1)!!}{n!!}$
- $n$ 为正偶数：$I_n=\dfrac{(n-1)!!}{n!!}\cdot\dfrac{\pi}{2}$

**常用结论：**

| 编号 | 公式 |
|------|------|
| ②-1 | $\displaystyle\int_0^{\pi}xf(\sin x)\,dx=\frac{\pi}{2}\int_0^{\pi}f(\sin x)\,dx$ |
| ②-2 | $\displaystyle\int_0^{\pi/2}f(\sin x)\,dx=\int_0^{\pi/2}f(\cos x)\,dx$ |
| ③-1 | $\displaystyle\int_{-a}^{a}f(x)\,dx=\int_0^{a}[f(x)+f(-x)]\,dx$ |
| ③-2 | $\displaystyle\int_a^b f(x)\,dx=\frac{1}{2}\int_a^b[f(x)+f(a+b-x)]\,dx$ |
| ③-3 | $\displaystyle\int_a^b f(x)\,dx=\int_a^{(a+b)/2}[f(x)+f(a+b-x)]\,dx$ |

---

### （四）分式三角函数型

对于 $\displaystyle\int\frac{A\sin x+B\cos x}{D\sin x+E\cos x}\,dx$，将分子拆成：

$$\frac{A\sin x+B\cos x}{D\sin x+E\cos x}=p+q\cdot\frac{D\cos x-E\sin x}{D\sin x+E\cos x}$$

用待定系数法求出 $p$、$q$，则：

$$\int\frac{A\sin x+B\cos x}{D\sin x+E\cos x}\,dx=px+q\ln\|D\sin x+E\cos x\|+C$$

**例：** $\displaystyle\int\frac{3\sin x-2\cos x}{\sin x+2\cos x}\,dx$

解得 $p=-\frac{1}{5}$，$q=-\frac{8}{5}$：

$$=-\frac{1}{5}x-\frac{8}{5}\ln\|\sin x+2\cos x\|+C$$
