## 二、基本概念

### 2.1 阶

微分方程中未知函数的**最高阶导数的阶数**。

### 2.2 通解与特解

| 名称 | 定义 |
|------|------|
| **通解** | 含独立任意常数（个数 = 阶数）的解 |
| **特解** | 不含任意常数的解（由初始条件确定）|

### 2.3 初始条件

定解条件：$y|_{x=x_0} = y_0$、$y'|_{x=x_0} = y_1$ 等

## 三、一阶微分方程

### 3.1 变量可分离 ⚠️ 必考

**【形式】** $f(x) g(y) dx + h(x) p(y) dy = 0$

**【解法】** 变量分离后两边积分
$$\int \frac{p(y)}{g(y)} dy = -\int \frac{h(x)}{f(x)} dx$$

**【例题7.1】** 求 $y' = xy$ 的通解。

**解**：
$\dfrac{dy}{y} = x \, dx$（$y \neq 0$）

两边积分：$\ln|y| = \dfrac{x^2}{2} + C_1$

$y = C e^{x^2/2}$（$C$ 是任意常数，吸收正负号）

**【例题7.2】** 求 $\dfrac{dy}{dx} = \dfrac{y}{1+x^2}$ 的通解。

**解**：
$\dfrac{dy}{y} = \dfrac{dx}{1+x^2}$

$\ln|y| = \arctan x + C$

$y = C e^{\arctan x}$

### 3.2 齐次方程

**【形式】** $y' = \varphi\left(\dfrac{y}{x}\right)$

**【解法】** 令 $u = \dfrac{y}{x}$，则 $y = ux$，$y' = u + xu'$

代入：$u + xu' = \varphi(u)$，化简得：
$$\frac{du}{\varphi(u) - u} = \frac{dx}{x}$$

**【例题7.3】** 求 $y' = \dfrac{y}{x} + \tan\dfrac{y}{x}$ 的通解。

**解**：
令 $u = y/x$，$y' = u + xu'$

$u + xu' = u + \tan u \Rightarrow xu' = \tan u \Rightarrow \cot u \, du = \dfrac{dx}{x}$

积分：$\ln|\sin u| = \ln|x| + C$

$\sin u = Cx$（$C$ 是任意常数）

$\sin\dfrac{y}{x} = Cx$

### 3.3 一阶线性方程 ⚠️ 必考

**【标准形式】** $y' + P(x) y = Q(x)$

**【公式解】** ⚠️ 必背
$$\boxed{y = e^{-\int P(x) dx} \left[\int Q(x) e^{\int P(x) dx} dx + C\right]}$$

**【记忆】** $y = e^{-\int P} \left[\int Q e^{\int P} dx + C\right]$

**【例题7.4】** 求 $y' + 2xy = x$ 的通解。

**解**：
$P(x) = 2x$，$Q(x) = x$

$\int P \, dx = x^2$，$e^{\int P} = e^{x^2}$，$e^{-\int P} = e^{-x^2}$

$y = e^{-x^2}\left[\int x e^{x^2} dx + C\right] = e^{-x^2}\left[\frac{e^{x^2}}{2} + C\right] = \frac{1}{2} + Ce^{-x^2}$

**【例题7.5】** 求 $xy' + y = x$ 的通解。

**解**：
化为标准形式：$y' + \dfrac{1}{x} y = 1$

$P = \dfrac{1}{x}$，$Q = 1$

$\int P \, dx = \ln|x|$，$e^{\int P} = |x|$

设 $x > 0$：$y = \dfrac{1}{x}\left[\int 1 \cdot x \, dx + C\right] = \dfrac{1}{x}\left[\dfrac{x^2}{2} + C\right] = \dfrac{x}{2} + \dfrac{C}{x}$

### 3.4 伯努利方程

**【形式】** $y' + P(x) y = Q(x) y^n$（$n \neq 0, 1$）

**【解法】** 令 $z = y^{1-n}$，原方程化为：
$$z' + (1-n) P(x) z = (1-n) Q(x)$$

## 四、可降阶的高阶微分方程

### 4.1 $y^{(n)} = f(x)$

**【解法】** 连续积分 $n$ 次

**【例题7.6】** 求 $y'' = \sin x$ 的通解。

**解**：
$y' = -\cos x + C_1$
$y = -\sin x + C_1 x + C_2$

### 4.2 $y'' = f(x, y')$（缺 $y$）

**【解法】** 令 $p = y'$，则 $y'' = p'$，方程化为 $p' = f(x, p)$，这是关于 $p$ 的一阶方程

**【例题7.7】** 求 $xy'' = y'$ 的通解。

**解**：
令 $p = y'$，$p' = \dfrac{1}{x} p$

$\dfrac{dp}{p} = \dfrac{dx}{x}$

$\ln|p| = \ln|x| + C_1 \Rightarrow p = C_1 x$

$y' = C_1 x \Rightarrow y = \dfrac{C_1}{2} x^2 + C_2$

### 4.3 $y'' = f(y, y')$（缺 $x$）

**【解法】** 令 $p = y'$，$y'' = p \dfrac{dp}{dy}$，方程化为 $p \dfrac{dp}{dy} = f(y, p)$

## 五、二阶常系数线性微分方程 ⚠️ 必考

### 5.1 形式

$$y'' + py' + qy = f(x)$$

其中 $p, q$ 是常数。

- $f(x) = 0$：齐次方程
- $f(x) \neq 0$：非齐次方程

### 5.2 二阶常系数齐次方程

**【特征方程】** $r^2 + pr + q = 0$

设特征根为 $r_1, r_2$：

| 特征根情况 | 通解形式 |
|-----------|---------|
| $r_1 \neq r_2$（两不等实根）| $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| $r_1 = r_2 = r$（两相等实根）| $y = (C_1 + C_2 x) e^{rx}$ |
| $r_{1,2} = \alpha \pm \beta i$（共轭复根）| $y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$ |

**【例题7.8】** 求 $y'' - 3y' + 2y = 0$ 的通解。

**解**：
特征方程 $r^2 - 3r + 2 = 0$，$(r-1)(r-2) = 0$，$r_1 = 1, r_2 = 2$

通解：$y = C_1 e^x + C_2 e^{2x}$

**【例题7.9】** 求 $y'' - 4y' + 4y = 0$ 的通解。

**解**：
特征方程 $r^2 - 4r + 4 = 0$，$(r-2)^2 = 0$，$r_1 = r_2 = 2$

通解：$y = (C_1 + C_2 x) e^{2x}$

**【例题7.10】** 求 $y'' + 4y' + 5y = 0$ 的通解。

**解**：
特征方程 $r^2 + 4r + 5 = 0$，$r = \dfrac{-4 \pm \sqrt{16-20}}{2} = -2 \pm i$

通解：$y = e^{-2x}(C_1 \cos x + C_2 \sin x)$

### 5.3 二阶常系数非齐次方程

**【通解结构】** $y = \bar{y}(x) + y^*(x)$

- $\bar{y}(x)$：对应齐次方程的通解
- $y^*(x)$：非齐次方程的特解

**【特解形式】**（待定系数法）

| $f(x)$ 形式 | 特解形式 |
|-------------|---------|
| $P_n(x)$（多项式）| $Q_n(x) x^k$（同次多项式）|
| $e^{\alpha x} P_n(x)$ | $x^k e^{\alpha x} Q_n(x)$ |
| $e^{\alpha x} P_n(x) \cos\beta x$ 或 $e^{\alpha x} P_n(x) \sin\beta x$ | $x^k e^{\alpha x}(Q_n^{(1)} \cos\beta x + Q_n^{(2)} \sin\beta x)$ |

其中 $k$ 由 $\alpha + \beta i$ 是否是特征根决定：
- 不是特征根：$k = 0$
- 是单特征根：$k = 1$
- 是二重特征根：$k = 2$

**【例题7.11】** 求 $y'' - 2y' - 3y = 3x + 1$ 的通解。

**解**：
特征方程 $r^2 - 2r - 3 = 0$，$(r-3)(r+1) = 0$，$r_1 = 3, r_2 = -1$

齐次通解：$\bar{y} = C_1 e^{3x} + C_2 e^{-x}$

$f(x) = 3x + 1$，$\alpha = 0$ 不是特征根，设特解 $y^* = ax + b$

$y^{*\prime} = a$，$y^{*\prime\prime} = 0$

代入：$0 - 2a - 3(ax + b) = 3x + 1$

$-3ax + (-2a - 3b) = 3x + 1$

$\begin{cases} -3a = 3 \\ -2a - 3b = 1 \end{cases}$

$a = -1$，$2 - 3b = 1$，$b = \dfrac{1}{3}$

$y^* = -x + \dfrac{1}{3}$

通解：$y = C_1 e^{3x} + C_2 e^{-x} - x + \dfrac{1}{3}$

**【例题7.12】** 求 $y'' - 2y' + y = e^x$ 满足 $y(0) = 1, y'(0) = 0$ 的特解。

**解**：
特征方程 $r^2 - 2r + 1 = 0$，$(r-1)^2 = 0$，$r_1 = r_2 = 1$

齐次通解：$\bar{y} = (C_1 + C_2 x) e^x$

$f(x) = e^x$，$\alpha = 1$ 是二重特征根，所以 $k = 2$

设 $y^* = ax^2 e^x$

$y^{*\prime} = (2ax + ax^2)e^x$
$y^{*\prime\prime} = (2a + 4ax + ax^2)e^x$

代入：$(2a + 4ax + ax^2 - 4ax - 2ax^2 + ax^2)e^x = e^x$

$2a = 1$，$a = \dfrac{1}{2}$

$y^* = \dfrac{1}{2} x^2 e^x$

通解：$y = (C_1 + C_2 x) e^x + \dfrac{1}{2} x^2 e^x$

由 $y(0) = 1$：$C_1 = 1$

由 $y'(0) = 0$：$(C_1 + C_2 x)e^x + (C_2 + x)e^x$ 在 $x=0$：$C_1 + C_2 = 0$，$C_2 = -1$

特解：$y = (1 - x) e^x + \dfrac{1}{2} x^2 e^x$

## 六、综合应用

### 6.1 物理应用

**【例题7.13】** 一曲线过 $(1, 1)$，曲线上任一点 $M(x, y)$ 处的切线斜率为 $2x$，求曲线方程。

**解**：
$y' = 2x$
$y = x^2 + C$
由 $y(1) = 1$：$1 = 1 + C$，$C = 0$

曲线方程：$y = x^2$

### 6.2 增长率模型

**【例题7.14】** 物体冷却速度与温差成正比，初始温差 $T_0$，求温差随时间变化规律。

**解**：
$\dfrac{dT}{dt} = -kT$（$k > 0$）

$T = T_0 e^{-kt}$

### 6.3 RLC 电路

RLC 串联电路方程：
$$L \frac{d^2 i}{dt^2} + R \frac{di}{dt} + \frac{i}{C} = E(t)$$

## 七、配套练习（10题精练）

### 练习1
求 $\dfrac{dy}{dx} = \dfrac{x}{y}$ 的通解。
<details>
<summary>🔑 答案</summary>

$y \, dy = x \, dx$
$\dfrac{y^2}{2} = \dfrac{x^2}{2} + C_1$
$y^2 = x^2 + C$（$C = 2C_1$）

</details>

### 练习2
求 $\dfrac{dy}{dx} + y = e^{-x}$ 的通解。
<details>
<summary>🔑 答案</summary>

$P = 1$，$Q = e^{-x}$
$\int P \, dx = x$，$e^x$ 是积分因子

$y = e^{-x}[\int e^{-x} \cdot e^x dx + C] = e^{-x}[x + C]$

</details>

### 练习3
求 $y' = \dfrac{y}{x} + \dfrac{x}{y}$ 的通解。
<details>
<summary>🔑 答案</summary>

这是齐次方程。令 $u = y/x$，$y' = u + xu'$

$u + xu' = u + \dfrac{1}{u}$
$xu' = \dfrac{1}{u}$
$u \, du = \dfrac{dx}{x}$

$\dfrac{u^2}{2} = \ln|x| + C$
$\dfrac{y^2}{x^2} = 2\ln|x| + C$

</details>

### 练习4
求 $y'' - 5y' + 6y = 0$ 的通解。
<details>
<summary>🔑 答案</summary>

$r^2 - 5r + 6 = 0$，$(r-2)(r-3) = 0$，$r_1 = 2, r_2 = 3$

$y = C_1 e^{2x} + C_2 e^{3x}$

</details>

### 练习5
求 $y'' - 4y' + 4y = e^{2x}$ 的通解。
<details>
<summary>🔑 答案</summary>

特征方程 $(r-2)^2 = 0$，$r = 2$（重根）
齐次通解：$y = (C_1 + C_2 x) e^{2x}$

$\alpha = 2$ 是二重特征根，$k = 2$，设 $y^* = ax^2 e^{2x}$

代入：$y^{*\prime} = (2ax + 2ax^2)e^{2x}$
$y^{*\prime\prime} = (2a + 8ax + 4ax^2)e^{2x}$

代入：$(2a + 8ax + 4ax^2 - 8ax - 8ax^2 + 4ax^2)e^{2x} = 2ae^{2x} = e^{2x}$

$a = \dfrac{1}{2}$

通解：$y = (C_1 + C_2 x) e^{2x} + \dfrac{1}{2} x^2 e^{2x}$

</details>

### 练习6
求 $y'' + y = \sin x$ 的通解。
<details>
<summary>🔑 答案</summary>

特征方程 $r^2 + 1 = 0$，$r = \pm i$
齐次通解：$y = C_1 \cos x + C_2 \sin x$

$\alpha = 0, \beta = 1$，$\alpha + \beta i = i$ 是特征根，$k = 1$

设 $y^* = x(a \cos x + b \sin x)$

代入... 详细计算后得 $y^* = -\dfrac{1}{2} x \cos x$

通解：$y = C_1 \cos x + C_2 \sin x - \dfrac{1}{2} x \cos x$

</details>

### 练习7
求 $y'' - 2y' + y = 0$ 满足 $y(0) = 1, y'(0) = 2$ 的特解。
<details>
<summary>🔑 答案</summary>

特征方程 $(r-1)^2 = 0$，$r = 1$（重根）
通解：$y = (C_1 + C_2 x) e^x$

$y(0) = C_1 = 1$
$y' = C_2 e^x + (C_1 + C_2 x) e^x = (C_1 + C_2 + C_2 x) e^x$
$y'(0) = C_1 + C_2 = 2$，$C_2 = 1$

特解：$y = (1 + x) e^x$

</details>

### 练习8
一曲线过原点，曲线上任一点的切线斜率等于 $2x + y$，求曲线方程。
<details>
<summary>🔑 答案</summary>

$y' = 2x + y$
$y' - y = 2x$

$P = -1, Q = 2x$

$y = e^x \left[\int 2x e^{-x} dx + C\right]$

$\int 2x e^{-x} dx = -2x e^{-x} - 2e^{-x} = -2(x+1)e^{-x}$

$y = e^x \left[-2(x+1)e^{-x} + C\right] = -2(x+1) + Ce^x$

由 $y(0) = 0$：$-2 + C = 0$，$C = 2$

曲线：$y = 2e^x - 2x - 2$

</details>

### 练习9
求 $y'' = x$ 的通解。
<details>
<summary>🔑 答案</summary>

$y' = \dfrac{x^2}{2} + C_1$

$y = \dfrac{x^3}{6} + C_1 x + C_2$

</details>

### 练习10
求 $\dfrac{dy}{dx} = \dfrac{y}{x} - 1$ 的通解。
<details>
<summary>🔑 答案</summary>

齐次方程。$y' = \dfrac{y}{x} - 1$

令 $u = y/x$，$y' = u + xu'$

$u + xu' = u - 1 \Rightarrow xu' = -1 \Rightarrow du = -\dfrac{dx}{x}$

$u = -\ln|x| + C$

$\dfrac{y}{x} = -\ln|x| + C$

$y = -x \ln|x| + Cx$

</details>
