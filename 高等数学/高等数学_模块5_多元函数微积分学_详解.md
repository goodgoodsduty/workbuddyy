## 二、二元函数基础

### 2.1 定义

设 $D$ 是平面上的一个区域，若对 $D$ 内每一点 $(x, y)$，都按某对应法则 $f$ 有唯一确定的 $z$ 与之对应，则称 $z$ 是 $x, y$ 的二元函数，记为 $z = f(x, y)$。

### 2.2 极限

$$\lim_{(x,y) \to (x_0, y_0)} f(x, y) = A$$

**【重要】** 二元函数极限要求从**所有路径**趋近都等于 $A$。

**【例题5.1】** 证明 $\lim_{(x,y) \to (0,0)} \dfrac{xy}{x^2 + y^2}$ 不存在。

**解**：
沿 $y = kx$ 趋近：
$$\lim_{x \to 0} \frac{x \cdot kx}{x^2 + k^2 x^2} = \frac{k}{1 + k^2}$$

结果与 $k$ 有关（$k=0$ 时为 0，$k=1$ 时为 $\frac{1}{2}$），所以极限不存在。

### 2.3 连续

如果 $\lim_{(x,y) \to (x_0, y_0)} f(x, y) = f(x_0, y_0)$，则 $f$ 在 $(x_0, y_0)$ 连续。

## 三、偏导数 ⚠️

### 3.1 定义

**对 $x$ 的偏导数**（$y$ 当常数）：
$$f_x(x_0, y_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x}$$

**对 $y$ 的偏导数**（$x$ 当常数）：
$$f_y(x_0, y_0) = \lim_{\Delta y \to 0} \frac{f(x_0, y_0 + \Delta y) - f(x_0, y_0)}{\Delta y}$$

**【关键】** 求偏导数 = 把另一个变量当常数，按一元函数求导。

### 3.2 偏导数记号

$$f_x = f'_x = \frac{\partial f}{\partial x} = \frac{\partial z}{\partial x}$$

### 3.3 几何意义

$f_x(x_0, y_0)$ = 曲面 $z = f(x, y)$ 与平面 $y = y_0$ 的交线在 $(x_0, y_0, f(x_0, y_0))$ 处的切线斜率

### 3.4 高阶偏导数

$$f_{xx} = \frac{\partial^2 z}{\partial x^2} = \frac{\partial}{\partial x}\left(\frac{\partial z}{\partial x}\right)$$

**【定理】** 若 $f_{xy}$ 和 $f_{yx}$ 在区域 $D$ 内连续，则 $f_{xy} = f_{yx}$（混合偏导数与求导顺序无关）

**【例题5.2】** 设 $z = x^3 y^2 - 3xy + 2$，求 $f_{xy}$。

**解**：
$f_x = 3x^2 y^2 - 3y$
$f_{xy} = 6x^2 y - 3$

## 四、全微分 ⚠️ 必考

### 4.1 定义

若 $\Delta z = f(x + \Delta x, y + \Delta y) - f(x, y) = A \Delta x + B \Delta y + o(\rho)$

其中 $\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}$，则 $A \Delta x + B \Delta y$ 称为**全微分**，记为 $dz$。

**【定理】** 可微 $\Rightarrow$ 偏导数存在，$A = f_x$，$B = f_y$

$$dz = \frac{\partial z}{\partial x} dx + \frac{\partial z}{\partial y} dy$$

### 4.2 可微的判别

| 关系 | 说明 |
|------|------|
| 可微 ⇒ 偏导数存在 | ✓ |
| 可微 ⇒ 连续 | ✓ |
| 偏导数存在 ⇒ 可微 | ✗（反例：$f(x,y) = \sqrt{|xy|}$ 在 $(0,0)$）|
| **偏导数连续 ⇒ 可微** | ✓（充分条件）|
| 偏导数连续 ⇒ 偏导数存在 | ✓ |

### 4.3 全微分计算

**【例题5.3】** 求 $z = x^2 y + \sin(xy)$ 的全微分。

**解**：
$\dfrac{\partial z}{\partial x} = 2xy + y\cos(xy)$
$\dfrac{\partial z}{\partial y} = x^2 + x\cos(xy)$

$dz = (2xy + y\cos xy)dx + (x^2 + x\cos xy)dy$

## 五、复合函数求导（链式法则）⚠️ 必考

### 5.1 中间变量为一元函数

设 $z = f(u, v)$，$u = \varphi(t)$，$v = \psi(t)$，则：
$$\frac{dz}{dt} = \frac{\partial z}{\partial u} \cdot \frac{du}{dt} + \frac{\partial z}{\partial v} \cdot \frac{dv}{dt}$$

### 5.2 中间变量为多元函数

设 $z = f(u, v)$，$u = \varphi(x, y)$，$v = \psi(x, y)$，则：
$$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u} \cdot \frac{\partial u}{\partial x} + \frac{\partial z}{\partial v} \cdot \frac{\partial v}{\partial x}$$

$$\frac{\partial z}{\partial y} = \frac{\partial z}{\partial u} \cdot \frac{\partial u}{\partial y} + \frac{\partial z}{\partial v} \cdot \frac{\partial v}{\partial y}$$

### 5.3 链式法则图

```
         z
        / \
       u   v
      /|   |\
     x  ...  y
```

求 $\dfrac{\partial z}{\partial x}$ 时，沿 $z \to u \to x$ 和 $z \to v \to x$ 两条路径求和。

**【例题5.4】** 设 $z = u^2 v$，$u = x + y$，$v = xy$，求 $\dfrac{\partial z}{\partial x}$。

**解**：
$\dfrac{\partial z}{\partial u} = 2uv$，$\dfrac{\partial z}{\partial v} = u^2$
$\dfrac{\partial u}{\partial x} = 1$，$\dfrac{\partial v}{\partial x} = y$

$\dfrac{\partial z}{\partial x} = 2uv \cdot 1 + u^2 \cdot y = 2uv + yu^2 = 2(x+y)xy + y(x+y)^2$

## 六、隐函数求偏导 ⚠️ 必考

### 6.1 公式法

设 $F(x, y, z) = 0$ 确定 $z = f(x, y)$：
$$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}$$

**【例题5.5】** 设 $e^z + xyz + 1 = 0$ 确定 $z = f(x, y)$，求 $\dfrac{\partial z}{\partial x}$。

**解**：
$F = e^z + xyz + 1$
$F_x = yz$，$F_y = xz$，$F_z = e^z + xy$

$\dfrac{\partial z}{\partial x} = -\dfrac{yz}{e^z + xy}$

## 七、多元函数极值 ⚠️ 必考

### 7.1 无条件极值

**【必要条件】** $f$ 在 $(x_0, y_0)$ 可微且取极值，则 $f_x(x_0, y_0) = 0$ 且 $f_y(x_0, y_0) = 0$

**【充分条件】** 设 $f_x(x_0, y_0) = 0$，$f_y(x_0, y_0) = 0$，记：

| 记号 | 含义 |
|------|------|
| $A = f_{xx}(x_0, y_0)$ | - |
| $B = f_{xy}(x_0, y_0)$ | - |
| $C = f_{yy}(x_0, y_0)$ | - |
| $\Delta = AC - B^2$ | - |

| 条件 | 结论 |
|------|------|
| $\Delta > 0, A > 0$ | 极小值 |
| $\Delta > 0, A < 0$ | 极大值 |
| $\Delta < 0$ | 不是极值 |
| $\Delta = 0$ | 不确定 |

**【例题5.6】** 求 $z = x^3 + y^3 - 3xy$ 的极值。

**解**：
$z_x = 3x^2 - 3y = 0 \Rightarrow y = x^2$
$z_y = 3y^2 - 3x = 0 \Rightarrow x = y^2$

联立：$x = x^4 \Rightarrow x(x^3 - 1) = 0 \Rightarrow x = 0$ 或 $x = 1$

驻点：$(0, 0)$ 和 $(1, 1)$

$z_{xx} = 6x$，$z_{yy} = 6y$，$z_{xy} = -3$

$(0, 0)$：$A = 0, B = -3, C = 0$，$\Delta = 0 - 9 = -9 < 0$，不是极值

$(1, 1)$：$A = 6, B = -3, C = 6$，$\Delta = 36 - 9 = 27 > 0$，$A > 0$，极小值

极小值 $z(1, 1) = 1 + 1 - 3 = -1$

### 7.2 条件极值（拉格朗日乘数法）

**求 $f(x, y)$ 在 $\varphi(x, y) = 0$ 下的极值：**

构造拉格朗日函数：
$$L(x, y, \lambda) = f(x, y) + \lambda \varphi(x, y)$$

解方程组：
$$\begin{cases} L_x = f_x + \lambda \varphi_x = 0 \\ L_y = f_y + \lambda \varphi_y = 0 \\ L_\lambda = \varphi(x, y) = 0 \end{cases}$$

**【例题5.7】** 求 $z = xy$ 在 $x + y = 1$ 下的最大值。

**解**：
$L = xy + \lambda(x + y - 1)$
$\begin{cases} y + \lambda = 0 \\ x + \lambda = 0 \\ x + y = 1 \end{cases}$

得 $x = y = \dfrac{1}{2}$，$z = \dfrac{1}{4}$

## 八、二重积分 ⚠️ 必考

### 8.1 定义

$$\iint_D f(x, y) d\sigma = \lim_{\lambda \to 0} \sum_{i=1}^n f(\xi_i, \eta_i) \Delta \sigma_i$$

**几何意义**：当 $f \geq 0$ 时，表示以 $D$ 为底、$f$ 为顶的**曲顶柱体体积**。

### 8.2 性质

| 性质 | 公式 |
|------|------|
| 线性 | $\iint_D [\alpha f + \beta g] d\sigma = \alpha \iint_D f d\sigma + \beta \iint_D g d\sigma$ |
| 区域可加 | $D = D_1 + D_2$ 时，$\iint_D = \iint_{D_1} + \iint_{D_2}$ |
| 估值 | $m \cdot A \leq \iint_D f d\sigma \leq M \cdot A$（$A$ 是 $D$ 的面积）|

### 8.3 直角坐标系下计算 ⚠️ 必考

**【X 型区域】** $D: a \leq x \leq b, \varphi_1(x) \leq y \leq \varphi_2(x)$
$$\iint_D f(x,y) d\sigma = \int_a^b dx \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y) dy$$

**【Y 型区域】** $D: c \leq y \leq d, \psi_1(y) \leq x \leq \psi_2(y)$
$$\iint_D f(x,y) d\sigma = \int_c^d dy \int_{\psi_1(y)}^{\psi_2(y)} f(x,y) dx$$

**【关键】** 画图定限！先画积分区域 $D$

**【例题5.8】** 计算 $\iint_D xy \, d\sigma$，其中 $D$ 由 $x = 1, x = 2, y = 0, y = x$ 围成。

**解**：
区域 $D: 1 \leq x \leq 2, 0 \leq y \leq x$

$$\iint_D xy \, d\sigma = \int_1^2 dx \int_0^x xy \, dy = \int_1^2 \left[\frac{xy^2}{2}\right]_0^x dx = \int_1^2 \frac{x^3}{2} dx = \left[\frac{x^4}{8}\right]_1^2 = 2 - \frac{1}{8} = \frac{15}{8}$$

**【交换积分次序】**

**【例题5.9】** 计算 $\int_0^1 dy \int_y^1 \sin x^2 \, dx$。

**解**：
因为 $\sin x^2$ 没有初等原函数，必须换序

原积分区域 $D: 0 \leq y \leq 1, y \leq x \leq 1$

换序后：$0 \leq x \leq 1, 0 \leq y \leq x$

$$= \int_0^1 dx \int_0^x \sin x^2 \, dy = \int_0^1 x \sin x^2 \, dx = \left[-\frac{\cos x^2}{2}\right]_0^1 = \frac{1 - \cos 1}{2}$$

### 8.4 极坐标系下计算 ⚠️ 必考

**适用情形**：被积函数含 $x^2 + y^2$，区域是圆/扇形/环形

**【公式】**
$$\iint_D f(x,y) d\sigma = \int_\alpha^\beta d\theta \int_{\rho_1(\theta)}^{\rho_2(\theta)} f(\rho\cos\theta, \rho\sin\theta) \rho \, d\rho$$

**【例题5.10】** 计算 $\iint_D e^{-x^2-y^2} d\sigma$，其中 $D: x^2 + y^2 \leq 1$。

**解**：
化为极坐标：$0 \leq \theta \leq 2\pi$，$0 \leq \rho \leq 1$

$$= \int_0^{2\pi} d\theta \int_0^1 e^{-\rho^2} \rho d\rho = 2\pi \left[-\frac{e^{-\rho^2}}{2}\right]_0^1 = \pi(1 - e^{-1})$$

### 8.5 选择坐标系的原则

| 区域类型 | 建议坐标系 |
|---------|----------|
| 矩形、直角三角形、直线围成 | 直角坐标 |
| 圆、扇形、环形、与 $x^2+y^2$ 有关 | 极坐标 |

## 九、配套练习（10题精练）

### 练习1
求 $z = x^2 + xy + y^2$ 在 $(1, 1)$ 处的偏导数。
<details>
<summary>🔑 答案</summary>

$z_x = 2x + y = 3$
$z_y = x + 2y = 3$

</details>

### 练习2
求 $z = e^{xy} \sin(x+y)$ 的全微分。
<details>
<summary>🔑 答案</summary>

$z_x = ye^{xy}\sin(x+y) + e^{xy}\cos(x+y) = e^{xy}[y\sin(x+y) + \cos(x+y)]$
$z_y = xe^{xy}\sin(x+y) + e^{xy}\cos(x+y) = e^{xy}[x\sin(x+y) + \cos(x+y)]$

$dz = z_x dx + z_y dy$

</details>

### 练习3
设 $z = u^2 + v^2$，$u = x + y$，$v = x - y$，求 $\dfrac{\partial z}{\partial x}, \dfrac{\partial z}{\partial y}$。
<details>
<summary>🔑 答案</summary>

$z_u = 2u$，$z_v = 2v$
$u_x = 1$，$u_y = 1$，$v_x = 1$，$v_y = -1$

$\dfrac{\partial z}{\partial x} = 2u \cdot 1 + 2v \cdot 1 = 2u + 2v = 2(x+y) + 2(x-y) = 4x$

$\dfrac{\partial z}{\partial y} = 2u \cdot 1 + 2v \cdot (-1) = 2u - 2v = 2(x+y) - 2(x-y) = 4y$

</details>

### 练习4
设 $x^2 + y^2 + z^2 = 3xyz$ 确定 $z = f(x, y)$，求 $\dfrac{\partial z}{\partial x}$。
<details>
<summary>🔑 答案</summary>

$F = x^2 + y^2 + z^2 - 3xyz$
$F_x = 2x - 3yz$
$F_z = 2z - 3xy$

$\dfrac{\partial z}{\partial x} = -\dfrac{2x - 3yz}{2z - 3xy}$

</details>

### 练习5
求 $z = x^2 + 2y^2$ 在 $(1, 1)$ 处的极值。
<details>
<summary>🔑 答案</summary>

$z_x = 2x = 0 \Rightarrow x = 0$
$z_y = 4y = 0 \Rightarrow y = 0$

驻点 $(0, 0)$，且显然 $z(0, 0) = 0$ 是极小值。

</details>

### 练习6
求 $z = x^2 + y^2$ 在 $x + y = 3$ 下的最小值。
<details>
<summary>🔑 答案</summary>

$L = x^2 + y^2 + \lambda(x + y - 3)$
$\begin{cases} 2x + \lambda = 0 \\ 2y + \lambda = 0 \\ x + y = 3 \end{cases}$

$x = y = \dfrac{3}{2}$，$z = 2 \cdot \dfrac{9}{4} = \dfrac{9}{2}$

</details>

### 练习7
计算 $\iint_D (x + y) d\sigma$，$D: 0 \leq x \leq 1, 0 \leq y \leq 1$。
<details>
<summary>🔑 答案</summary>

$$\int_0^1 dx \int_0^1 (x + y) dy = \int_0^1 \left[xy + \frac{y^2}{2}\right]_0^1 dx = \int_0^1 \left(x + \frac{1}{2}\right) dx = \frac{1}{2} + \frac{1}{2} = 1$$

</details>

### 练习8
计算 $\iint_D \sqrt{x^2 + y^2} d\sigma$，$D: x^2 + y^2 \leq 1$。
<details>
<summary>🔑 答案</summary>

极坐标：$0 \leq \theta \leq 2\pi$，$0 \leq \rho \leq 1$

$$= \int_0^{2\pi} d\theta \int_0^1 \rho \cdot \rho d\rho = 2\pi \cdot \frac{1}{3} = \frac{2\pi}{3}$$

</details>

### 练习9
交换积分次序：$\int_0^1 dy \int_y^1 f(x, y) dx$。
<details>
<summary>🔑 答案</summary>

原区域：$0 \leq y \leq 1, y \leq x \leq 1$

换序：$0 \leq x \leq 1, 0 \leq y \leq x$

$$= \int_0^1 dx \int_0^x f(x, y) dy$$

</details>

### 练习10
计算 $\iint_D xy \, d\sigma$，$D: x^2 + y^2 \leq 1, x \geq 0, y \geq 0$（第一象限单位圆）。
<details>
<summary>🔑 答案</summary>

极坐标：$0 \leq \theta \leq \dfrac{\pi}{2}$，$0 \leq \rho \leq 1$

$$= \int_0^{\pi/2} d\theta \int_0^1 \rho\cos\theta \cdot \rho\sin\theta \cdot \rho d\rho = \int_0^{\pi/2} \sin\theta\cos\theta d\theta \int_0^1 \rho^3 d\rho$$
$$= \frac{1}{2} \cdot \frac{1}{4} = \frac{1}{8}$$

</details>
