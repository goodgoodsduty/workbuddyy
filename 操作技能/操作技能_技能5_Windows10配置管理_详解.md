## 一、Windows 10 安装

### 1.1 系统要求

| 项目 | 最低配置 |
|------|---------|
| 处理器 | 1 GHz 64位 |
| 内存 | 2 GB |
| 硬盘 | 32 GB |
| 显卡 | DirectX 9 |

### 1.2 安装方式

| 方式 | 说明 |
|------|------|
| **U盘安装** | 用 Rufus 制作启动盘 |
| **光盘安装** | 传统方式 |
| **硬盘安装** | 双系统 |
| **虚拟机** | VMware 安装 |

### 1.3 安装步骤

1. 设置 BIOS 从 U盘启动
2. 启动安装程序
3. 选择语言、键盘
4. 选择"自定义安装"
5. 磁盘分区
6. 等待安装
7. 初始设置

## 二、用户账户管理 ⚠️ 必考

### 2.1 账户类型

| 账户 | 权限 |
|------|------|
| **Administrator** | 最高权限 |
| **Standard User** | 普通用户 |
| **Guest** | 访客（最受限）|

### 2.2 创建账户

**【方法一】设置**
```
设置 → 账户 → 家庭和其他用户 → 将其他人添加到这台电脑
```

**【方法二】控制面板**
```
控制面板 → 用户账户 → 管理其他账户 → 创建一个新账户
```

**【方法三】命令】**
```cmd
net user 用户名 密码 /add
net localgroup administrators 用户名 /add
```

### 2.3 删除账户

```cmd
net user 用户名 /delete
```

### 2.4 修改密码

**【方法一】**
```
设置 → 账户 → 登录选项 → 密码 → 更改
```

**【方法二】命令】
```cmd
net user 用户名 新密码
```

**【例题1】** 创建用户 tom，密码 123，加入管理员组
```cmd
net user tom 123 /add
net localgroup administrators tom /add
```

## 三、文件系统 ⚠️ 必考

### 3.1 NTFS 权限 ⚠️ 必考

| 权限 | 说明 |
|------|------|
| 完全控制 | 所有操作 |
| 修改 | 修改、读取、运行 |
| 读取和执行 | 读+执行 |
| 读取 | 仅读 |
| 写入 | 仅写 |
| 特殊权限 | 自定义 |

### 3.2 文件共享权限

| 权限 | 说明 |
|------|------|
| 读取 | 看、复制文件 |
| 更改 | 读+修改+删除 |
| 完全控制 | 所有操作 |

### 3.3 共享文件夹

**【创建共享】**
```
右键文件夹 → 属性 → 共享 → 共享
```

**【访问共享】**
```
\\IP地址\共享名
\\服务器名\共享名
```

**【隐藏共享】** 共享名后加 `$`

### 3.4 NTFS vs FAT32

| 特性 | NTFS | FAT32 |
|------|------|-------|
| 文件大小 | 无限制 | 4GB |
| 分区大小 | 大 | 32GB |
| 权限 | 支持 | 不支持 |
| 加密 | 支持 | 不支持 |
| 压缩 | 支持 | 不支持 |

## 五、网络配置 ⚠️ 必考

### 5.1 IP 地址配置

**【静态 IP】**
```
控制面板 → 网络和共享中心 → 适配器设置
→ 右键 → 属性 → IPv4 → 手动设置
```

**【自动获取】**
```
自动获得 IP 地址
```

### 5.2 常用网络命令 ⚠️ 必考

| 命令 | 作用 |
|------|------|
| ipconfig | 查看 IP |
| ipconfig /all | 查看详细 |
| ipconfig /release | 释放 |
| ipconfig /renew | 重获 |
| ping | 测试连通 |
| tracert | 跟踪路由 |
| nslookup | DNS 查询 |
| netstat | 网络状态 |
| arp -a | 查看ARP表 |

**【例题2】** 用 ipconfig 查看 MAC 地址
```cmd
ipconfig /all
```
看"物理地址"那行

### 5.3 网络位置

| 位置 | 防火墙 |
|------|--------|
| 家庭网络 | 较低 |
| 工作网络 | 中等 |
| 公共网络 | 最高 |

### 5.4 网络共享

```
高级共享设置 → 启用网络发现
启用文件和打印机共享
关闭密码保护共享
```

## 六、防火墙 ⚠️

### 6.1 启用/关闭

```cmd
# 关闭所有防火墙
netsh advfirewall set allprofiles state off

# 启用
netsh advfirewall set allprofiles state on
```

### 6.2 添加入站规则

```
Windows Defender 防火墙 → 高级设置 → 入站规则 → 新建规则
```

### 6.3 开放端口

```cmd
netsh advfirewall firewall add rule name="HTTP" dir=in action=allow protocol=TCP localport=80
```

## 八、注册表

### 8.1 打开注册表

```cmd

```

### 8.2 五大根键

| 根键 | 作用 |
|------|------|
| HKEY_CLASSES_ROOT | 文件关联 |
| HKEY_CURRENT_USER | 当前用户配置 |
| HKEY_LOCAL_MACHINE | 本机配置 |
| HKEY_USERS | 所有用户 |
| HKEY_CURRENT_CONFIG | 当前硬件配置 |

### 8.3 常见用途

- 修改系统设置
- 查看安装信息
- 修改开机启动项
