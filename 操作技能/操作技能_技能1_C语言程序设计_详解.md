## 一、C语言概述

### 1.1 C 语言的特点

| 特点 | 说明 |
|------|------|
| **简洁紧凑** | 32个关键字 |
| **运算符丰富** | 34种运算符 |

## 二、基本语法

### 2.1 标识符

**【命名规则】** ⚠️
- 字母、数字、下划线
- 不能以数字开头
- 不能是关键字
- 区分大小写

合法：sum, _temp, a1, MAX
非法：1num, int, a-b

### 2.2 关键字 ⚠️ 必背

| 类型 | 关键字 |
|------|------|
| **数据类型** | int, float, double, char, void, short, long, signed, unsigned |
| **控制语句** | if, else, switch, case, default, for, while, do, break, continue, return, goto |
| **存储类** | auto, static, register, extern |
| **其他** | sizeof, const, volatile, struct, union, enum, typedef |

### 2.3 数据类型

| 类型 | 字节 | 范围 |
|------|------|------|
| char | 1 | -128 ~ 127 |
| int | 4 | -2³¹ ~ 2³¹-1 |
| float | 4 | 6-7位有效数字 |
| double | 8 | 15-16位有效数字 |
| short | 2 | -32768 ~ 32767 |
| long | 4/8 | 依平台 |

### 2.4 常量与变量

```c
// 常量
#define PI 3.14159
const int MAX = 100;

// 变量
int age = 18;
float score = 95.5;
char grade = 'A';
```

### 2.5 运算符

| 类别 | 运算符 | 说明 |
|------|--------|------|
| **算术** | +, -, *, /, % | % 只用于整数 |
| **关系** | ==, !=, >, <, >=, <= | 返回 0 或 1 |
| **逻辑** | &&, \|\|, ! | 短路求值 |
| **赋值** | =, +=, -=, *=, /= | |

## 四、控制结构 ⚠️ 必考

### 4.1 if 语句

**【单分支】**
```c
if (条件) {
    语句;
}
```

**【双分支】**
```c
if (条件) {
    语句1;
} else {
    语句2;
}
```

**【多分支】**
```c
if (条件1) {
    语句1;
} else if (条件2) {
    语句2;
} else {
    语句3;
}
```

### 4.2 switch 语句

```c
switch (表达式) {
    case 常量1: 语句1; break;
    case 常量2: 语句2; break;
    default: 语句3; break;
}
```

**【例题3】** 分数分级
```c
int score = 85;
switch (score / 10) {
    case 10:
    case 9: printf("A\n"); break;
    case 8: printf("B\n"); break;
    case 7: printf("C\n"); break;
    case 6: printf("D\n"); break;
    default: printf("E\n");
}
```

### 4.3 for 循环

```c
for (初始化; 条件; 更新) {
    循环体;
}
```

**【例题4】** 1+2+...+100
```c
int sum = 0;
for (int i = 1; i <= 100; i++) {
    sum += i;
}
printf("%d\n", sum);
```

### 4.4 while 循环

```c
while (条件) {
    循环体;
}
```

**【例题5】** 阶乘
```c
int n = 5, fact = 1, i = 1;
while (i <= n) {
    fact *= i;
    i++;
}
printf("%d! = %d\n", n, fact);
```

### 4.5 do-while 循环

```c
do {
    循环体;
} while (条件);
```
**【特点】** 至少执行一次

### 4.6 break 和 continue

| 关键字 | 作用 |
|--------|------|
| **break** | 跳出当前循环/switch |
| **continue** | 跳过本次循环，继续下一次 |

### 4.7 循环嵌套 ⚠️ 必考

**【例题6】** 打印九九乘法表
```c
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {
        printf("%dx%d=%d\t", j, i, i*j);
```

### 5.4 数组常见算法 ⚠️ 必考

**【冒泡排序】**
```c
void bubbleSort(int a[], int n) {
    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-1-i; j++) {
            if (a[j] > a[j+1]) {
                int t = a[j];
                a[j] = a[j+1];
                a[j+1] = t;
            }
        }
    }
}
```

**【选择排序】**
```c
void selectSort(int a[], int n) {
    for (int i = 0; i < n-1; i++) {
        int min = i;
        for (int j = i+1; j < n; j++) {
            if (a[j] < a[min]) {
                min = j;
            }
        }
        int t = a[i];
        a[i] = a[min];
        a[min] = t;
    }
}
```

**【顺序查找】**
```c
int find(int a[], int n, int x) {
    for (int i = 0; i < n; i++) {
        if (a[i] == x) return i;
    }
    return -1;
}
```

## 六、函数 ⚠️ 必考

### 6.1 函数定义

### 6.2 函数调用

```c
int add(int a, int b) {     // 函数定义
    return a + b;
}

int main() {
    int sum = add(3, 5);    // 函数调用
    printf("%d\n", sum);
    return 0;
}
```

### 6.3 函数参数 ⚠️ 必考

**【值传递】** 实参值传给形参，**不改变实参**

```c
void swap(int a, int b) {
    int t = a; a = b; b = t;
}
int main() {
    int x = 3, y = 5;
    swap(x, y);
    printf("%d %d\n", x, y);   // 输出 3 5，未交换
    return 0;
}
```

**【地址传递】** 用指针，**可改变实参**

```c
void swap(int *a, int *b) {
    int t = *a; *a = *b; *b = t;
}
int main() {
    int x = 3, y = 5;
    swap(&x, &y);
    printf("%d %d\n", x, y);   // 输出 5 3，已交换
    return 0;
}
```

### 6.4 函数的声明

```c
int add(int, int);    // 函数声明

int main() { ... }

int add(int a, int b) { return a + b; }  // 函数定义
```

## 七、指针 ⚠️ 必考

### 7.1 指针的概念

**【指针】** 存放变量地址的变量

```c
int a = 10;
int *p = &a;      // p 指向 a
```

### 7.2 指针运算

| 运算 | 含义 |
|------|------|
| & | 取地址 |
| * | 取值（解引用）|
| ++ | 指针+1（移动一个单位）|

### 7.3 指针与数组 ⚠️ 必考

```c
int a[5] = {1,2,3,4,5};
int *p = a;      // p 指向 a[0]

// 以下都等价
printf("%d\n", a[0]);
printf("%d\n", *p);
printf("%d\n", *(p+0));
```

**【例题10】** 用指针遍历数组
```c
int a[5] = {1,2,3,4,5};
int *p = a;
for (int i = 0; i < 5; i++) {
    printf("%d ", *(p+i));
}
```

### 7.4 指针作为函数参数 ⚠️ 必考

```c
void addTen(int *p) {
    *p = *p + 10;
}

int main() {
    int a = 5;
    addTen(&a);
    printf("%d\n", a);   // 输出 15
    return 0;
}
```

## 八、常见编程题 ⚠️ 必考

### 8.1 数值计算类

**【求阶乘】**
```c
int factorial(int n) {
    int f = 1;
    for (int i = 1; i <= n; i++) f *= i;
    return f;
}
```

**【斐波那契数列】**
```c
void fib(int n) {
    int a = 0, b = 1, c;
    printf("%d %d ", a, b);
    for (int i = 2; i < n; i++) {
        c = a + b;
        printf("%d ", c);
        a = b;
        b = c;
    }
}
```

**【判断素数】**
```c
int isPrime(int n) {
    if (n < 2) return 0;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return 0;
    }
    return 1;
}
```

**【水仙花数】**
```c
void narcissistic() {
    for (int n = 100; n < 1000; n++) {
        int a = n / 100, b = n / 10 % 10, c = n % 10;
        if (a*a*a + b*b*b + c*c*c == n) {
            printf("%d\n", n);
        }
    }
}
```

### 8.2 数组操作类

**【数组逆序】**
```c
void reverse(int a[], int n) {
    for (int i = 0; i < n/2; i++) {
        int t = a[i];
        a[i] = a[n-1-i];
        a[n-1-i] = t;
    }
}
```

**【数组元素去重】**
```c
int unique(int a[], int n) {
    int k = 0;
    for (int i = 0; i < n; i++) {
        int j;
        for (j = 0; j < k; j++) {
            if (a[i] == a[j]) break;
        }
        if (j == k) a[k++] = a[i];
    }
    return k;
}
```

### 8.3 字符串处理类

**【统计字符数】**
```c
int countChar(char s[], char c) {
    int count = 0;
    for (int i = 0; s[i] != '\0'; i++) {
        if (s[i] == c) count++;
    }
    return count;
}
```

**【字符串逆序】**
```c
void reverseStr(char s[]) {
    int n = strlen(s);
    for (int i = 0; i < n/2; i++) {
        char t = s[i];
        s[i] = s[n-1-i];
        s[n-1-i] = t;
    }
}
```

### 8.4 综合应用类

**【学生成绩管理】**
```c
struct Student {
    char name[20];
    int score;
};

void input(struct Student s[], int n) {
    for (int i = 0; i < n; i++) {
        scanf("%s %d", s[i].name, &s[i].score);
    }
}

float average(struct Student s[], int n) {
    int sum = 0;
    for (int i = 0; i < n; i++) sum += s[i].score;
    return (float)sum / n;
}
```
