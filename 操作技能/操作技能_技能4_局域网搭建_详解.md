## 一、局域网基础

### 1.1 局域网定义

**【LAN】** Local Area Network，覆盖范围小（<10km），速率高

### 1.2 局域网分类

| 类型 | 速率 | 介质 |
|------|------|------|
| 以太网 | 10Mbps-10Gbps | 双绞线/光纤 |
| 令牌环 | 4/16 Mbps | 双绞线 |
| FDDI | 100Mbps | 光纤 |
| 无线局域网(WiFi) | 11Mbps-9.6Gbps | 无线电 |

### 1.3 拓扑结构 ⚠️ 必考

| 拓扑 | 优点 | 缺点 |
|------|------|------|
| **总线型** | 简单、成本低 | 一断全断 |
| **星型** | 易扩展、故障隔离 | 中心设备故障全瘫 |
| **环型** | 公平 | 单点故障 |
| **树型** | 分层管理 | 根节点重要 |
| **网状** | 高可靠 | 复杂、成本高 |

**【最常用】** 星型（以交换机为中心）

## 二、网络设备 ⚠️ 必考

### 2.1 网卡（NIC）

**【作用】** 计算机连接网络的接口

**【分类】**
- 有线网卡
- 无线网卡
- 千兆网卡、万兆网卡

### 2.2 集线器（Hub）⚠️ 必考

**【工作层】** 物理层

**【特点】**
- 信号放大、多端口共享
- 所有端口同一冲突域、同一广播域
- 半双工
- 已基本淘汰

### 2.3 交换机（Switch）⚠️ 必考

**【工作层】** 数据链路层

**【特点】**
- 每个端口独立冲突域
- 所有端口同一广播域
- MAC地址表学习
- 全双工

**【工作原理】** MAC地址表 + 帧转发

### 2.4 路由器（Router）⚠️ 必考

**【工作层】** 网络层

**【特点】**
- 不同网络互联
- 路由表 + IP转发
- 分割广播域

### 2.5 设备对比 ⚠️ 必考

| 设备 | 工作层 | 冲突域 | 广播域 |
|------|-------|--------|--------|
| 集线器 | 物理层 | 1个 | 1个 |
| 交换机 | 数据链路层 | 每个端口1个 | 1个 |
| 路由器 | 网络层 | 每个端口1个 | 每个端口1个 |

## 三、网线制作 ⚠️ 必考

### 3.1 双绞线标准

**【T568A 线序】**
- 绿白、绿、橙白、蓝、蓝白、橙、棕白、棕

**【T568B 线序】** ⚠️ 国内常用
- 橙白、橙、绿白、蓝、蓝白、绿、棕白、棕

**【记忆】** T568B = "**橙橙绿绿蓝蓝棕棕**"，但每对是"白-纯-白-纯"

### 3.2 直通线 vs 交叉线 ⚠️

| 类型 | 两端 | 用途 |
|------|------|------|
| **直通线** | 两端相同（T568B-T568B）| 不同设备互联（PC-交换机）|
| **交叉线** | 两端不同（T568A-T568B）| 同类设备互联（PC-PC、交换机-交换机）|

**【现在】** 多数设备支持自动翻转，交叉线用得少了

### 3.3 制作步骤

1. 剥线（约3cm）
2. 排线序（T568B）
3. 剪齐（留1.5cm）
4. 插入水晶头
5. 用压线钳压紧
6. 用测线仪测试

## 四、IP 地址配置 ⚠️ 必考

### 4.1 静态 IP 配置

**【Windows 系统】**
```
控制面板 → 网络和共享中心 → 更改适配器设置 → 右键属性
→ IPv4 → 使用下面的 IP 地址
```

**【配置参数】**
| 项目 | 示例 |
|------|------|
| IP 地址 | 192.168.1.100 |
| 子网掩码 | 255.255.255.0 |
| 默认网关 | 192.168.1.1 |
| 首选 DNS | 8.8.8.8 |
| 备用 DNS | 114.114.114.114 |

### 4.2 DHCP 自动获取

```
自动获得 IP 地址
自动获得 DNS 服务器地址
```

### 4.3 子网掩码计算 ⚠️ 必考

**【作用】** 区分网络号和主机号

**【例题1】** IP: 192.168.1.100, 子网掩码: 255.255.255.0，求网络号？

**解**：
```
  11000000.10101000.00000001.01100100
AND
  11111111.11111111.11111111.00000000
= 11000000.10101000.00000001.00000000
= 192.168.1.0
```

**网络号**：192.168.1.0

### 4.4 子网划分 ⚠️

**【题目】** 192.168.1.0/24 划分为 4 个子网

**【解】**
- 原掩码 /24，借 2 位
- 新掩码 /26 = 255.255.255.192
- 4 个子网：
  - 192.168.1.0/26（0-63）
  - 192.168.1.64/26（64-127）
  - 192.168.1.128/26（128-191）
  - 192.168.1.192/26（192-255）

## 五、连通性测试 ⚠️ 必考

### 5.1 ping 命令 ⚠️ 必考

**【基本用法】**
```bash
ping 192.168.1.1
ping www.baidu.com
```

**【参数】**
- `-t`：持续ping
- `-n N`：ping N次
- `-l size`：包大小

**【判断】**
- 有回复：网络通
- 请求超时：网络不通
- TTL：每经过一个路由减1

### 5.2 tracert 命令

**【作用】** 跟踪到目标的路由路径

```bash
tracert www.baidu.com
```

### 5.3 其他命令

| 命令 | 作用 |
|------|------|
| ipconfig / all | 查看所有网络配置 |
| ipconfig / release | 释放 IP |
| ipconfig / renew | 重新获取 IP |
| arp -a | 查看 ARP 缓存 |
| netstat -an | 查看网络连接 |
| nslookup | DNS 查询 |

## 七、网络故障排查 ⚠️

### 7.1 排查思路

1. **物理层**：网线是否插好、网线是否损坏
2. **数据链路层**：MAC地址冲突、交换机配置
3. **网络层**：IP地址配置、子网掩码、网关
4. **应用层**：DNS、HTTP等

### 7.2 排查流程

```
PC1 ↔ PC2 不能ping通
  ↓
1. 物理连接？网线、接口灯
  ↓
2. IP配置？ipconfig
  ↓
3. 子网掩码？是否同一网段
  ↓
4. 网关？是否同网关
  ↓
5. 防火墙？临时关闭
  ↓
6. ping localhost（127.0.0.1）测试本机
```

### 7.3 常见故障

| 现象 | 原因 |
|------|------|
| IP地址冲突 | 同一网段有相同IP |
| 子网掩码错 | 跨网段 |
| 网关错 | 不能访问外网 |
| DNS错 | 不能解析域名 |
| 网线损坏 | 物理连接问题 |

## 八、VLAN 基础

### 8.1 VLAN 概念

**【VLAN】** Virtual LAN，虚拟局域网，**分割广播域**

### 8.2 VLAN 作用

- 隔离广播域
- 提高安全性
- 便于管理

### 8.3 VLAN 划分方式

| 方式 | 描述 |
|------|------|
| 基于端口 | 最常用 |
| 基于MAC | 灵活 |
| 基于IP | 灵活 |
| 基于协议 | 特殊场景 |

### 8.4 VLAN 配置

```
vlan 10
interface g0/0/1
  port link-type access
  port default vlan 10
```
