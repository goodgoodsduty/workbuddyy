## 极限

lim(x→0) sinx⁄x = 1

lim(x→∞) (1+¹⁄ₓ)ˣ = e ； lim(x→0) (1+x)¹⁄ₓ = e

lim(x→0) tanx⁄x = 1 ； lim(x→0) arcsinx⁄x = 1 ； lim(x→0) arctanx⁄x = 1

lim(x→0) (eˣ-1)⁄x = 1 ； lim(x→0) ln(1+x)⁄x = 1 ； lim(x→0) (aˣ-1)⁄x = ln a

lim(x→0) (1-cosx)⁄x² = ¹⁄₂ ； lim(x→0) ⁿ√(1+x)-1⁄x = ¹⁄ₙ

lim(x→∞) lnx⁄xᵖ = 0(p>0) ； lim(x→∞) xᵖ⁄aˣ = 0(a>1)

### 等价无穷小（x→0）

sinx~x ； tanx~x ； arcsinx~x ； arctanx~x

eˣ-1~x ； ln(1+x)~x ； aˣ-1~x·ln a

1-cosx~¹⁄₂x² ； ⁿ√(1+x)-1~¹⁄ₙx ； (1+x)ᵃ-1~ax

tanx-sinx~¹⁄₂x³ ； x-sinx~¹⁄₆x³ ； tanx-x~¹⁄₃x³

> ⚠️ 等价无穷小只用于乘除，不用于加减

### 求极限方法

直接代入 → 因式分解(⁰⁄₀) → 有理化(根号) → 重要极限 → 等价替换 → 洛必达(⁰⁄₀或∞⁄∞) → 抓大头(∞⁄∞) → 夹逼

洛必达：lim f⁄g = lim f'⁄g'，可连续用，每次验类型

抓大头：分子次<分母次→0；=→系数比；>→∞

### 间断点

可去(极限存在≠函数值) | 跳跃(左右极限不等) → 第一类

无穷(一侧→∞) | 振荡(极限不存在) → 第二类

闭区间：有界 | 最值 | 介值 | 零点(f(a)f(b)<0→有根)

## 导数与微分

### 求导公式

| f | f' |
|---|----|
| C | 0 |
| xⁿ | nxⁿ⁻¹ |
| eˣ | eˣ |
| aˣ | aˣ·ln a |
| lnx | ¹⁄ₓ |
| logₐx | ¹⁄₍ₓln a₎ |
| sinx | cosx |
| cosx | -sinx |
| tanx | sec²x |
| cotx | -csc²x |
| secx | secx·tanx |
| cscx | -cscx·cotx |
| arcsinx | ¹⁄√₍₁₋ₓ²₎ |
| arccosx | -¹⁄√₍₁₋ₓ²₎ |
| arctanx | ¹⁄₍₁₊ₓ²₎ |
| arccotx | -¹⁄₍₁₊ₓ²₎ |

### 求导法则

(u±v)' = u'±v' ； (uv)' = u'v+uv' ； (u⁄v)' = (u'v-uv')⁄v²

链式：y=f(u),u=g(x) → y' = f'(u)·g'(x)

反函数：dx⁄dy = ¹⁄f'(x)

隐函数：F(x,y)=0 → 两边对x求导解出y'

参数方程：dy⁄dx = y'(t)⁄x'(t) ； d²y⁄dx² = [y''x'-y'x'']⁄(x')³

对数求导：幂指函数y=uᵛ → lny=v·lnu → ¹⁄y·y'=...

### 高阶导数

(eˣ)⁽ⁿ⁾ = eˣ ； (aˣ)⁽ⁿ⁾ = aˣ(ln a)ⁿ

(sinx)⁽ⁿ⁾ = sin(x+nπ⁄₂) ； (cosx)⁽ⁿ⁾ = cos(x+nπ⁄₂)

(lnx)⁽ⁿ⁾ = (-1)ⁿ⁻¹(n-1)!⁄xⁿ ； (¹⁄₍ₓ₊ₐ₎)⁽ⁿ⁾ = (-1)ⁿn!⁄(x+a)ⁿ⁺¹

### 微分

dy = f'(x)dx ； d(u±v)=du±dv ； d(uv)=vdu+udv ； d(u⁄v)=(vdu-udv)⁄v²

形式不变性：x是自变量或中间变量，dy=f'(x)dx都成立

### 导数应用

切线：y-f(x₀)=f'(x₀)(x-x₀) ； 法线：y-f(x₀)=-¹⁄f'(x₀)(x-x₀)

单调：f'>0增 f'<0减

极值必要：f'(x₀)=0或不存在

极值判别①：f'左正右负→极大；左负右正→极小；同号→非极值

极值判别②：f'(x₀)=0，f''(x₀)>0极小，f''(x₀)<0极大，=0改用①

最值：驻点+不可导点+端点，比大小

凹凸：f''>0凹(∪)；f''<0凸(∩)

拐点：f''=0且两侧变号

渐近线：水平lim f=A→y=A；垂直lim f=∞→x=x₀；斜lim f⁄x=k,lim(f-kx)=b→y=kx+b

## 不定积分

### 基本公式

∫ k dx = kx+C

∫ xⁿ dx = xⁿ⁺¹⁄ₙ₊₁+C (n≠-1)

∫ ¹⁄ₓ dx = ln|x|+C

∫ eˣ dx = eˣ+C

∫ aˣ dx = aˣ⁄ln(a)+C

∫ sinx dx = -cosx+C ； ∫ cosx dx = sinx+C

∫ tanx dx = -ln|cosx|+C ； ∫ cotx dx = ln|sinx|+C

∫ sec²x dx = tanx+C ； ∫ csc²x dx = -cotx+C

∫ secx·tanx dx = secx+C ； ∫ cscx·cotx dx = -cscx+C

∫ ¹⁄₍₁₊ₓ²₎ dx = arctanx+C

∫ ¹⁄√₍₁₋ₓ²₎ dx = arcsinx+C

∫ ¹⁄√₍ₐ²₋ₓ²₎ dx = arcsin(x⁄ₐ)+C

∫ ¹⁄₍ₐ²₊ₓ²₎ dx = ¹⁄ₐ·arctan(x⁄ₐ)+C

∫ ¹⁄₍ₓ²₋ₐ²₎ dx = ¹⁄₂ₐ·ln|₍ₓ₋ₐ₎⁄₍ₓ₊ₐ₎|+C

∫ ¹⁄√₍ₓ²±ₐ²₎ dx = ln|x+√(x²±a²)|+C

∫ √(a²-x²) dx = ⁽ˣ⁄₂⁾√(a²-x²)+⁽ᵃ²⁄₂⁾arcsin(x⁄ₐ)+C

### 凑微分

dx→¹⁄ₐd(ax+b) ； xⁿdx→¹⁄₍ₙ₊₁₎d(xⁿ⁺¹) ； ¹⁄ₓdx→d(ln|x|)

cosx dx→d(sinx) ； ¹⁄₍₁₊ₓ²₎dx→d(arctanx)

口诀：复合函数→里层导数在不在旁边

### 第二换元

√(a²-x²)→x=a·sint ； √(a²+x²)→x=a·tant ； √(x²-a²)→x=a·sect

ⁿ√(ax+b)→t=ⁿ√(ax+b)

### 分部积分

∫ u dv = uv - ∫ v du

选u口诀：**反·对·幂·指·三**（排前面选u，排后面选dv）

### 有理函数

假分式→长除→真分式→分母因式分解→拆部分分式→逐项积分

## 定积分

### 性质

∫ₐᵇ f = -∫ᵇₐ f ； ∫ₐₐ f = 0 ； ∫ₐᵇ(αf+βg) = α∫ₐᵇf+β∫ₐᵇg

∫ₐᵇf = ∫ₐᶜf+∫ᶜᵇf ； f≤g → ∫ₐᵇf≤∫ₐᵇg

m(b-a)≤∫ₐᵇf≤M(b-a) ； ∫ₐᵇf = f(ξ)(b-a) (积分中值)

### 牛顿-莱布尼茨

∫ₐᵇ f(x)dx = F(b)-F(a)

### 奇偶性

偶函数[-a,a]：∫₋ₐᵃf = 2∫₀ᵃf ； 奇函数[-a,a]：∫₋ₐᵃf = 0

### 换元

∫ₐᵇ f(x)dx = ∫ᵅᵝ f(φ(t))φ'(t)dt → **换元必换限**

分部：∫ₐᵇ u dv = uv|ₐᵇ - ∫ₐᵇ v du

### 应用

面积：S=∫ₐᵇ[f-g]dx(上下) ； S=∫ᶜᵈ[φ-ψ]dy(左右) ； S=¹⁄₂∫ₐᵝr²dθ(极坐标)

体积：Vₓ=π∫ₐᵇf²dx(绕x) ； Vᵧ=2π∫ₐᵇxf dx(柱壳绕y)

弧长：s=∫ₐᵇ√(1+y'²)dx ； s=∫ₐᵇ√(x'²+y'²)dt ； s=∫ₐᵝ√(r²+r'²)dθ

## 常微分方程

### 一阶

可分离：dy⁄g(y)=f(x)dx → 两边积

齐次：dy⁄dx=f(y⁄x) → 令u=y⁄x，y=ux

一阶线性y'+Py=Q：y=e⁻∫Pdx[∫Q·e∫Pdx dx+C]

伯努利y'+Py=Qyⁿ：令z=y¹⁻ⁿ

### 可降阶

缺y：y''=f(x,y') → p=y'，y''=p'

缺x：y''=f(y,y') → p=y'，y''=p·dp⁄dy

### 二阶常系数线性

齐次y''+py'+qy=0 → 特征方程r²+pr+q=0

r₁≠r₂：y=C₁e^(r₁x)+C₂e^(r₂x)

r₁=r₂=r：y=(C₁+C₂x)e^(rx)

r=α±βi：y=e^(αx)(C₁cosβx+C₂sinβx)

非齐次=齐次通解+特解

特解设：多项式→xˢQₙ(x)；e^(αx)·多项式→xˢe^(αx)Qₙ(x)；e^(αx)三角→xˢe^(αx)[Ccosβx+Dsinβx]

s看特征根重数：不重→0，重1→1，重2→2

## 偏导数·全微分·全导数

### 偏导

fₓ=∂f⁄∂x：y当常数对x求导

fₓₓ=∂²f⁄∂x² ； fₓᵧ=∂²f⁄∂x∂y ； fᵧᵧ=∂²f⁄∂y²

混合偏导连续→fₓᵧ=fᵧₓ

### 全微分

dz = fₓdx + fᵧdy ； du = fₓdx+fᵧdy+f_zdz(三元)

关系链：**偏导连续→可微→偏导存在+连续→连续**

近似：f(x₀+Δx,y₀+Δy)≈f(x₀,y₀)+fₓΔx+fᵧΔy

### 链式法则

z=f(u,v),u=u(t),v=v(t) → dz⁄dt=∂z⁄∂u·du⁄dt+∂z⁄∂v·dv⁄dt

z=f(u,v),u=u(x,y),v=v(x,y) → ∂z⁄∂x=∂z⁄∂u·∂u⁄∂x+∂z⁄∂v·∂v⁄∂x

z=f(u,v,x,y) → ∂z⁄∂x=∂f⁄∂u·∂u⁄∂x+∂f⁄∂v·∂v⁄∂x+∂f⁄∂x

> 有几条路径到目标变量，就有几项相加。z直接含自变量时别漏最后一项！

### 全导数

所有中间变量→单自变量t：dz⁄dt=Σ∂z⁄∂uᵢ·duᵢ⁄dt

### 隐函数

F(x,y)=0：dy⁄dx=-Fₓ⁄Fᵧ

F(x,y,z)=0：∂z⁄∂x=-Fₓ⁄F_z ； ∂z⁄∂y=-Fᵧ⁄F_z

口诀：负号，分子F对自变量偏导，分母F对因变量偏导

### 极值判别

一元：f'(x₀)=0→驻点；f''(x₀)>0极小，<0极大，=0改用一阶判别

二元：A=fₓₓ B=fₓᵧ C=fᵧᵧ Δ=AC-B²

Δ>0,A>0→极小 ； Δ>0,A<0→极大 ； Δ<0→非极值 ； Δ=0→不确定

条件极值：L=f+λφ → 解Lₓ=0 Lᵧ=0 L_λ=0

### 方向导数与梯度

∂f⁄∂l=fₓcosα+fᵧcosβ+f_zcosγ

grad f=(fₓ,fᵧ,f_z)=∇f

方向导数最大值=|∇f|，方向=梯度方向
