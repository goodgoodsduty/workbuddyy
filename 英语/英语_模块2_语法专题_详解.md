## 二、专题一：时态 ⚠️ 必考

### 2.1 16种时态总表

| 时间/状态 | 一般 | 进行 | 完成 | 完成进行 |
|---------|------|------|------|---------|
| 过去 | did | was doing | had done | had been doing |
| 现在 | do/does | is/am doing | have done | have been doing |
| 将来 | will do | will be doing | will have done | - |

### 2.2 常用时态

**1. 一般现在时**
- 习惯性动作：I get up at 7 every day.
- 客观真理：The sun rises in the east.

**2. 一般过去时**
- 过去发生的动作：I visited Beijing last year.

**3. 一般将来时**
- 将来发生的：I will call you tomorrow.
- be going to：计划做某事

**4. 现在进行时**
- 正在进行的：I am reading a book.
- 近期计划：I am leaving tomorrow.

**5. 现在完成时**
- 过去发生但与现在有联系：I have finished my homework.
- 注意：不可与具体时间状语（yesterday, last week）连用

**6. 过去完成时**
- 过去的过去：When I arrived, he had left.

**7. 过去进行时**
- 过去某时正在做：I was reading at 8 yesterday.

**8. 过去将来时**
- 从过去看将来：He said he would come.

**9. 现在完成进行时**
- 一直持续到现在：I have been waiting for 2 hours.

### 2.3 时态判断三步法

1. **找时间状语**：yesterday→过去；now→现在/进行；already→完成
2. **判断主从句时态关系**：主现从将，主过从过
3. **结合语境判断**

**【例题1】** 选择正确时态：
By the time you arrive, I _____ (finish) my work.

**答案**：will have finished（到将来某时已完成）
翻译：等你到的时候，我就会完成工作。

## 三、专题二：被动语态

### 3.1 结构

**一般时**：be + 及物动词过去分词
**完成时**：have been + 过去分词
**进行时**：be being + 过去分词
**将来时**：will be + 过去分词

### 3.2 适用情况

1. **动作承受者作主语**：The book was written by him.
2. **强调动作本身**：The window was broken.
3. **不知道/不强调执行者**：My bike was stolen.

### 3.3 不能用被动的情况

- 不及物动词
- 系动词：be, become, look, feel
- 表示归属：have, own, belong to

**【例题2】** 改被动：They built the bridge in 1990.
**答案**：The bridge was built in 1990.

## 四、专题三：三大从句 ⚠️ 核心

### 4.1 名词性从句

**包括**：主语从句、宾语从句、表语从句、同位语从句

**【结构】** 用连接词引导一个完整句子作名词成分

**【连接词】**

| 词 | 含义 | 例子 |
|----|------|------|
| that | 无意义 | I know that he is honest. |
| whether | 是否 | I don't know whether he will come. |
| if | 是否（口语）| Ask him if he knows. |
| who/whom | 谁 | I know who he is. |
| what | 什么 | I don't know what he said. |
| when | 何时 | Tell me when you will come. |
| where | 何地 | I know where he lives. |
| why | 为何 | Tell me why you are late. |
| how | 怎样 | I know how to do it. |

**【例题3】** 选择：_____ he said is true.
A. What B. That C. Which D. Whether
**答案**：**A. What**（what 引导主语从句且在从句中作宾语）

**【例题4】** The fact _____ he succeeded surprised us.
**答案**：**that**（同位语从句，解释 fact 的内容）

### 4.2 定语从句 ⚠️ 必考

**【结构】** 先行词 + 关系词 + 从句

**【关系词】**

| 关系词 | 含义 | 例 |
|--------|------|---|
| **who** | 人（主语）| the man who is here |
| **whom** | 人（宾语）| the man whom I met |
| **whose** | 谁的 | the man whose car is red |
| **which** | 物 | the book which I bought |
| **that** | 人/物 | the book that I bought |
| **when** | 时间 | the day when I met him |
| **where** | 地点 | the place where I was born |
| **why** | 原因 | the reason why he left |

**【限制性 vs 非限制性】**

| 类型 | 形式 | 逗号 |
|------|------|------|
| 限制性 | 必须，无逗号 | - |
| 非限制性 | 补充说明 | 有逗号 |

**【that 和 which 的区别】**

- 只能用 that：最高级、序数词、all/any/every、the only、the very
- 只能用 which：非限制性定语从句

**【例题5】** 改写：The book is interesting. I bought it yesterday.
**答案**：The book **which/that** I bought yesterday is interesting.

**【例题6】** 填空：This is the best book _____ I have ever read.
**答案**：**that**（最高级后只能用 that）

**【例题7】** 翻译：他有一个女儿，住在上海。
**答案**：He has a daughter, **who** lives in Shanghai.（非限制性）

### 4.3 状语从句 ⚠️ 必考

| 类型 | 连接词 |
|------|------|
| **时间** | when, while, as, before, after, since, until, as soon as, the moment |
| **原因** | because, since, as, for |
| **条件** | if, unless, as long as, on condition that |
| **让步** | though, although, even though, even if |
| **目的** | so that, in order that |
| **结果** | so... that, such... that |
| **方式** | as, as if, as though |
| **地点** | where, wherever |
| **比较** | than, as... as |

**【例题8】** 选择：_____ he is young, he knows a lot.
A. Because B. Although C. Since D. So
**答案**：**B. Although**（虽然）

**【例题9】** 翻译：他说得太快，我听不清。
**答案**：He spoke **so** fast **that** I couldn't hear clearly.

## 五、专题四：非谓语动词 ⚠️ 重点

### 5.1 三种非谓语动词

| 形式 | 时态/语态 | 主动 | 被动 |
|------|----------|------|------|
| 不定式 | 一般 | to do | to be done |
| | 完成 | to have done | to have been done |
| 动名词 | 一般 | doing | being done |
| | 完成 | having done | having been done |
| 分词 | 一般 | doing | being done |
| | 完成 | having done | having been done |
| | 过去分词 | done | - |

### 5.2 句法功能

| 形式 | 主语 | 宾语 | 定语 | 状语 | 补语 |
|------|------|------|------|------|------|
| 不定式 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 动名词 | ✓ | ✓ | ✓ | - | - |
| 现在分词 | - | - | ✓ | ✓ | ✓ |
| 过去分词 | - | - | ✓ | ✓ | ✓ |

### 5.3 关键用法

**【不定式作宾语】**
- want to do, decide to do, hope to do, plan to do

**【动名词作宾语】**
- enjoy doing, avoid doing, mind doing, finish doing, practice doing
- suggest doing, recommend doing

**【分词作状语】**
- 时间：Walking on the street, I met my old friend.
- 原因：Being tired, he went to bed early.
- 伴随：He sat there, reading a book.
- 结果：He worked so hard, **exhausting** himself.（主动）/ He worked so hard, **exhausted** himself.（被动）

**【分词作定语】**
- V-ing：表示主动/进行：the man **standing** there
- V-ed：表示被动/完成：the book **written** by him

**【例题10】** 选择：I enjoy _____ music.
A. listen to B. listening to C. to listen to D. listened to
**答案**：**B. listening to**（enjoy + doing）

**【例题11】** 翻译：他决定去上海。
**答案**：He decided **to go to** Shanghai.

**【例题12】** 选择：The man _____ the car is my father.
A. driving B. driven C. to drive D. drove
**答案**：**A. driving**（主动进行中）

**【例题13】** 翻译：被妈妈表扬，他很高兴。
**答案**：**Praised by** his mother, he felt very happy.

## 六、专题五：虚拟语气 ⚠️ 必考

### 6.1 三种基本句型

| 句型 | 与现在相反 | 与过去相反 | 与将来相反 |
|------|----------|----------|----------|
| **if 条件句** | 主过，从过 | 主过，从过完 | 主过，should/were to + 动原 |
| **wish 后的宾语从句** | 从过（be 用 were）| 从过完 | - |
| **as if / as though** | 从过 | 从过完 | - |

### 6.2 重要句型

| 句型 | 用法 |
|------|------|
| **would rather** | 从过/从过完 |
| **suggest/insist/order** | (should) + 动原 |
| **It is (high) time** | 从过 |
| **If only** | 从过/从过完 |

### 6.3 虚拟语气公式 ⚠️ 必背

| 时态 | 主句 | 从句 |
|------|------|------|
| 现在 | would/should + 动原 | 过去式（be 用 were）|
| 过去 | would/should have done | had done |
| 将来 | would/should + 动原 | should + 动原 / were to + 动原 |

**【例题14】** 翻译：如果我是你，我会去。
**答案**：If I **were** you, I **would** go.

**【例题15】** 翻译：他建议我们早点出发。
**答案**：He suggested that we (should) **start** early.

**【例题16】** 翻译：如果昨天我没有迟到，我就见到他了。
**答案**：If I **had not** been late yesterday, I **would have** seen him.

## 七、专题六：情态动词

### 7.1 常见情态动词

| 情态动词 | 用法 |
|---------|------|
| can | 能力/许可 |
| could | 过去能力/委婉 |
| may | 许可/可能 |
| might | 也许/委婉 |
| must | 必须/一定 |
| should | 应该 |
| would | 愿意/过去习惯 |
| need | 需要 |
| dare | 敢 |

### 7.2 重要用法

**【must have done】** 一定做了（对过去的推测）
**【can't have done】** 不可能做了
**【should have done】** 本应该做（实际没做）
**【needn't have done】** 本不必做

**【例题17】** 翻译：他不可能偷了钱。
**答案**：He **can't have** stolen the money.

**【例题18】** 翻译：你本应该早点来。
**答案**：You **should have** come earlier.

## 八、专题七：主谓一致 ⚠️ 常考

### 8.1 原则

**单数主语 → 单数谓语；复数主语 → 复数谓语**

### 8.2 特殊规则

| 情况 | 谓语 |
|------|------|
| each/every + 名词 | 单数 |
| 主语 + with/together with | 单数 |
| many a + 单数 | 单数 |
| either/neither of | 单数 |
| the number of | 单数 |
| a number of | 复数 |
| police/cattle | 复数 |
| news/maths | 单数 |
| 分数/百分数 + of | 取决于 of 后名词 |
| 主语从句、不定式、动名词 | 一般单数 |

**【例题19】** 选择：The number of students _____ increasing.
A. is B. are C. were D. have been
**答案**：**A. is**（the number of 表"…的数量"是单数）

**【例题20】** 翻译：每个学生都应该努力学习。
**答案**：Each student **should** study hard.（each 形式上单数）

## 九、专题八：倒装句 ⚠️ 常考

### 9.1 完全倒装（谓语全部前置）

- Here comes the bus.
- There goes the bell.
- In came the teacher.

### 9.2 部分倒装（助动词提前）

**【否定副词开头】** Never, Hardly, Seldom, Little, Not only, Neither, Nor
- Never have I seen such a beautiful scene.
- Hardly did I know what to do.

**【only + 状语】**
- Only then did I realize my mistake.
- Only in this way can you succeed.

**【if 虚拟省略】** Were I you, I would go.

**【例题21】** 翻译：我从未去过北京。
**答案**：Never **have** I **been** to Beijing.

**【例题22】** 翻译：直到他回来，我才知道真相。
**答案**：Not until he came back **did** I know the truth.

## 十、专题九：It 用法

| 用法 | 例 |
|------|---|
| 形式主语 | **It** is important to learn English. |
| 形式宾语 | I find **it** hard to understand. |
| 强调句 | **It was** yesterday **that** I met him. |
| 指代时间/天气/距离 | **It** is 5 o'clock now. |

**【强调句】** It is/was + 被强调部分 + that/who + 其余

**【例题23】** 强调：I met him yesterday.
**答案**：**It was** him **that** I met yesterday.（强调宾语）
或：**It was** yesterday **that** I met him.（强调时间）
