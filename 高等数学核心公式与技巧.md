### 1.3 求极限方法

#### 直接代入法
若函数在该点连续，直接代入求值

#### 因式分解法（⁰⁄₀型）
分子分母因式分解，约去零因子

#### 有理化法（含根号）
分子或分母有理化，消去根号

#### 重要极限法
凑成 sinx⁄x 或 (1+¹⁄ₓ)ˣ 的形式

#### 等价无穷小替换法
将复杂函数替换为等价无穷小，简化计算

#### 洛必达法则（⁰⁄₀ 或 ∞⁄∞）
若 lim f(x)⁄g(x) 为 ⁰⁄₀ 或 ∞⁄∞，则 lim f(x)⁄g(x) = lim f'(x)⁄g'(x)

**注意：**
- 只能用于 ⁰⁄₀ 或 ∞⁄∞ 型
- 可以连续使用多次
- 每用一次都要检验类型

#### 抓大头法（∞⁄∞型）
分子分母同除以最高次幂

**结论：**
- 分子次数 < 分母次数 → 极限 = 0
- 分子次数 = 分母次数 → 极限 = 最高次系数比
- 分子次数 > 分母次数 → 极限 = ∞

#### 夹逼准则
若 g(x) ≤ f(x) ≤ h(x)，且 lim g(x) = lim h(x) = A，则 lim f(x) = A

#### 单调有界准则
单调递增有上界（或递减有下界）的数列必有极限

### 1.4 连续性

#### 连续定义
lim(x→x₀) f(x) = f(x₀)

#### 间断点分类

| 类型 | 条件 | 例子 |
|------|------|------|
| 可去间断点 | 极限存在 ≠ 函数值（或没定义） | f(x)=sinx⁄x 在 x=0 |
| 跳跃间断点 | 左右极限存在但不相等 | f(x)=\|x\|⁄x 在 x=0 |
| 无穷间断点 | 至少一侧极限为∞ | f(x)=¹⁄ₓ 在 x=0 |
| 振荡间断点 | 极限不存在且不为∞ | f(x)=sin¹⁄ₓ 在 x=0 |

**前两类为第一类间断点，后两类为第二类间断点**

#### 闭区间连续函数性质
- 有界性：闭区间连续 → 有界
- 最值性：闭区间连续 → 有最大值和最小值
- 介值性：闭区间连续，可取到最大值和最小值之间的一切值
- 零点定理：f(a)·f(b)<0 → 存在 c∈(a,b) 使 f(c)=0

## 二、导数与微分

### 2.1 导数定义

f'(x₀) = lim(Δx→0) [f(x₀+Δx)-f(x₀)]⁄Δx = lim(x→x₀) [f(x)-f(x₀)]⁄(x-x₀)

**几何意义：** 切线斜率

### 2.2 基本求导公式

|------|------|------|
| C（常数） | 0 | |
| xⁿ | nxⁿ⁻¹ | 幂函数 |
| eˣ | eˣ | 指数 |
| aˣ | aˣ·ln a | 一般指数 |
| lnx | ¹⁄ₓ | 对数 |
| logₐx | ¹⁄₍ₓ·ln a₎ | 一般对数 |
| sinx | cosx | 三角 |
| cosx | -sinx | 三角 |
| tanx | sec²x | 三角 |
| cotx | -csc²x | 三角 |
| secx | secx·tanx | 三角 |
| cscx | -cscx·cotx | 三角 |
| arcsinx | ¹⁄√₍₁₋ₓ²₎ | 反三角 |
| arccosx | -¹⁄√₍₁₋ₓ²₎ | 反三角 |
| arctanx | ¹⁄₍₁₊ₓ²₎ | 反三角 |
| arccotx | -¹⁄₍₁₊ₓ²₎ | 反三角 |

### 2.3 求导法则

#### 四则运算

| 运算 | 公式 |
|------|------|
| 加减 | (u±v)' = u'±v' |
| 乘法 | (uv)' = u'v + uv' |
| 除法 | (u⁄v)' = (u'v-uv')⁄v² |
| 数乘 | (Cu)' = Cu' |

**乘法推广（莱布尼茨公式）：**
(uv)⁽ⁿ⁾ = Σ Cₙᵏ u⁽ⁿ⁻ᵏ⁾ v⁽ᵏ⁾

#### 复合函数求导（链式法则）

y = f(u), u = g(x) → y' = f'(u)·g'(x) = dy⁄du · du⁄dx

**口诀：** 由外到内，逐层求导，相乘连接

#### 反函数求导

y = f(x) 的反函数 x = f⁻¹(y)，则 dx⁄dy = ¹⁄₍dy⁄dx₎ = ¹⁄f'(x)

#### 隐函数求导

方程 F(x,y)=0 确定 y=y(x)，两边对x求导，解出 y'

**技巧：** y看作x的函数，用链式法则

#### 参数方程求导

x = x(t), y = y(t)

dy⁄dx = y'(t)⁄x'(t)

d²y⁄dx² = [y''(t)x'(t)-y'(t)x''(t)]⁄[x'(t)]³

#### 对数求导法

适用于：幂指函数 y = u(x)ᵛ⁽ˣ⁾ 或 多因子乘积

步骤：
1. 两边取对数：ln y = v(x)·ln u(x)
2. 两边求导：¹⁄y · y' = ...
3. 解出 y' = y·(...)

### 2.4 高阶导数

#### 常见函数的高阶导数

| 函数 | n阶导数 |
|------|---------|
| eˣ | eˣ |
| aˣ | aˣ·(ln a)ⁿ |
| sinx | sin(x+n·π⁄₂) |
| cosx | cos(x+n·π⁄₂) |
| lnx | (-1)ⁿ⁻¹·(n-1)!⁄xⁿ |
| ¹⁄₍ₓ₊ₐ₎ | (-1)ⁿ·n!⁄(x+a)ⁿ⁺¹ |
| xⁿ | n!（n阶后） |

#### 归纳法求高阶导数

先求前几阶，找规律，再用数学归纳法证明

### 2.5 微分

#### 微分定义
dy = f'(x)dx

#### 微分形式不变性
无论x是自变量还是中间变量，dy = f'(x)dx 都成立

#### 微分运算法则

| 运算 | 微分 |
|------|------|
| d(u±v) | du±dv |
| d(uv) | vdu+udv |
| d(u⁄v) | (vdu-udv)⁄v² |

### 2.6 导数应用

#### 几何应用

**切线方程：** y-f(x₀) = f'(x₀)(x-x₀)

**法线方程：** y-f(x₀) = -¹⁄f'(x₀)(x-x₀)

#### 函数性态分析

**单调性：**
- f'(x)>0 → f(x)单调递增
- f'(x)<0 → f(x)单调递减

**极值：**
- 必要条件：f'(x₀)=0 或 f'(x₀)不存在
- 第一充分条件：f'在x₀两侧变号（左正右负→极大，左负右正→极小）
- 第二充分条件：f'(x₀)=0，f''(x₀)>0→极小，f''(x₀)<0→极大

**最值：** 比较极值点和端点的函数值

**凹凸性：**
- f''(x)>0 → 凹（∪）
- f''(x)<0 → 凸（∩）

**拐点：** f''(x₀)=0 且 f''在x₀两侧变号

**渐近线：**
- 水平：lim(x→∞) f(x) = A → y=A
- 垂直：lim(x→x₀) f(x) = ∞ → x=x₀
- 斜：lim(x→∞) f(x)⁄x = k，lim(x→∞) [f(x)-kx] = b → y=kx+b

### 1.2 积分方法技巧

#### 第一换元法（凑微分法）
核心思想：把被积函数的一部分凑成 d(u) 的形式

**常见凑微分模式：**

| 类型 | 凑法 | 例子 |
|------|------|------|
| 线性凑 | dx = ¹⁄ₐ d(ax+b) | ∫ e³ˣ⁺¹dx = ¹⁄₃ e³ˣ⁺¹+C |
| 幂凑 | xⁿ dx = ¹⁄₍ₙ₊₁₎ d(xⁿ⁺¹) | ∫ x·eˣ²dx = ¹⁄₂ eˣ²+C |
| 对数凑 | ¹⁄ₓ dx = d(ln\|x\|) | ∫ (lnx)⁄ₓ dx = ¹⁄₂(lnx)²+C |
| 三角凑 | cosx dx = d(sinx) | ∫ sinx·cosx dx = ¹⁄₂ sin²x+C |
| 反三角凑 | ¹⁄₍₁₊ₓ²₎ dx = d(arctanx) | ∫ arctanx⁄₍₁₊ₓ²₎ dx = ¹⁄₂(arctanx)²+C |

**技巧口诀：** 看到复合函数，想想里面函数的导数在不在旁边

#### 第二换元法
核心思想：设 x = φ(t)，消去根号或简化形式

**常用换元：**

| 被积函数含 | 换元设 | 利用恒等式 |
|-----------|--------|-----------|
| √(a²-x²) | x = a·sin(t) | 1-sin²t = cos²t |
| √(a²+x²) | x = a·tan(t) | 1+tan²t = sec²t |
| √(x²-a²) | x = a·sec(t) | sec²t-1 = tan²t |
| ⁿ√(ax+b) | t = ⁿ√(ax+b) | 直接消根号 |
| 高次根式 | 取各根指数的最小公倍数 | 统一换元 |

**注意：** 换元后积分完，必须用原变量回代

#### 分部积分法
公式：∫ u dv = uv - ∫ v du

**选择 u 的口诀："反对幂指三"（优先级从高到低）**

| 优先级 | 类型 | 例子 |
|--------|------|------|
| 1 | 反三角函数 | arctanx, arcsinx |
| 2 | 对数函数 | lnx, logx |
| 3 | 幂函数 | xⁿ |
| 4 | 指数函数 | eˣ, aˣ |
| 5 | 三角函数 | sinx, cosx |

**记忆技巧：** 排在前面的选作 u，排在后面的选作 dv

**典型题型：**
- ∫ x·eˣ dx → u=x, dv=eˣdx
- ∫ x·lnx dx → u=lnx, dv=xdx
- ∫ x·arctanx dx → u=arctanx, dv=xdx
- ∫ eˣ·sinx dx → 两次分部积分，移项求解

#### 有理函数积分
步骤：
1. 假分式 → 先化为多项式+真分式（长除法）
2. 分母因式分解
3. 拆成部分分式
4. 逐项积分

**部分分式拆分规则：**

| 分母因子 | 对应部分分式 |
|---------|-------------|
| (x-a) | A⁄(x-a) |
| (x-a)ᵏ | A₁⁄(x-a) + A₂⁄(x-a)² + ... + Aₖ⁄(x-a)ᵏ |
| (x²+px+q) | (Ax+B)⁄(x²+px+q) |
| (x²+px+q)ᵏ | (A₁x+B₁)⁄(x²+px+q) + ... + (Aₖx+Bₖ)⁄(x²+px+q)ᵏ |

## 二、定积分

### 2.1 基本性质

| 性质 | 公式 |
|------|------|
| 积分限交换 | ∫ₐᵇ f(x)dx = -∫ᵇₐ f(x)dx |
| 相同积分限 | ∫ₐₐ f(x)dx = 0 |
| 线性性 | ∫ₐᵇ [αf(x)+βg(x)]dx = α∫ₐᵇ f(x)dx + β∫ₐᵇ g(x)dx |
| 区间可加 | ∫ₐᵇ f(x)dx = ∫ₐᶜ f(x)dx + ∫ᶜᵇ f(x)dx |
| 比较定理 | f(x)≤g(x) → ∫ₐᵇ f(x)dx ≤ ∫ₐᵇ g(x)dx |
| 估值定理 | m(b-a) ≤ ∫ₐᵇ f(x)dx ≤ M(b-a) |
| 积分中值 | ∫ₐᵇ f(x)dx = f(ξ)(b-a), ξ∈[a,b] |

### 2.2 牛顿-莱布尼茨公式（核心）

∫ₐᵇ f(x)dx = F(b) - F(a) = F(x)\|ₐᵇ

**其中 F(x) 是 f(x) 的一个原函数**

### 2.3 定积分计算方法

#### 换元法
设 x = φ(t)，则：
∫ₐᵇ f(x)dx = ∫ᵅᵝ f(φ(t))·φ'(t)dt

**关键：换元必换限！** 新的积分上下限对应新的变量值

#### 分部积分法
∫ₐᵇ u dv = uv\|ₐᵇ - ∫ₐᵇ v du

#### 利用奇偶性简化

| 条件 | 结论 |
|------|------|
| f(x)是偶函数，积分区间[-a,a] | ∫₋ₐᵃ f(x)dx = 2∫₀ᵃ f(x)dx |
| f(x)是奇函数，积分区间[-a,a] | ∫₋ₐᵃ f(x)dx = 0 |

**判断奇偶性技巧：**
- f(-x) = f(x) → 偶函数
- f(-x) = -f(x) → 奇函数
- 常见偶函数：x², cosx, \|x\|
- 常见奇函数：x, x³, sinx, tanx

#### 利用周期性
若 f(x) 周期为 T，则 ∫ₐ^(a+T) f(x)dx = ∫₀^T f(x)dx（与起点无关）

### 2.4 重要公式

| 公式 | 内容 |
|------|------|
| Wallis公式 | ∫₀^(π/2) sinⁿx dx = ∫₀^(π/2) cosⁿx dx = ⁽ⁿ⁻¹⁾⁄ₙ·⁽ⁿ⁻³⁾⁄₍ₙ₋₂₎·...·¹⁄₂·(π⁄₂)（n偶）或 ...·²⁄₃·1（n奇） |
| 华里士公式记忆 | 从n-1开始倒数，到1或2为止，n偶乘π/2，n奇乘1 |
| 对称区间 | ∫₀^(π/2) f(sinx)dx = ∫₀^(π/2) f(cosx)dx |

### 2.5 定积分应用

#### 平面图形面积

| 类型 | 公式 |
|------|------|
| 上下曲线 | S = ∫ₐᵇ [f(x)-g(x)]dx |
| 左右曲线 | S = ∫ᶜᵈ [φ(y)-ψ(y)]dy |
| 参数方程 | S = ∫ₐᵇ y(t)·x'(t)dt |
| 极坐标 | S = ¹⁄₂∫ₐᵝ r²(θ)dθ |

#### 旋转体体积

| 类型 | 公式 |
|------|------|
| 绕x轴 | Vₓ = π∫ₐᵇ [f(x)]²dx |
| 绕y轴 | Vᵧ = π∫ᶜᵈ [φ(y)]²dy |
| 柱壳法 | Vᵧ = 2π∫ₐᵇ x·f(x)dx |

#### 平面曲线弧长

| 类型 | 公式 |
|------|------|
| y=f(x) | s = ∫ₐᵇ √(1+y'²)dx |
| 参数方程 | s = ∫ₐᵇ √(x'²+y'²)dt |
| 极坐标 | s = ∫ₐᵝ √(r²+r'²)dθ |

## 三、常微分方程

### 3.1 一阶微分方程

#### 可分离变量方程
形式：dy/dx = f(x)·g(y)

解法：分离变量 → dy/g(y) = f(x)dx → 两边积分

**技巧：** 把含y的放一边，含x的放另一边

#### 齐次方程
形式：dy/dx = f(y/x)

解法：令 u = y/x，则 y = ux，dy/dx = u + x·du/dx
代入后化为可分离变量方程

#### 一阶线性方程
形式：y' + P(x)y = Q(x)

**通解公式（常数变易法/公式法）：**
y = e^(-∫P(x)dx) · [∫Q(x)·e^(∫P(x)dx)dx + C]

**记忆口诀：** 先算P的积分，再算e的指数，然后套公式

**技巧：** 如果方程是 x' + P(y)x = Q(y)（把x看作y的函数），公式同样适用

#### 伯努利方程
形式：y' + P(x)y = Q(x)yⁿ (n≠0,1)

解法：令 z = y¹⁻ⁿ，化为关于z的一阶线性方程

### 3.2 可降阶的高阶方程

| 类型 | 形式 | 解法 |
|------|------|------|
| 缺y | y⁽ⁿ⁾ = f(x) | 连续积分n次 |
| 缺y | y'' = f(x,y') | 令 p = y'，则 y'' = p'，降为一阶 |
| 缺x | y'' = f(y,y') | 令 p = y'，则 y'' = p·dp/dy，降为一阶 |

### 3.3 二阶常系数线性方程

#### 齐次方程：y'' + py' + qy = 0

**解法步骤：**
1. 写特征方程：r² + pr + q = 0
2. 求特征根
3. 根据根的情况写通解

| 特征根情况 | 通解形式 |
|-----------|---------|
| 两个不等实根 r₁≠r₂ | y = C₁e^(r₁x) + C₂e^(r₂x) |
| 两个相等实根 r₁=r₂=r | y = (C₁ + C₂x)e^(rx) |
| 共轭复根 r=α±βi | y = e^(αx)(C₁cosβx + C₂sinβx) |

**记忆技巧：** 实根写指数，复根写指数乘三角，重根多乘个x

#### 非齐次方程：y'' + py' + qy = f(x)

**通解 = 齐次通解 + 非齐次特解**

**特解求法——待定系数法：**

| f(x)形式 | 设特解形式 | 注意 |
|---------|-----------|------|
| Pₙ(x)（n次多项式） | xˢ·Qₙ(x) | s=0,1,2看0是不是特征根 |
| e^(αx)·Pₙ(x) | xˢ·e^(αx)·Qₙ(x) | s看α是不是特征根 |
| e^(αx)·[Acosβx+Bsinβx] | xˢ·e^(αx)·[Ccosβx+Dsinβx] | s看α±βi是不是特征根 |

**技巧：** 如果f(x)是几种形式的和，用叠加原理分别求特解再相加

### 3.4 微分方程应用建模步骤

1. 分析问题，确定未知函数和自变量
2. 根据物理/几何规律建立方程
3. 确定初始条件
4. 解方程
5. 验证结果

**常见模型：**
- 增长/衰减模型：dy/dt = ky
- 牛顿冷却定律：dT/dt = -k(T-T₀)
- 力学问题：F=ma 建立方程

## 四、偏导数与多元函数微分

### 4.1 偏导数定义

对x的偏导数：fₓ(x,y) = ∂f/∂x = lim(Δx→0) [f(x+Δx,y)-f(x,y)]/Δx

**求法：** 把y看作常数，对x求导

### 4.2 高阶偏导数

| 记号 | 含义 |
|------|------|
| ∂²f/∂x² 或 fₓₓ | 对x求两次偏导 |
| ∂²f/∂y² 或 fᵧᵧ | 对y求两次偏导 |
| ∂²f/∂x∂y 或 fₓᵧ | 先对x后对y求偏导 |
| ∂²f/∂y∂x 或 fᵧₓ | 先对y后对x求偏导 |

**重要定理：** 若 fₓᵧ 和 fᵧₓ 连续，则 fₓᵧ = fᵧₓ（混合偏导数相等）

### 4.3 全微分

#### 二元函数
若 z = f(x,y) 可微，则：
dz = (∂z/∂x)dx + (∂z/∂y)dy = fₓdx + fᵧdy

#### 可微的充分条件
若 fₓ 和 fᵧ 在点(x,y)连续，则f在该点可微

#### 可微的必要条件
可微 → 连续，可微 → 偏导数存在

**关系链：** 偏导连续 → 可微 → 偏导存在 + 连续

### 4.4 复合函数求导（链式法则）

#### 中间变量都是一元函数
z = f(u,v), u=u(t), v=v(t)

dz/dt = (∂z/∂u)·(du/dt) + (∂z/∂v)·(dv/dt)

**画树状图帮助记忆：** z→u→t, z→v→t

#### 中间变量都是多元函数
z = f(u,v), u=u(x,y), v=v(x,y)

∂z/∂x = (∂z/∂u)·(∂u/∂x) + (∂z/∂v)·(∂v/∂x)
∂z/∂y = (∂z/∂u)·(∂u/∂y) + (∂z/∂v)·(∂v/∂y)

**技巧：** 有几条路径到x，就有几项相加

#### 既有中间变量又有自变量
z = f(u,v,x,y), u=u(x,y), v=v(x,y)

∂z/∂x = (∂f/∂u)·(∂u/∂x) + (∂f/∂v)·(∂v/∂x) + ∂f/∂x

**注意：** 最后一项 ∂f/∂x 是f对第四个位置变量x的偏导

### 4.5 隐函数求导

#### 一个方程情形
F(x,y) = 0 确定 y = y(x)：
dy/dx = -Fₓ/Fᵧ

F(x,y,z) = 0 确定 z = z(x,y)：
∂z/∂x = -Fₓ/Fᵧ, ∂z/∂y = -Fᵧ/Fᵧ

**记忆口诀：** 负号，下对下，上对下（分子是F对自变量的偏导，分母是F对因变量的偏导）

#### 方程组情形
用雅可比行列式或两边直接求导再解方程组

### 4.6 方向导数与梯度

#### 方向导数
∂f/∂l = fₓ·cosα + fᵧ·cosβ + fᵧ·cosγ

其中 (cosα, cosβ, cosγ) 是方向l的方向余弦

#### 梯度
grad f = (fₓ, fᵧ, fᵧ) = ∇f

**性质：** 方向导数最大值 = |grad f|，方向为梯度方向

### 4.7 多元函数极值

#### 无条件极值

**步骤：**
1. 求驻点：解方程组 fₓ = 0, fᵧ = 0
2. 判别：设 A = fₓₓ, B = fₓᵧ, C = fᵧᵧ
   - AC - B² > 0 且 A > 0 → 极小值
   - AC - B² > 0 且 A < 0 → 极大值
   - AC - B² < 0 → 不是极值（鞍点）
   - AC - B² = 0 → 不确定

**记忆：** 判别式 Δ = AC - B²，类似一元二次方程的判别式

#### 条件极值（拉格朗日乘数法）

求 z = f(x,y) 在约束 φ(x,y) = 0 下的极值

**步骤：**
1. 构造 L = f(x,y) + λ·φ(x,y)
2. 解方程组：Lₓ = 0, Lᵧ = 0, L_λ = 0
3. 判断极值类型（根据实际问题）

**多个约束：** L = f + λ₁φ₁ + λ₂φ₂ + ...

## 五、解题技巧总结

### 5.1 不定积分解题策略

1. **先看能否直接套公式**
2. **有复合函数 → 尝试凑微分**
3. **有根号 → 换元消根号**
4. **两类不同函数乘积 → 分部积分**
5. **有理函数 → 部分分式分解**
6. **三角函数积分 → 利用恒等变形**

### 5.2 定积分解题策略

1. **先看对称区间 → 判断奇偶性**
2. **有周期函数 → 利用周期性简化**
3. **换元法 → 记住换限**
4. **分部积分 → 带上下限**
5. **几何意义 → 有时能直接看出结果**

### 5.3 微分方程识别流程

```
一阶方程？
  ├─ 能分离变量？ → 分离变量法
  ├─ 能写成 y/x 形式？ → 齐次方程换元
  ├─ 线性？ → 套通解公式
  └─ 伯努利？ → 换元 z=y¹⁻ⁿ

高阶方程？
  ├─ 缺y？ → 令 p=y'
  ├─ 缺x？ → 令 p=y', y''=p·dp/dy
  └─ 常系数线性？ → 特征方程法
```

### 5.4 偏导数解题要点

1. **求偏导 → 其他变量当常数**
2. **复合函数 → 画树状图，链式法则**
3. **隐函数 → 套公式或两边求导**
4. **极值问题 → 先找驻点，再判别**
5. **条件极值 → 拉格朗日乘数法**

## 六、常见错误提醒

| 错误类型 | 正确做法 |
|---------|---------|
| 忘记+C | 不定积分必须加常数C |
| 换元不换限 | 定积分换元必须换积分限 |
| 分母为0不讨论 | 注意定义域和可积条件 |
| 特征根算错 | 仔细解二次方程 |
| 特解形式设错 | 对照特征根情况选s值 |
| 偏导符号混淆 | 明确对谁求导，其他当常数 |
| 链式法则漏项 | 画树状图，数清楚路径 |
