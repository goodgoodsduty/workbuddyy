## 第一章 计算机系统组成与原理

### 1.1 计算机发展史 ⚠️ 必考

| 代次 | 时间 | 逻辑器件 | 编程语言 |
|------|------|---------|---------|
| 第一代 | 1946-1957 | 电子管 | 机器语言 |
| 第二代 | 1958-1964 | 晶体管 | 汇编语言 |
| 第三代 | 1965-1970 | 中小规模集成电路 | 高级语言 |
| 第四代 | 1971-至今 | 大规模/超大规模集成电路 | 操作系统 |

**【记忆】** "**管**1、**晶**2、**集**3、**大**4"

### 1.2 冯·诺依曼体系结构 ⚠️ 必考

**【五大部件】**

```
输入设备 → 存储器 ← 运算器
              ↑         ↑
              └── 控制器 ┘
              ↓
           输出设备
```

| 部件 | 功能 |
|------|------|
| **运算器** | 算术运算、逻辑运算（ALU）|
| **控制器** | 指令控制、程序控制（CU）|
| **存储器** | 存储程序和数据 |
| **输入设备** | 接收外部数据 |
| **输出设备** | 输出处理结果 |

**【冯·诺依曼思想核心】** ⚠️ 必背
- **存储程序**：程序和数据都存放在内存中
- **程序控制**：CPU自动按指令顺序执行

**【例题1.1】** 冯·诺依曼计算机的核心思想是？
A. 二进制
B. 存储程序
C. 大规模集成电路
D. 高速缓存

**答案**：**B. 存储程序**

### 1.3 CPU（中央处理器）

**【组成】** 运算器 + 控制器 + 寄存器

**【性能指标】** ⚠️

| 指标 | 说明 |
|------|------|
| **主频** | CPU时钟频率，单位GHz，越高越快 |
| **字长** | CPU一次能处理的二进制位数（32位/64位）|
| **核心数** | 多核CPU可并行处理 |
| **缓存** | L1/L2/L3 高速缓存 |

### 1.4 存储器层次结构 ⚠️ 必考

**【金字塔结构】（从上到下速度递减，容量递增）**

```
        寄存器
          ↓
        Cache（高速缓存）
          ↓
        主存（内存）
          ↓
        辅存（硬盘/SSD）
```

**【内存储器分类】**

| 类型 | 特点 |
|------|------|
| **RAM** | 随机存取，可读可写，易失性 |
| **ROM** | 只读，断电不丢失 |

**【外存储器】**

| 类型 | 特点 |
|------|------|
| 硬盘（HDD） | 容量大、速度慢、机械结构 |
| 固态硬盘（SSD） | 速度快、寿命有限、电子结构 |
| U盘 | 便携、容量适中 |
| 光盘 | CD/DVD/蓝光 |

### 1.5 总线

**【三类总线】**

| 类型 | 功能 |
|------|------|
| **数据总线** | 传输数据（双向）|
| **地址总线** | 传输地址（单向）|
| **控制总线** | 传输控制信号（双向）|

**【例题1.2】** 决定CPU最大寻址范围的是？
A. 数据总线宽度
B. 地址总线宽度
C. 控制总线宽度
D. 时钟频率

**答案**：**B. 地址总线宽度**（$2^n$ 决定可寻址范围）

### 1.6 数制转换 ⚠️ 必考

**【十进制 ↔ 二进制】**
- 十→二：除2取余，倒序
- 二→十：按权展开

**【二进制 ↔ 八进制】**
- 二→八：3位一组，不足补0
- 八→二：每1位展3位

**【二进制 ↔ 十六进制】**
- 二→十六：4位一组
- 十六→二：每1位展4位

**【例题1.3】** (123)₁₀ = ( ？)₂
**解**：123 ÷ 2 = 61...1；61 ÷ 2 = 30...1；30 ÷ 2 = 15...0；15 ÷ 2 = 7...1；7 ÷ 2 = 3...1；3 ÷ 2 = 1...1；1 ÷ 2 = 0...1
倒序：1111011

### 1.7 原码、反码、补码 ⚠️ 必考

**【正数】** 原码 = 反码 = 补码

**【负数】**
- **原码**：符号位1，数值位是绝对值
- **反码**：符号位1，数值位取反
- **补码**：反码 + 1

**【例题1.4】** -7 的8位二进制表示
- 原码：10000111
- 反码：11111000
- 补码：11111001

**【例题1.5】** 已知补码 11111010，求原码
- 补码：11111010
- 反码：11111001
- 原码：10000110（即 -6）

## 第二章 操作系统基础

### 2.1 操作系统的定义

**【定义】** 操作系统（OS）是管理计算机硬件和软件资源、为用户提供交互界面的系统软件。

### 2.2 操作系统的功能 ⚠️ 必考

**【五大功能】**

| 功能 | 说明 |
|------|------|
| **进程管理** | 进程调度、同步、通信 |
| **存储管理** | 内存分配、虚拟存储 |
| **文件管理** | 文件的存储、检索、保护 |
| **设备管理** | I/O设备管理 |
| **作业管理** | 用户作业的调度 |

### 2.3 操作系统的特征

| 特征 | 说明 |
|------|------|
| **并发** | 多个事件同时发生 |
| **共享** | 资源供多个用户使用 |
| **虚拟** | 把一个物理实体映射为多个逻辑实体 |
| **异步** | 事件发生顺序不可预测 |

### 2.4 操作系统分类 ⚠️ 必考

| 类别 | 特点 | 例子 |
|------|------|------|
| **批处理** | 批量处理作业 | DOS |
| **分时** | 多个用户分时使用 | UNIX, Linux |
| **实时** | 实时响应 | RTOS |
| **网络** | 网络管理 | Windows Server |
| **分布式** | 多台计算机协同 | 分布式系统 |
| **嵌入式** | 嵌入式设备 | Android, iOS |

### 2.5 进程管理 ⚠️ 必考

**【进程的三个基本状态】**

```
   就绪 ←→ 运行 → 阻塞
                ↓
              退出
```

| 状态 | 说明 |
|------|------|
| **就绪** | 等待CPU调度 |
| **运行** | 正在CPU上执行 |
| **阻塞** | 等待某个事件（如I/O）|

### 2.6 存储管理

**【常用技术】**

| 技术 | 说明 |
|------|------|
| **分区** | 把内存分若干区 |
| **分页** | 把内存分等大小的页 |
| **分段** | 按逻辑单位分 |
| **虚拟存储** | 内存+外存结合 |

### 2.7 文件系统 ⚠️ 必考

**【常见文件系统】**

| 文件系统 | 平台 |
|---------|------|
| FAT32 | Windows |
| NTFS | Windows |
| ext4 | Linux |
| APFS | macOS |
| HFS+ | macOS |

## 第三章 程序设计语言基础

### 3.1 语言分类 ⚠️ 必考

| 类别 | 特点 | 例子 |
|------|------|------|
| **机器语言** | 二进制代码，计算机直接执行 | - |

| **混合型** | 先编译字节码，再解释 | 综合 | Java |

### 3.3 程序三种基本结构 ⚠️ 必考

| 结构 | 特点 |
|------|------|
| **顺序** | 自上而下 |
| **选择** | if, switch |
| **循环** | for, while, do-while |

### 3.4 面向对象 vs 面向过程

| 类别 | 核心 | 例子 |
|------|------|------|
| **面向过程** | 函数/过程 | C, Pascal |
| **面向对象** | 类和对象 | Java, C++, Python |

**【OOP 三大特性】** 封装、继承、多态

## 第四章 计算机网络基础 ⚠️ 重要

### 4.1 网络分类

| 分类标准 | 类型 |
|---------|------|
| **地理范围** | PAN/LAN/MAN/WAN |
| **拓扑结构** | 总线型/星型/环型/树型/网状 |
| **传输介质** | 有线/无线 |
| **使用性质** | 专用网/公用网 |

### 4.2 OSI 七层模型 ⚠️ 必背

**【从上到下】** ⚠️ 记忆口诀：**"物链网传会表应"**

| 层 | 名称 | 功能 | 协议/设备 |
|----|------|------|----------|
| 7 | 应用层 | 用户接口 | HTTP, FTP, DNS, SMTP |
| 6 | 表示层 | 数据表示、加密 | SSL/TLS, JPEG |
| 5 | 会话层 | 会话管理 | RPC, NetBIOS |
| 4 | 传输层 | 端到端传输 | TCP, UDP |
| 3 | 网络层 | 路由选择 | IP, ICMP, 路由器 |
| 2 | 数据链路层 | 帧传输、差错控制 | MAC, 交换机, PPP |
| 1 | 物理层 | 比特流传输 | 网线, 光纤, 集线器 |

### 4.3 TCP/IP 四层模型 ⚠️ 必考

| 层 | 对应 OSI | 协议 |
|----|---------|------|
| **应用层** | 5-7 | HTTP, FTP, DNS |
| **传输层** | 4 | TCP, UDP |
| **网络层** | 3 | IP, ICMP, ARP |
| **网络接口层** | 1-2 | Ethernet, PPP |

### 4.4 重要协议 ⚠️ 必考

| 协议 | 层 | 用途 |
|------|----|------|
| **HTTP** | 应用层 | 网页传输 |
| **HTTPS** | 应用层 | 加密网页传输 |
| **FTP** | 应用层 | 文件传输 |
| **DNS** | 应用层 | 域名解析 |
| **SMTP** | 应用层 | 邮件发送 |
| **POP3** | 应用层 | 邮件接收 |
| **TCP** | 传输层 | 可靠传输 |
| **UDP** | 传输层 | 不可靠传输 |
| **IP** | 网络层 | IP地址 |
| **ICMP** | 网络层 | 网络控制（如ping）|

### 4.5 IP 地址 ⚠️ 必考

**【IPv4 地址】** 32位，4段8位二进制

**【分类】**

| 类别 | 范围 | 默认子网掩码 | 用途 |
|------|------|------------|------|
| A类 | 1.0.0.0 - 126.255.255.255 | 255.0.0.0 | 大型网络 |
| B类 | 128.0.0.0 - 191.255.255.255 | 255.255.0.0 | 中型网络 |
| C类 | 192.0.0.0 - 223.255.255.255 | 255.255.255.0 | 小型网络 |
| D类 | 224.0.0.0 - 239.255.255.255 | - | 组播 |
| E类 | 240.0.0.0 - 255.255.255.255 | - | 保留 |

**【特殊 IP】** ⚠️
- 127.0.0.1：回环地址（localhost）
- 255.255.255.255：广播地址
- 0.0.0.0：本网络

**【子网掩码】** 用来区分网络号和主机号

**【例题4.1】** 192.168.1.100/24 的网络号是？
A. 192.168.1.0
B. 192.168.1.100
C. 192.168.0.0
D. 192.168.1.255

**答案**：**A. 192.168.1.0**（/24 表示子网掩码 255.255.255.0）

**【例题4.2】** 一个B类网络，默认子网掩码是？
**答案**：255.255.0.0

## 第五章 多媒体技术

### 5.1 多媒体的特征

| 特征 | 说明 |
|------|------|
| 多样性 | 多种媒体 |
| 集成性 | 综合处理 |
| 交互性 | 人机交互 |
| 实时性 | 同步处理 |

### 5.2 图像基础 ⚠️ 必考

**【图像数字化】**
- **采样**：空间离散化
- **量化**：幅度离散化
- **编码**：转换为数字信号

**【图像文件大小计算】** ⚠️ 必背

$$\text{文件大小} = \text{水平像素} \times \text{垂直像素} \times \text{颜色位数} \div 8$$

**【例题5.1】** 一幅 1024×768 像素的24位真彩色图像，不压缩情况下大小是？
A. 1.5MB
B. 2.25MB
C. 3MB
D. 4MB

**答案**：**B. 2.25MB**（1024×768×24÷8 = 2359296 字节 = 2.25MB）

### 5.3 图像格式 ⚠️ 必考

| 格式 | 特点 |
|------|------|
| **BMP** | 无压缩，文件大 |
| **JPEG** | 有损压缩，照片 |
| **GIF** | 256色，动画 |
| **PNG** | 无损压缩，支持透明 |
| **TIFF** | 高质量印刷 |

### 5.4 音频基础 ⚠️

**【音频数字化】**
- 采样
- 量化
- 编码

**【音频文件大小】** = 采样频率 × 量化位数 × 声道数 × 时间 ÷ 8

**【例题5.2】** 1分钟 CD 音质音频（44.1kHz, 16bit, 立体声）大小是？
**解**：44100 × 16 × 2 × 60 ÷ 8 = 10584000 字节 ≈ 10MB

### 5.5 视频基础

**【视频格式】** AVI, MP4, MKV, MOV, FLV

**【视频 = 图像 + 音频】**，加上时间维度

### 5.6 数据压缩 ⚠️

| 类型 | 特点 | 例子 |
|------|------|------|
| **无损压缩** | 可完全还原 | ZIP, RAR |
| **有损压缩** | 不可完全还原 | JPEG, MP3 |
