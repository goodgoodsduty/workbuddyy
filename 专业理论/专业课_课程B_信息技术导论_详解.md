## 第一章 云计算 ⚠️ 必考

### 1.1 云计算的概念

**【定义】** 通过网络"按需分配"获得计算资源（服务器、存储、网络、软件）的服务模式。

**【类比】** 像水/电一样使用，按用量付费。

### 1.2 云计算的五大特征 ⚠️

| 特征 | 英文 | 说明 |
|------|------|------|
| **按需自助** | On-demand self-service | 随时获取资源 |
| **广泛网络访问** | Broad network access | 网络可达 |
| **资源池化** | Resource pooling | 共享资源池 |
| **快速弹性** | Rapid elasticity | 快速伸缩 |
| **可计量服务** | Measured service | 按用量计费 |

### 1.3 云计算服务模式 ⚠️ 必考

| 模式 | 全称 | 说明 | 例子 |
|------|------|------|------|
| **IaaS** | Infrastructure as a Service | 提供基础设施 | 阿里云ECS, AWS EC2 |
| **PaaS** | Platform as a Service | 提供平台 | Google App Engine |
| **SaaS** | Software as a Service | 提供软件 | Office 365, 钉钉 |

**【口诀】** "**I**自己搭环境，**P**提供平台，**S**直接用软件"

### 1.4 云计算部署模式 ⚠️ 必考

| 模式 | 说明 |
|------|------|
| **公有云** | 第三方提供商，面向公众 |
| **私有云** | 仅为一个组织服务 |
| **混合云** | 公有云+私有云 |
| **社区云** | 为特定社区服务 |

### 1.5 虚拟化技术

**【定义】** 把一个物理资源虚拟成多个逻辑资源

**【类型】**
- 服务器虚拟化
- 存储虚拟化
- 网络虚拟化
- 桌面虚拟化

### 1.6 云存储

**【代表】** 阿里云OSS、腾讯云COS、AWS S3

**【优势】** 高可用、可扩展、低成本

**【例题1.1】** 阿里云ECS 属于哪种云服务模式？
A. IaaS
B. PaaS
C. SaaS
D. DaaS

**答案**：**A. IaaS**（提供虚拟机）

## 第二章 大数据 ⚠️ 必考

### 2.1 大数据的4V特征 ⚠️ 必考

| 特征 | 英文 | 说明 |
|------|------|------|
| **大量** | Volume | 数据量大（TB/PB级）|
| **高速** | Velocity | 处理速度快 |
| **多样** | Variety | 类型多样（结构化/非结构化）|
| **价值** | Value | 价值密度低 |

**【口诀】** "**多快好省**"（多=Volume, 快=Velocity, 好=Variety, 省=Value低）

### 2.2 大数据处理流程

```
数据采集 → 数据存储 → 数据清洗 → 数据分析 → 数据可视化
```

### 2.3 Hadoop 生态系统 ⚠️

| 组件 | 功能 |
|------|------|
| **HDFS** | 分布式文件系统（存储）|
| **MapReduce** | 分布式计算框架 |
| **HBase** | 分布式数据库 |
| **Hive** | 数据仓库 |
| **Pig** | 数据流语言 |
| **Sqoop** | 数据导入导出 |
| **Flume** | 日志收集 |

### 2.4 数据挖掘

**【定义】** 从大量数据中提取隐含信息的过程

**【任务类型】**
- 分类
- 聚类
- 关联规则
- 回归
- 异常检测

### 2.5 数据可视化

**【工具】** Tableau、Power BI、ECharts、Matplotlib

**【常见图表】** 折线图、柱状图、饼图、散点图、热力图

### 2.6 大数据应用场景

- 电商推荐系统
- 金融风控
- 智慧城市
- 医疗健康
- 交通预测

**【例题2.1】** 大数据的4V特征不包括？
A. Volume
B. Velocity
C. Variety
D. Visibility

**答案**：**D. Visibility**

## 第三章 人工智能 ⚠️ 必考

### 3.1 AI 的定义

**【定义】** 研究使机器具有人类智能的学科，包括感知、推理、学习、决策等能力。

### 3.2 AI 发展简史

| 阶段 | 时间 | 特征 |
|------|------|------|
| 起源 | 1950s | 图灵测试 |
| 符号主义 | 1960s-1980s | 专家系统 |
| 连接主义 | 1980s-2000s | 神经网络 |
| 深度学习 | 2010s-至今 | 突破性进展 |
| 大模型 | 2020s- | ChatGPT |

### 3.3 AI 技术分类 ⚠️ 必考

| 类别 | 英文 | 说明 | 例子 |
|------|------|------|------|
| **机器学习** | Machine Learning | 让机器从数据中学习 | - |
| **深度学习** | Deep Learning | 多层神经网络 | CNN, RNN |
| **自然语言处理** | NLP | 处理人类语言 | 机器翻译 |
| **计算机视觉** | CV | 让机器"看" | 人脸识别 |
| **语音识别** | Speech | 语音转文字 | 智能音箱 |
| **专家系统** | Expert System | 模拟专家决策 | 医疗诊断 |

### 3.4 机器学习 ⚠️ 必考

**【监督学习】** 有标签数据
- 分类（Classification）：预测离散值
- 回归（Regression）：预测连续值
- 例子：垃圾邮件分类、房价预测

**【无监督学习】** 无标签数据
- 聚类（Clustering）：分组
- 降维（Dimensionality Reduction）
- 例子：客户分群

**【强化学习】** 通过试错学习
- 例子：AlphaGo、游戏AI

### 3.5 神经网络

**【神经元模型】**

```
       输入x1  →  w1
                    \
       输入x2  →  w2 → Σ → 激活函数 → 输出
                    /
       输入xn  →  wn
```

**【深度学习】** 多层神经网络（输入层+多个隐藏层+输出层）

### 3.6 AI 应用场景

- **图像识别**：人脸识别、医疗影像
- **语音识别**：智能音箱、语音助手
- **自然语言处理**：机器翻译、智能客服
- **自动驾驶**：特斯拉、Waymo
- **推荐系统**：抖音、淘宝推荐
- **医疗诊断**：AI辅助诊断

**【例题3.1】** 下列哪项不属于AI技术？
A. 机器学习
B. 深度学习
C. 关系数据库
D. 自然语言处理

**答案**：**C. 关系数据库**

## 第四章 区块链 ⚠️ 必考

### 4.1 区块链定义

**【定义】** 分布式数据存储、点对点传输、共识机制、加密算法等计算机技术的新型应用模式。

**【本质】** 不可篡改的分布式账本

### 4.2 区块链的特征 ⚠️

| 特征 | 说明 |
|------|------|
| **去中心化** | 不依赖中心服务器 |
| **不可篡改** | 一旦写入难以修改 |
| **可追溯** | 全部交易可查 |
| **公开透明** | 公开账本 |
| **集体维护** | 全网节点共同维护 |

### 4.3 区块链技术框架

```
应用层
合约层（如智能合约）
激励层
共识层（PoW/PoS/DPoS）
网络层（P2P）
数据层（区块+链）
```

### 4.4 共识机制 ⚠️ 必考

| 机制 | 全称 | 原理 | 例子 |
|------|------|------|------|
| **PoW** | Proof of Work 工作量证明 | 算力竞争 | 比特币 |
| **PoS** | Proof of Stake 权益证明 | 持币量+币龄 | 以太坊 |
| **DPoS** | Delegated PoS 委托权益证明 | 投票选举 | EOS |

**【口诀】** "**W**ork算力，**S**take币，**D**elegate代表"

### 4.5 智能合约

**【定义】** 自动执行的合约条款，以代码形式部署在区块链上

**【特点】** 不可篡改、自动执行、公开透明

### 4.6 区块链应用场景

- 加密货币（比特币、以太坊）
- 供应链溯源
- 数字身份
- 版权保护
- 金融结算

**【例题4.1】** 比特币采用的共识机制是？
A. PoS
B. PoW
C. DPoS
D. PBFT

**答案**：**B. PoW**

**【例题4.2】** 下列哪个不是区块链的特征？
A. 去中心化
B. 不可篡改
C. 高吞吐量
D. 可追溯

**答案**：**C. 高吞吐量**（区块链的吞吐量目前是瓶颈）

## 第五章 物联网与5G

### 5.1 物联网 IoT ⚠️ 必考

**【定义】** 通过信息传感设备，按约定的协议，把任何物品与互联网相连接，进行信息交换和通信。

**【三层架构】**
- **感知层**：传感器、RFID
- **网络层**：通信网络
- **应用层**：智能应用

### 5.2 物联网关键技术 ⚠️

| 技术 | 作用 |
|------|------|
| **传感器** | 采集数据 |
| **RFID** | 射频识别 |
| **嵌入式系统** | 设备控制 |
| **通信技术** | 数据传输 |
| **云计算** | 数据处理 |
| **大数据** | 数据分析 |

### 5.3 物联网应用

- 智能家居
- 智能交通
- 智能医疗
- 智慧农业
- 工业物联网

### 5.4 5G 通信 ⚠️

**【5G 三大场景】**
- **eMBB**：增强移动宽带
- **uRLLC**：超可靠低时延
- **mMTC**：海量机器通信

**【5G 关键性能】**
- 峰值速率：10Gbps
- 时延：1ms
- 连接数：100万/km²

### 5.5 5G vs 4G

| 指标 | 4G | 5G |
|------|-----|-----|
| 峰值速率 | 1Gbps | 10-20Gbps |
| 时延 | 10ms | 1ms |
| 连接密度 | 10万/km² | 100万/km² |
